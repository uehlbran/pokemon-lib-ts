## Quick Start: What Changed from Gen 8

- Terastallization replaces Dynamax as the single battle gimmick
- No Mega Evolution, Z-Moves, or Dynamax — only Terastallization
- Snow replaces Hail: no residual damage, instead boosts Ice-type Defense by 1.5×
- Terrain boost: 1.3× (same as Gen 8, changed from 1.5× in Gen 7)
- Paralysis speed: ×0.5 (same as Gen 7-8)
- Burn damage: 1/16 max HP per turn (same as Gen 7-8)
- Critical hit: 1/24 base, 1.5× multiplier (same as Gen 7-8)
- Protean/Libero: once per switchin (nerfed from Gen 8's every-move activation)
- Intrepid Sword/Dauntless Shield: once per battle (nerfed from Gen 8's every-switchin)
- Stellar Tera Type (DLC): special STAB rules, one-time 2× per type

## Terastallization

### Concept

Terastallization is a transformative battle gimmick where a Pokémon's type temporarily changes to its **Tera Type**:

- **Permanent for the battle**: once Terastallized, it remains Tera-typed for the entire battle (does not revert on switch)
- **One per battle per team**: each trainer can use Terastallization once per battle on any of their Pokémon
- **Single type, not dual**: the Pokémon becomes a single Tera Type defensively, replacing its original type(s)
- **Original types still matter for STAB**: moves matching original types still get 1.5× STAB even after Terastallization

### Tera Type Mechanics

Each Pokémon has a Tera Type assigned before battle. On Terastallization:
- Defensive typing becomes the Tera Type only (single type)
- STAB is computed based on both current (Tera) type AND original base types (see STAB section)

### STAB (Same-Type Attack Bonus) Interaction with Tera

Showdown `sim/battle-actions.ts:1760-1793`:

```typescript
// Step 1: Check if move type matches current type OR original base types
const isSTAB = move.forceSTAB || pokemon.hasType(type) || pokemon.getTypes(false, true).includes(type);
if (isSTAB) {
  stab = 1.5;
}

// Step 2: If Tera type matches original type AND move type, boost to 2×
if (pokemon.terastallized === type && pokemon.getTypes(false, true).includes(type)) {
  stab = 2;
}
```

**Rules**:
1. **Move matches Tera type AND Tera type is an original type**: **2.0× STAB** (e.g., Fire/Flying Charizard with Fire Tera using Flamethrower)
2. **Move matches Tera type but Tera type is NOT an original type**: **1.5× STAB** (e.g., Fire/Flying Charizard with Grass Tera using Energy Ball)
3. **Move matches an original type but NOT Tera type**: **1.5× STAB** (e.g., Fire/Flying Charizard with Grass Tera using Flamethrower — Fire is still an original type)
4. **Move doesn't match Tera type or any original type**: **1.0× (no STAB)**

### Stellar Tera Type (DLC: The Hidden Treasure of Area Zero)

Showdown `sim/battle-actions.ts:1770-1785`:

The Stellar Tera Type has unique STAB rules:
- **Move matches one of the user's base types**: 2× STAB boost (first use per type only, then reverts to 1.5× for that type)
- **Move does NOT match base types**: 1.2× damage boost (ratio [4915, 4096])
- Tracks `stellarBoostedTypes` per Pokémon — each type can only get the enhanced boost once
- Exception: Terapagos-Stellar does not consume stellar boosts

**Tera Blast with Stellar** (Showdown `data/moves.ts:19919-19955`):
- Base power: **100** (up from 80 for normal Tera)
- Self-debuff: **-1 Attack, -1 Special Attack** after use
- Type: Stellar (unique — hits all types neutrally except Stellar-on-Stellar)

### Tera Blast Move

### Adaptability Interaction with Tera STAB

Showdown `data/abilities.ts:43-56`:

```typescript
// Adaptability does NOT simply double STAB
onModifySTAB(stab, source, target, move) {
  if (move.forceSTAB || source.hasType(move.type)) {
    if (stab === 2) return 2.25;  // 2× Tera STAB → 2.25×
    return 2;                      // 1.5× normal STAB → 2×
  }
}
```

| Base STAB | With Adaptability | Notes |
|-----------|-------------------|-------|
| 1.5× (current type match) | 2.0× | Normal STAB on current type becomes 2× |
| 2.0× (Tera+original match) | 2.25× | Tera STAB becomes 2.25× |
| 1.5× (original type only, Tera'd away) | 1.5× (NO boost) | Adaptability does NOT apply — `hasType()` returns Tera type only |
| 1.0× (no STAB) | 1.0× | No change |

**Critical edge case**: When Terastallized to a different type, `hasType(moveType)` only checks the current (Tera) type. Original-type moves still get 1.5× STAB via the `getTypes(false, true)` check in the STAB code, but Adaptability's `source.hasType(move.type)` condition fails for those original types. So a Fire/Flying Charizard with Adaptability Tera'd to Grass using Flamethrower gets 1.5× (not 2×).

**Stellar exclusion**: `ModifySTAB` event (which triggers Adaptability) is only called in the non-Stellar branch. Adaptability has no effect when using Stellar Tera.

## Snow Replaces Hail

In Gen 9, Hail weather is replaced by Snow (called "Snowscape" internally in Showdown).

## Damage Formula

### Complete Damage Formula (Gen 9)

Same foundation as Gen 3+:

```
baseDamage = floor(floor(floor(2 × Level / 5 + 2) × Power × A / D) / 50) + 2
```

### Modifier Chain (Showdown Exact Order)

Showdown `sim/battle-actions.ts:1733-1826`:

Applied in this specific sequence:

1. **Spread hit** (0.75× in doubles, 0.5× in free-for-all)
2. **Parental Bond** (0.25× for second hit in Gen 7+)
3. **Weather** (Sun: Fire 1.5×/Water 0.5×; Rain: Water 1.5×/Fire 0.5×)
4. **Critical hit** (1.5× in Gen 6+)
5. **Random factor** (85-100, integer division by 100)
6. **STAB** (1.5× or 2.0× based on Tera rules)
7. **Type effectiveness** (0.25×, 0.5×, 1×, 2×, 4×)
8. **Burn** (0.5× for Physical moves unless Guts ability)
9. **Final modifier** (ModifyDamage event: Screens, Life Orb, etc.)

### Burn
- Residual damage: **1/16 max HP per turn** (Showdown `data/conditions.ts:17`)
- Physical damage reduction: 0.5× (unless Guts ability)
- **Facade exception**: in Gen 6+, Facade ignores the burn damage penalty (`battle-actions.ts:1815-1816`)

### Paralysis
- Speed reduction: ×0.5 (Showdown `data/conditions.ts:35`: `Math.floor(spe * 50 / 100)`)

### Poison
- Regular poison: 1/8 max HP per turn
- Toxic (badly poisoned): escalating N/16 per turn (N increments each turn, resets on switch-in)

- Self-hit chance: 33% per turn (Gen 7+ value)
- Duration: 2-5 turns

### Freeze
- 20% chance to thaw per turn (same as Gen 3+)
- Thaws immediately when hit by Fire-type move or using Scald/Flame Wheel/etc.

## End-of-Turn Order

Showdown processes end-of-turn effects via the "Residual" field event. Effects are sorted by their `onResidualOrder` priority value. There is no single hardcoded sequence — each condition/ability registers its own priority.

### Key onResidualOrder Values (from conditions.ts / moves.ts)

| Order | Sub | Effect |
|-------|-----|--------|
| 1 | — | Weather field effects (upkeep messages) |
| 5 | 2 | Grassy Terrain heal (1/16 HP to grounded Pokémon) |
| 9 | — | Poison/Toxic damage |
| 10 | — | Burn damage (1/16) |
| 13 | — | Salt Cure damage (1/8 or 1/4 for Water/Steel) |
| 27 | 7 | Terrain duration/end |

### Important Clarifications

- **Entry hazards** (Spikes, Stealth Rock, Toxic Spikes, Sticky Web) trigger **on switch-in**, NOT at end of turn
- **Terrain effects** (Electric/Misty/Grassy/Psychic Terrain) are **passive modifiers**, not end-of-turn damage sources:
  - Electric Terrain: prevents Sleep for grounded Pokémon, boosts Electric moves 1.3×
  - Misty Terrain: prevents Status/Confusion for grounded Pokémon, halves Dragon-type damage to grounded Pokémon
  - Grassy Terrain: heals grounded Pokémon 1/16 HP per turn, boosts Grass moves 1.3×
  - Psychic Terrain: blocks priority moves (>0) targeting grounded Pokémon, boosts Psychic moves 1.3×
- **Psychic Terrain does NOT extend itself** — fixed 5 turns (8 with Terrain Extender)
- **Electric Terrain does NOT halve speed or heal Electric types** — it only prevents Sleep and boosts Electric moves

## Implementation Notes

### Gen9Ruleset Class

```typescript
export class Gen9Ruleset extends BaseRuleset {
  readonly name = 'generation-9';
  readonly generation = 9;

  // Only Terastallization allowed
  readonly allowedGimmicks: string[] = ['terastallization'];

  canUseDynamax(): boolean { return false; }
  canMegaEvolve(): boolean { return false; }
  canUseZMove(): boolean { return false; }

  // Snow replaces Hail
  getWeatherEffects(weatherType: string): WeatherEffect[] {
    if (weatherType === 'snow') {
      return [{
        apply: (pokemon) => {
          if (pokemon.hasType('ice')) {
            pokemon.defenseMultiplier *= 1.5;
          }
          // No residual damage
        }
      }];
    }
    return super.getWeatherEffects(weatherType);
  }

  // Tera STAB rules
  calculateStab(attacker: Pokemon, move: Move): number {
    const moveType = move.type;
    const hasCurrentType = attacker.hasType(moveType);
    const hasOriginalType = attacker.getBaseTypes().includes(moveType);
    const isSTAB = hasCurrentType || hasOriginalType;

    let stab = 1.0;
    if (isSTAB) stab = 1.5;

    // Stellar Tera: special one-time boosts
    if (attacker.terastallized === 'stellar') {
      if (!attacker.stellarBoostedTypes.includes(moveType)) {
        stab = isSTAB ? 2.0 : 4915 / 4096; // ~1.2× for non-STAB
        attacker.stellarBoostedTypes.push(moveType);
      }
    } else if (attacker.terastallized === moveType && hasOriginalType) {
      // Tera matches original type — 2× STAB
      stab = 2.0;
    }

    // Adaptability modifies STAB (handled via ModifySTAB event)
    // 1.5→2.0, 2.0→2.25 (NOT simple doubling)

    return stab;
  }
}
```
