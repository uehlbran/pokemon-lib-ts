# Pokémon Gen 2 (Gold/Silver/Crystal) Battle Mechanics Specification

> Types and interfaces reference `@pokemon-lib-ts/core` — see `core/01-entities.md` for canonical definitions.

## 1. Overview

Gen 2 introduced fundamental changes to the core mechanics established in Gen 1, most notably the **Special stat split** into Special Attack and Special Defense, along with a suite of new systems that expanded strategic depth.

### Major Changes from Gen 1

- **Special Stat Split**: The single Special stat is now split into SpA (Special Attack) and SpD (Special Defense)
- **Type Chart Expansion**: Added Dark and Steel types (17 types total), fixing Ghost immunities to Psychic
- **Held Items**: Pokémon can now hold items that provide passive effects during battle
- **Weather Systems**: Rain, Sun, and Sandstorm with turn-based duration and type-specific effects
- **New Status Mechanics**: Sleep duration (1-6 turns), Freeze thaw chance (25/256, ~9.8% per turn), burn damage, badly poisoned resets on switch
- **Breeding System**: First generation enabling Pokémon reproduction with inheritance mechanics
- **Happiness/Friendship**: Stats and move behavior influenced by friendship value
- **Shiny Pokémon**: Rare color variants determined by DV values (not introduced mid-battle, determined at generation)
- **Time of Day**: In-game time affects move power and evolution
- **Pokérus**: Beneficial virus that doubles stat experience (EV) gain — does not affect regular EXP
- **Critical Hit Overhaul**: Changed from Speed-based formula to stage-based (Stage 0 = 17/256 ≈ 6.64%, Stage 1 = 32/256 = 12.5%, Stage 2 = 64/256 = 25%, Stage 3 = 85/256 ≈ 33.2%, Stage 4 = 128/256 = 50%)
- **New Moves & Move Categories**: Thief, Mean Look, Protect, Belly Drum, etc.
- **Stat Experience (EVs)**: Foundation for what becomes modern EV system

## 2. Stat System

### DVs (Determinant Values) vs IVs

Gen 2 uses **DVs (0-15 per stat)**, not IVs (0-31). DVs are determined at generation and are **immutable** during play.

### Stat Formula (Gen 1/2 unified)

For **HP**:
```typescript
HP = floor(((Base + DV) * 2 + floor(ceil(sqrt(StatExp)) / 4)) * Level / 100) + Level + 10
```

For **Other Stats**:
```typescript
Stat = floor(((Base + DV) * 2 + floor(ceil(sqrt(StatExp)) / 4)) * Level / 100) + 5
```

Where:
- `Base` = Pokémon's base stat
- `DV` = 0-15 (gen-specific at creation)
- `StatExp` = Stat Experience points (0-65535 per stat); the square-root term converts raw StatExp to a 0-255 effective value
- `Level` = Current level (1-100)

### HP DV Derivation

The HP DV is not stored independently — it is derived from the other DVs:

```
HP_DV = (Atk_DV & 1) << 3 | (Def_DV & 1) << 2 | (Spd_DV & 1) << 1 | (Spc_DV & 1)
```

Each DV contributes its lowest bit to the HP DV.

### Shiny Determination (Gen 2)

A Pokémon is **Shiny** based on its DVs meeting exact constraints. The requirements are:

- **Defense DV** = 10
- **Speed DV** = 10
- **Special DV** = 10
- **Attack DV** ∈ {2, 3, 6, 7, 10, 11, 14, 15} (values where the lowest bit of Atk_DV matches the lowest bit of Def_DV = 10, i.e., Atk_DV is even... actually the constraint is that the derived HP_DV from these DVs must also be odd, driving the Atk constraint to those 8 values)

These constraints derive from the Defense DV being odd (10 in binary is `1010`, lowest bit = 0 — correction: Def_DV = 10 has lowest bit 0, Speed_DV = 10 lowest bit 0, Spc_DV = 10 lowest bit 0, so HP_DV's bits 2,1,0 are all 0; bit 3 = Atk_DV & 1). The Atk DV values {2, 3, 6, 7, 10, 11, 14, 15} are the 8 of 16 possible values that satisfy the full shiny condition given Def/Spd/Spc all = 10. This yields approximately 1/8192 overall shiny probability.

```typescript
isShiny(pokémon: Pokémon): boolean {
  const { ATK_DV, DEF_DV, SPD_DV, SPC_DV } = pokémon.dvs;
  const AtkValid = [2, 3, 6, 7, 10, 11, 14, 15].includes(ATK_DV);
  return AtkValid && DEF_DV === 10 && SPD_DV === 10 && SPC_DV === 10;
}
```

### Stat Caps

- Base stats cap at 255
- Final stat calculation produces caps naturally via formula
- HP stat formula differs from other stats (see above)

### Gen 2 Specific Changes

1. **Ghost is no longer immune to Normal/Fighting**: In Gen 1, Ghost resisted Poison and Bug, and was immune to Normal and Fighting. Gen 2 removes this immunity; Ghost is still immune to Normal and Fighting is **NOT** blocked by Ghost type.
   - **Actually**: Ghost remains immune to Normal and Fighting in Gen 2. This is unchanged from Gen 1.

2. **Psychic weakness to Dark and Ghost**: Dark type introduced in Gen 2, making Psychic weak to both Dark and Ghost.

3. **Steel type relationships**: Steel is weak to Fire, Fighting, and Ground. Water and Electric do **NOT** hit Steel for 2x in Gen 2 (this changes in later generations). Steel resists Normal, Flying, Rock, Bug, Grass, Psychic, Ice, Dragon, and Steel itself.

## 4. Damage Formula

### Gen 2 Damage Formula

The Gen 2 damage formula is nearly identical to Gen 1, with the key difference being the Special split:

```typescript
function calculateDamage(
  attacker: Pokémon,
  defender: Pokémon,
  move: Move,
  weather: Weather | null,
  isCritical: boolean,
): number {
  const level = attacker.level;

  // Determine if move is Physical or Special based on TYPE (not move category)
  // In Gen 2, type determines category:
  const isPhysicalType = isPhysicalMoveType(move.type);
  const isSpecialType = isSpecialMoveType(move.type);

  // Burn halves the Attack stat BEFORE damage calculation (not a post-modifier)
  let atkStat = isPhysicalType ? attacker.stats.ATK : attacker.stats.SPA;
  if (attacker.status === 'burn' && isPhysicalType) {
    atkStat = Math.floor(atkStat / 2);
  }
  const defStat = isPhysicalType ? defender.stats.DEF : defender.stats.SPD;

  // Base damage calculation — floor at each intermediate step
  let damage = Math.floor(
    (Math.floor(
      (Math.floor(2 * level / 5) + 2) * move.power * atkStat / defStat
    ) / 50) + 2
  );

  // Critical hit multiplier (always 2x in Gen 2)
  if (isCritical) {
    damage = Math.floor(damage * 2);
  }

  // STAB (Same Type Attack Bonus) - 1.5x if attacker shares move type
  const stab = attacker.types.includes(move.type) ? 1.5 : 1.0;
  if (stab !== 1.0) {
    damage = Math.floor(damage * stab);
  }

  // Type effectiveness (0.5x, 1x, 2x) — floor after each application
  const typeMultiplier = getTypeEffectiveness(move.type, defender);
  damage = Math.floor(damage * typeMultiplier);

  // Weather modifier
  const weatherModifier = getWeatherModifier(move.type, weather);
  if (weatherModifier !== 1.0) {
    damage = Math.floor(damage * weatherModifier);
  }

  // Random factor: random(217..255) / 255 — Gen 2 specific range
  const randomRoll = 217 + Math.floor(Math.random() * 39); // 217-255 inclusive
  damage = Math.floor(damage * randomRoll / 255);

  return Math.max(1, damage);
}

function isPhysicalMoveType(type: PokemonType): boolean {
  // Poison is PHYSICAL in Gen 2 (not Special)
  const PHYSICAL_TYPES = ['normal', 'fighting', 'flying', 'poison', 'ground', 'rock', 'bug', 'ghost', 'steel'];
  return PHYSICAL_TYPES.includes(type);
}

function isSpecialMoveType(type: PokemonType): boolean {
  const SPECIAL_TYPES = ['fire', 'water', 'electric', 'grass', 'ice', 'psychic', 'dragon', 'dark'];
  return SPECIAL_TYPES.includes(type);
}

function getWeatherModifier(moveType: PokemonType, weather: Weather | null): number {
  if (!weather) return 1.0;

  if (weather.type === 'rain') {
    if (moveType === 'water') return 1.5;
    if (moveType === 'fire') return 0.5;
  } else if (weather.type === 'sun') {
    if (moveType === 'fire') return 1.5;
    if (moveType === 'water') return 0.5;
  } else if (weather.type === 'sandstorm') {
    // Sandstorm does not modify move damage in Gen 2, only deals damage per turn
  }

  return 1.0;
}
```

### Damage Formula Notes

- **Order of operations**: Base damage (with burn-halved Atk if applicable) → Crit → STAB → Type effectiveness → Weather → Random
- **Caps**: Damage is capped at max HP value; minimum 1 HP
- **Rounding**: `Math.floor()` is applied at every intermediate step — do not accumulate floating point and round once at the end
- **Random factor**: `random(217..255) / 255` (approximately 85.1%–100%), not 85/100–100/100
- **Burn**: Halves the Attack stat value before the damage formula runs; it is not a post-damage multiplier

## 5. Critical Hits

### Critical Stage System

Gen 2 replaced Gen 1's Speed-based crit formula with a **stage-based system**:

```typescript
interface CriticalStage {
  stage: number; // 0, 1, 2, 3, 4
  thresholdOutOf256: number; // exact threshold
}

// Exact Gen 2 crit thresholds (roll < threshold out of 256)
const CRIT_THRESHOLDS: Record<number, number> = {
  0:  17,  // Base: 17/256 ≈ 6.64%
  1:  32,  // High-crit moves: 32/256 = 12.5%
  2:  64,  // Stage 2: 64/256 = 25%
  3:  85,  // Stage 3: 85/256 ≈ 33.2%
  4: 128,  // Stage 4: 128/256 = 50%
};

function isCriticalHit(attacker: Pokémon, move: Move): boolean {
  const moveStageBonus = move.critStage || 0;
  const focusEnergyBonus = attacker.battleStats.focusEnergy ? 1 : 0;
  const totalStage = Math.min(moveStageBonus + focusEnergyBonus, 4);

  const threshold = CRIT_THRESHOLDS[totalStage];
  const roll = Math.floor(Math.random() * 256);
  return roll < threshold;
}
```

### Critical Hit Moves

Certain moves have increased critical hit ratio:
- **Stage 1 (high-crit) moves**: Slash, Karate Chop, Crabhammer, Razor Leaf, Blizzard (and others with boosted crit in Gen 2) — use threshold 32/256
- **Focus Energy**: Grants +1 crit stage (moves user from stage 0 to stage 1 threshold)

### Focus Energy Fix

In Gen 1, Focus Energy **lowered** critical hit chance by mistake. Gen 2 correctly gives +1 crit stage:

```typescript
class FocusEnergy extends Move {
  execute(user: Pokémon, target: Pokémon): void {
    // Gen 2 correct behavior: +1 crit stage (NOT the broken +4 of Gen 1)
    user.battleStats.focusEnergy = true; // flagged; adds +1 stage in isCriticalHit()
  }
}
```

### Crit Stat Interaction

When a critical hit occurs, stat stages interact as follows:
- If the **defender's defense stage ≥ attacker's attack stage**, the crit ignores all stat stages (uses base stats)
- Otherwise, the attacker's positive attack stages still apply normally

### Critical Multiplier

Critical hits deal **exactly 2x damage** in Gen 2 (no variation).

### Held Item Mechanics

- **One item per Pokémon**: Only one held item active at a time
- **Not consumed by default**: Unlike berries consumed by moves (Gen 3+), Gen 2 held items persist
- **Dropped on switch**: Thief move can steal held items (new in Gen 2)
- **Order of application**: Held item effects apply after move damage, before end-of-turn effects

## 7. Weather Systems

### Weather Types

```typescript
enum WeatherType {
  NONE = 'none',
  RAIN = 'rain',         // Rain Dance
  SUN = 'sun',           // Sunny Day
  SANDSTORM = 'sandstorm', // Sandstorm
  // SNOW/HAIL not in Gen 2 (added Gen 3+)
}

interface Weather {
  type: WeatherType;
  turnsRemaining: number;
  setByMove?: MoveId; // Track which move set it
}
```

### Weather Effects

#### Rain
- **Duration**: 5 turns (set by Rain Dance)
- **Move power modifier**: Water-type moves gain +50% power, Fire-type moves deal -50% damage
- **No automatic damage**: Rain doesn't deal residual damage in Gen 2
- **Visual feedback**: Animation and message each turn

#### Sun / Clear Sky
- **Duration**: 5 turns (set by Sunny Day)
- **Move power modifier**: Fire-type moves gain +50% power, Water-type moves deal -50% damage
- **No automatic damage**: Sun doesn't heal status in Gen 2 (that's Gen 5+)

#### Sandstorm
- **Duration**: 5 turns (set by Sandstorm move, new in Gen 2)
- **Automatic damage**: Non-Rock/Ground/Steel Pokémon take 1/8 max HP damage at end of turn
- **Type of damage**: Fixed recoil, applies before status damage (see: End-of-Turn Order)
- **No move modifier**: Sandstorm doesn't modify move power in Gen 2

### Weather Implementation

```typescript
class WeatherSystem {
  currentWeather: Weather | null = null;

  setWeather(type: WeatherType, duration: number = 5): void {
    this.currentWeather = {
      type,
      turnsRemaining: duration,
    };
  }

  decrementWeather(): void {
    if (this.currentWeather) {
      this.currentWeather.turnsRemaining--;
      if (this.currentWeather.turnsRemaining <= 0) {
        this.currentWeather = null;
      }
    }
  }

  applySandstormDamage(pokémon: Pokémon): void {
    if (this.currentWeather?.type !== WeatherType.SANDSTORM) return;

    const isRockGroundOrSteel = pokémon.types.some(t =>
      ['rock', 'ground', 'steel'].includes(t)
    );

    if (!isRockGroundOrSteel) {
      const damage = Math.ceil(pokémon.stats.HP / 8);
      pokémon.takeDamage(damage, 'sandstorm');
    }
  }
}
```

## 8. Status Conditions

### Sleep

- **Duration**: 1-6 turns (chosen randomly when status applied)
- **Wake behavior**: Pokémon wakes up when duration expires OR by Full Heal/Mint Berry/MiracleBerry item
- **Act on wake turn**: In Gen 2, a Pokémon **CAN act on the turn it wakes up** — the wake turn is not wasted (unlike Gen 1 where waking consumed the entire turn)
- **Note**: Wake-Up Slap does NOT exist in Gen 2 — it is introduced in Gen 4

### Paralysis

- **Speed reduction**: Reduces Speed stat to 25% of normal
- **Effect chance**: Some moves (Thunder Wave) paralyze with 100% accuracy in Gen 2 (nerfed in later gens)
- **Cured by**: Paralysis Heal, Full Heal, PRZCureBerry (held item), Full Restore
- **Type immunity**: Electric-type Pokémon **CAN** be paralyzed in Gen 2 — there is no Electric immunity to paralysis in Gen 2

### Poison

#### Regular Poison
- **Damage**: 1/8 max HP per turn
- **Cured by**: Antidote, Full Heal, PSNCureBerry (held item), Full Restore

#### Badly Poisoned (Purple Poison)
- **Damage**: Starts at 1/16 max HP, increases by 1/16 each turn (1/16, 2/16, 3/16, etc.) up to max damage
- **Resets on switch**: Counter resets to 1/16 when switching (NEW in Gen 2; Gen 1 persisted across switches)
- **Mechanics**: Badly poisoned status persists, but damage multiplier increases each turn

```typescript
interface BadlyPoisonedStatus {
  counter: number; // 1, 2, 3, ... up to ~16
  damagePerTurn(): number {
    return Math.ceil((this.counter / 16) * maxHP);
  }
  incrementCounter(): void {
    this.counter = Math.min(this.counter + 1, 15); // Cap at reasonable level
  }
  resetOnSwitch(): void {
    this.counter = 1; // NEW in Gen 2
  }
}
```

### Burn

- **Damage**: 1/8 max HP per turn
- **Attack reduction**: Halves Physical Attack stat (ATK)
- **Cured by**: Full Heal, Full Restore, BRNCureBerry (held item), MiracleBerry (held item)
- **Note**: Burn doesn't reduce Special Attack in Gen 2 (only ATK). Lum Berry does NOT exist in Gen 2 — use BRNCureBerry or MiracleBerry

### Freeze

- **Thaw chance**: 25/256 (~9.8%) chance to thaw each turn at end of turn
- **Immobility**: Frozen Pokémon cannot attack (0% chance to move per turn)
- **Manual thaw**: Can use Full Heal, Full Restore, or other items
- **Weather**: Fire-type moves (attacking) can thaw opponent

### Confusion

- **Duration**: 2-5 turns (random)
- **Chance per turn**: 50% chance to hit self with 40 Base Power Physical attack
- **Damage type**: Counted as Normal-type, not type of move, for STAB/weakness
- **Cured by**: Full Heal, Full Restore, Bitter Berry (held item), MiracleBerry (held item)

## 9. Move Mechanics

### Move Categories by Type

In Gen 2, **moves are categorized as Physical or Special by their TYPE**, not by the individual move:

```typescript
// NOTE: Poison is PHYSICAL in Gen 2 — do not place it in SPECIAL_TYPES
const PHYSICAL_TYPES = ['normal', 'fighting', 'flying', 'poison', 'ground', 'rock', 'bug', 'ghost', 'steel'];
const SPECIAL_TYPES = ['fire', 'water', 'electric', 'grass', 'ice', 'psychic', 'dragon', 'dark'];

function getMoveCategory(move: Move): 'physical' | 'special' {
  if (PHYSICAL_TYPES.includes(move.type)) return 'physical';
  if (SPECIAL_TYPES.includes(move.type)) return 'special';
  throw new Error(`Unknown type for categorization: ${move.type}`);
}
```

### New Moves (Gen 2 Introduction)

#### Protect
- **Effect**: User is protected from attacks this turn; prevents damage and contact-based effects
- **Multi-turn usage**: Success chance = `1 / 3^n` where n is the number of consecutive prior uses. First use always succeeds; second consecutive use has ~33% chance; third has ~11%; etc. (denominator multiplies by 3 each consecutive use)
- **Priority**: +4 priority (highest in game)

#### Thief
- **Effect**: Deals damage and steals opponent's held item (first move to interact with held items)
- **Interaction**: Stolen item is placed into attacker's bag if inventory space; otherwise stays in battle

#### Attract
- **Effect**: Opposite-gender target becomes attracted; 50% chance to fail action each turn
- **Requirement**: Requires opposite gender; same-gender Pokémon immune
- **Interaction**: Physical/Special moves have no distinction here; move just affects gender

#### Mean Look / Spider Web
- **Effect**: Prevents opponent from switching out; traps on switch
- **Duration**: Persists until move user switches out or battle ends

#### Belly Drum
- **Effect**: User maximizes Attack stat (+6 stage) but takes 50% of max HP as recoil damage
- **Risk/Reward**: High risk but guarantees max Attack

#### Pursuit
- **Effect**: Deals damage; if opponent switches, damage is doubled and move goes first

#### Ancientpower
- **Effect**: 10% chance to raise all stats by 1 stage

#### Trapping Moves (Bind, Wrap, Clamp, Fire Spin) — Gen 2 Changes

Gen 2 changed trapping move mechanics significantly from Gen 1:
- **Target CAN attack** while trapped — the trap only prevents switching, not attacking
- **User is NOT locked** into the trapping move (can use other moves while trapping)
- **Duration**: 2-5 turns (random)
- **Damage**: 1/16 max HP per turn (end-of-turn)

#### Thunder and SolarBeam — Weather Interactions (Gen 2)

- **Thunder**: Accuracy is 100% in Rain (always hits); accuracy drops to 50% in Sun
- **SolarBeam**: In Sun, charges and fires in a single turn (no charge turn needed); in Rain, Sand, or Hail, SolarBeam's power is halved (deals 50% damage)

#### Reflect and Light Screen — Duration

- **Duration**: Reflect and Light Screen last exactly **5 turns** in Gen 2
- This is a change from Gen 1, where screens were permanent (lasted until the user switched out or the battle ended)

#### OHKO Moves (Fissure, Horn Drill, Guillotine, Sheer Cold) — Gen 2 Mechanics

Gen 2 changed OHKO from Gen 1's Speed comparison:
- **Level check**: OHKO moves **fail** if the user's level is less than the target's level (not Speed comparison)
- **Base accuracy**: 30% accuracy
- **Level scaling**: Accuracy increases by +1% for each level the user is above the target (so a level 60 user vs level 50 target = 40% accuracy)

#### Counter

In Gen 2, Counter reflects damage from **any physical-type move** (based on the Gen 2 physical/special type classification), not just Normal and Fighting moves as in Gen 1. If the last hit taken was physical-type (by type, not move category), Counter reflects double that damage.

#### Baton Pass

Baton Pass transfers the following from the user to the incoming replacement:
- Stat stage changes (Attack, Defense, Speed, Special Attack, Special Defense, Accuracy, Evasion)
- Substitute (if active)
- Trapped state (Mean Look / Spider Web effect)
- Confusion
- Leech Seed
- Focus Energy
- Curse (if Ghost-type Curse was used)
- Perish Song countdown

### Move Pool Changes

### Breeding System

```typescript
interface BreedingResult {
  egg: Pokémon;
  baseSpecies: PokémonId;
  parentMale: Pokémon;
  parentFemale: Pokémon;
}

class BreedingSystem {
  canBreed(pokémon1: Pokémon, pokémon2: Pokémon): boolean {
    // Both must be opposite genders
    // Both must be in same egg group
    // Neither can be legendary/unbreedable
    return pokémon1.gender !== pokémon2.gender &&
           pokémon1.eggGroup === pokémon2.eggGroup &&
           !pokémon1.isLegendary &&
           !pokémon2.isLegendary;
  }

  breed(male: Pokémon, female: Pokémon): BreedingResult {
    // Egg inherits species from mother, potential moves from father
    const egg = new Pokémon(female.species); // Inherits mother's species
    egg.level = 5; // Starting level
    egg.happiness = 0; // New eggs start with 0 happiness

    // Gen 2 DV inheritance rules (NOT "50% random per stat"):
    // - Attack DV: inherited from one parent chosen at random (50% male, 50% female)
    // - Defense DV: inherited from the OTHER parent (not the same as Attack DV source)
    // - Speed DV: random (0-15), NOT inherited
    // - Special DV: random (0-15), NOT inherited
    // - HP DV: derived from other DVs (Atk_DV & 1) << 3 | (Def_DV & 1) << 2 | (Spd_DV & 1) << 1 | (Spc_DV & 1)
    const atkFromFather = Math.random() < 0.5;
    egg.dvs.ATK = atkFromFather ? male.dvs.ATK : female.dvs.ATK;
    egg.dvs.DEF = atkFromFather ? female.dvs.DEF : male.dvs.DEF; // Opposite parent
    egg.dvs.SPD = Math.floor(Math.random() * 16); // Random
    egg.dvs.SPC = Math.floor(Math.random() * 16); // Random
    egg.dvs.HP = ((egg.dvs.ATK & 1) << 3) | ((egg.dvs.DEF & 1) << 2) |
                 ((egg.dvs.SPD & 1) << 1) | (egg.dvs.SPC & 1);

    return { egg, baseSpecies: female.species, parentMale: male, parentFemale: female };
  }
}
```

## 11. End-of-Turn Order

The exact order of end-of-turn effects is crucial for competitive accuracy. **Leech Seed exists in Gen 2** — prior claims that it does not are incorrect.

### Nightmare and Curse (Ghost) Mechanics

**Nightmare**:
- Inflicted on a sleeping Pokémon via the Nightmare move
- Deals **1/4 max HP** damage per turn while the target remains asleep
- Ends when the target wakes up

**Ghost-type Curse**:
- When a Ghost-type Pokémon uses Curse, it costs the **user 1/2 of their max HP**
- The target is then inflicted with a curse that deals **1/4 max HP** damage per turn at end of turn
- Distinct from the non-Ghost use of Curse (which raises Atk/Def, lowers Speed)

### Correct Gen 2 End-of-Turn Order

```typescript
function executeEndOfTurnEffects(battle: Battle): void {
  // Step 1: Future attack damage (Doom Desire / Future Sight if turn arrived)
  for (const pokémon of battle.allActivePokémon()) {
    applyFutureAttackDamage(pokémon);
  }

  // Step 2: Leftovers recovery (1/16 max HP)
  for (const pokémon of battle.allActivePokémon()) {
    if (pokémon.heldItem?.name === 'Leftovers') {
      const recovery = Math.floor(pokémon.stats.HP / 16); // 1/16, NOT 1/8
      pokémon.currentHP = Math.min(pokémon.currentHP + recovery, pokémon.stats.HP);
    }
  }

  // Step 3: Burn damage (1/8 max HP)
  for (const pokémon of battle.allActivePokémon()) {
    if (pokémon.status === 'burn') {
      pokémon.takeDamage(Math.floor(pokémon.stats.HP / 8), 'burn');
    }
  }

  // Step 4: Poison / Badly Poisoned damage
  for (const pokémon of battle.allActivePokémon()) {
    if (pokémon.status === 'poison') {
      pokémon.takeDamage(Math.floor(pokémon.stats.HP / 8), 'poison');
    }
    if (pokémon.status === 'badly-poisoned') {
      const damage = Math.floor((pokémon.badlyPoisoned.counter / 16) * pokémon.stats.HP);
      pokémon.takeDamage(damage, 'badly-poisoned');
      pokémon.badlyPoisoned.incrementCounter();
    }
  }

  // Step 5: Leech Seed drain (1/8 max HP; Leech Seed exists in Gen 2)
  for (const pokémon of battle.allActivePokémon()) {
    if (pokémon.volatileStatus.leechSeed) {
      const drain = Math.floor(pokémon.stats.HP / 8);
      pokémon.takeDamage(drain, 'leech-seed');
      // Heal the Pokémon that planted Leech Seed
      const seeder = battle.getSeeder(pokémon);
      if (seeder) {
        seeder.currentHP = Math.min(seeder.currentHP + drain, seeder.stats.HP);
      }
    }
  }

  // Step 6: Nightmare damage (1/4 max HP while asleep)
  for (const pokémon of battle.allActivePokémon()) {
    if (pokémon.status === 'sleep' && pokémon.volatileStatus.nightmare) {
      pokémon.takeDamage(Math.floor(pokémon.stats.HP / 4), 'nightmare');
    }
  }

  // Step 7: Ghost-type Curse damage (1/4 max HP)
  for (const pokémon of battle.allActivePokémon()) {
    if (pokémon.volatileStatus.cursed) {
      pokémon.takeDamage(Math.floor(pokémon.stats.HP / 4), 'curse');
    }
  }

  // Step 8: Bind/Wrap/Clamp/Fire Spin damage (1/16 max HP)
  for (const pokémon of battle.allActivePokémon()) {
    if (pokémon.volatileStatus.trapped) {
      pokémon.takeDamage(Math.floor(pokémon.stats.HP / 16), 'bind');
      pokémon.volatileStatus.trapTurnsRemaining--;
      if (pokémon.volatileStatus.trapTurnsRemaining <= 0) {
        pokémon.volatileStatus.trapped = false;
      }
    }
  }

  // Step 9: Sandstorm damage (1/8 max HP for non-Rock/Ground/Steel)
  for (const pokémon of battle.allActivePokémon()) {
    if (battle.weather?.type === 'sandstorm') {
      const isImmune = pokémon.types.some(t => ['rock', 'ground', 'steel'].includes(t));
      if (!isImmune) {
        pokémon.takeDamage(Math.floor(pokémon.stats.HP / 8), 'sandstorm');
      }
    }
  }

  // Step 10: Perish Song countdown
  for (const pokémon of battle.allActivePokémon()) {
    if (pokémon.volatileStatus.perishCount !== undefined) {
      pokémon.volatileStatus.perishCount--;
      if (pokémon.volatileStatus.perishCount <= 0) {
        pokémon.currentHP = 0; // Faint from Perish Song
      }
    }
  }

  // Step 11: Screen countdown (Reflect / Light Screen last 5 turns)
  battle.decrementScreens(); // Reflect and Light Screen each decrement independently

  // Step 12: Weather turn decrement
  battle.weather?.decrement();

  // Step 13: Faint check
  for (const pokémon of battle.allActivePokémon()) {
    if (pokémon.currentHP <= 0) {
      handleFaint(pokémon, battle);
    }
  }
}
```

**Key considerations:**
- **Leftovers is 1/16 max HP**, not 1/8
- **Leech Seed is present in Gen 2** and drains 1/8 max HP per turn
- **Burn and Poison apply in separate phases** (Burn before Poison)
- **Nightmare and Ghost Curse** both deal 1/4 max HP per turn, after Leech Seed
- **Screens last exactly 5 turns** in Gen 2; the countdown happens at end of turn
- **Sandstorm damage** is 1/8 max HP and applies after trapping damage
- **Weather decrements after effects** so weather lasts a full final turn before clearing

### Key Structural Differences

**Gen 1 → Gen 2 Transition**:
- Gen 1: Single `Special` stat
- Gen 2: `Special` splits into `SpA` and `SpD`
- Gen 3+: Move categories change from Type-based to Move-based

**Shared with Gen 1**:
- Stat formula (modified for Special split)
- DV system (not IVs)
- No EVs as modern system (pre-EVs)
- Basic type matchups (17 types added Dark/Steel)

**Changes in Gen 3+**:
- Move categories become individual (Physical/Special per move, not type)
- IVs replace DVs
- EVs become standardized (252/252/4 format)
- Abilities introduced
- New items (Choice Scarf, Life Orb, etc.)
- New weather/terrain systems
