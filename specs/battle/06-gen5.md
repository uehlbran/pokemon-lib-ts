## Overview

Generation 5 (Black/White and Black 2/White 2) introduced several significant mechanical changes that reshaped the competitive metagame:

- **Hidden Abilities**: Third ability slot available through special encounters, creating new strategic options
- **Scaled Experience Formula**: Major change to EXP calculation based on level difference between combatants
- **Team Preview**: Competitive feature allowing players to see opponent's full team before battle
- **Triple Battles**: New 3v3 format with positional mechanics
- **Rotation Battles**: Unique format where 3 Pokémon are available but only 1 is active at a time
- **Gems**: Single-use type-boosting items that provide 1.5x damage multiplier to matching move type
- **Sturdy Rework**: Changed from OHKO immunity to Focus Sash-like mechanic (survive any hit at 1 HP)
- **Weather System**: Still permanent from Abilities (permanent nerf comes in Gen 6)
- **Sleep Clause**: In competitive, only 1 Pokémon per team can be put to sleep
- **Move Priorities**: New priority system with moves like Prankster ability granting +1 priority to status moves

## Hidden Abilities

Hidden Abilities (also called Dream World Abilities) are third ability slots that provide alternative effects to a Pokémon's two standard abilities.

### Mechanics

```typescript
interface Ability {
  name: string;
  isHidden: boolean;
  generation: number; // Generation when ability was introduced
  effect: (context: BattleContext) => void;
}

interface PokemonSpecies {
  abilities: {
    normal1: Ability;
    normal2: Ability | null;
    hidden?: Ability; // Available only in Gen 5+ through Dream World
  };
}
```

### Key Points

- Hidden Abilities are **not** guaranteed on caught Pokémon; they require special encounters (Dream World in BW, Pokémon Dream Radar in B2W2)
- Only one ability can be active at a time—the one the individual Pokémon has
- Hidden Abilities can dramatically change a Pokémon's viability (e.g., Gliscor with Poison Heal)
- When bred, offspring inherit the mother's ability (60% chance for normal ability, 40% for hidden if mother has hidden)
- Hidden Abilities cannot be changed once set—no ability capsule exists in Gen 5

### Integration with Battle Engine

```typescript
class Pokemon {
  private abilitySlot: 'normal1' | 'normal2' | 'hidden';
  private currentAbility: Ability;

  constructor(species: PokemonSpecies, abilitySource: 'normal1' | 'normal2' | 'hidden' = 'normal1') {
    this.abilitySlot = abilitySource;

    if (abilitySource === 'hidden' && !species.abilities.hidden) {
      throw new Error(`Species ${species.name} does not have a hidden ability in Gen 5`);
    }

    this.currentAbility = this.getAbilityBySlot(species, abilitySource);
  }

  private getAbilityBySlot(species: PokemonSpecies, slot: string): Ability {
    if (slot === 'normal1') return species.abilities.normal1;
    if (slot === 'normal2') return species.abilities.normal2 || species.abilities.normal1;
    if (slot === 'hidden') return species.abilities.hidden || species.abilities.normal1;
    return species.abilities.normal1;
  }

  getAbility(): Ability {
    return this.currentAbility;
  }
}
```

## EXP Formula Change

This is the **most significant mechanical change** in Gen 5. The new formula scales experience gained based on the level difference between the winning Pokémon and the defeated Pokémon.

### The New Formula (Gen 5+)

```typescript
// Gen 5 Scaled EXP Formula
function calculateExpGen5(
  baseExp: number,
  ownLevel: number,
  oppLevel: number,
  isBoss: boolean = false, // Boss Pokémon in B2W2 Challenge Mode
  isExpShare: boolean = false,
  isTraded: boolean = false
): number {
  // Base experience yield from the defeated Pokémon species
  let exp = baseExp;

  // Factor in level difference
  // If battling lower-level opponent: gain more EXP (learning is faster)
  // If battling higher-level opponent: gain less EXP (learning is slower)
  const levelFactor = oppLevel / ownLevel;

  // Scale the experience (if level factor is 1.0, no change)
  // Battling level 5 Pokémon with level 50: 5/50 = 0.1x modifier
  // Battling level 50 Pokémon with level 50: 50/50 = 1.0x modifier
  // Battling level 80 Pokémon with level 50: 80/50 = 1.6x modifier
  const scaledExp = Math.floor(baseExp * levelFactor);

  // Base calculation (before multipliers)
  let result = Math.floor((scaledExp * ownLevel) / 5);

  // Trainer battles give 1.5x
  // Trainer battles defined as: NPCs, Gym Leaders, Elite Four, Champion
  const trainerMultiplier = isTrainerBattle ? 1.5 : 1.0;

  // Boss multiplier in B2W2 Challenge Mode
  const bossMultiplier = isBoss ? 1.5 : 1.0;

  // Traded Pokémon get 1.2x bonus
  const tradedMultiplier = isTraded ? 1.2 : 1.0;

  result = Math.floor(result * trainerMultiplier * bossMultiplier * tradedMultiplier);

  // EXP Share holder gets half (if held by different Pokémon)
  if (isExpShare) {
    result = Math.floor(result / 2);
  }

  return Math.max(1, result); // Minimum 1 EXP
}

// Example calculations:
// Defeating level 5 Pokémon with Lv 50 Pokémon (same trainer battle):
//   scaledExp = floor(100 * 5/50) = floor(100 * 0.1) = 10
//   result = floor((10 * 50) / 5) = floor(100) = 100
//   final = floor(100 * 1.5) = 150 EXP

// Defeating level 50 Pokémon with Lv 50 Pokémon (same trainer battle):
//   scaledExp = floor(100 * 50/50) = floor(100 * 1.0) = 100
//   result = floor((100 * 50) / 5) = floor(1000) = 1000
//   final = floor(1000 * 1.5) = 1500 EXP

// Defeating level 80 Pokémon with Lv 50 Pokémon (trainer battle):
//   scaledExp = floor(100 * 80/50) = floor(100 * 1.6) = 160
//   result = floor((160 * 50) / 5) = floor(1600) = 1600
//   final = floor(1600 * 1.5) = 2400 EXP
```

### Impact on Gameplay

- **Lower-level teams train slower** when facing higher-level opponents, forcing grinding against appropriate encounters
- **Over-leveling is penalized**: A level 80 Pokémon gets significantly less EXP from level 10 trainers
- **Balanced teams are encouraged**: Players can't just overlevel one Pokémon and breeze through
- **Late-game scaling**: Trainers in post-game content have higher-level teams to offset the reduced EXP modifier

## Triple Battles

Triple Battles are a 3v3 format where positional mechanics matter significantly.

### Battle Format

```typescript

  private state: TripleBattleState;
  private maxActiveTeam: number = 3;

  constructor(playerTeam: Pokemon[], opponentTeam: Pokemon[]) {
    super();
    // Initialize with first 3 Pokémon in each position
    this.state = {
      player: {
        left: playerTeam[0],
        center: playerTeam[1],
        right: playerTeam[2]
      },
      opponent: {
        left: opponentTeam[0],
        center: opponentTeam[1],
        right: opponentTeam[2]
      }
    };
  }
}
```

### Range Restrictions (Critical Mechanic)

Each position has specific targets it can attack:

```typescript
enum TargetType {
  OPPONENT_LEFT = 'opponent_left',
  OPPONENT_CENTER = 'opponent_center',
  OPPONENT_RIGHT = 'opponent_right',
  ALLY_LEFT = 'ally_left',
  ALLY_CENTER = 'ally_center',
  ALLY_RIGHT = 'ally_right'
}

class MoveTargeting {
  /**
   * Returns valid targets based on attacker position and move type
   *
   * Center Pokémon:
   * - Can attack any opposing position (left, center, right)
   * - Can target allies (left, center, right)
   *
   * Side Pokémon (Left/Right):
   * - Can attack center only
   * - Can attack adjacent side (left can hit right center, right can hit left center)
   * - Can target adjacent ally
   */
  getValidTargets(
    attackerPosition: TripleBattlePosition,
    moveTargetScope: 'single' | 'spread' | 'self'
  ): TargetType[] {
    if (moveTargetScope === 'self') {
      return []; // Self-targeting moves
    }

    if (attackerPosition === TripleBattlePosition.CENTER) {
      // Center can hit anything
      return [
        TargetType.OPPONENT_LEFT,
        TargetType.OPPONENT_CENTER,
        TargetType.OPPONENT_RIGHT,
        TargetType.ALLY_LEFT,
        TargetType.ALLY_CENTER,
        TargetType.ALLY_RIGHT
      ];
    }

    // Side positions (LEFT or RIGHT)
    const isLeftAttacker = attackerPosition === TripleBattlePosition.LEFT;
    const adjacentAlly = isLeftAttacker ? TripleBattlePosition.CENTER : TripleBattlePosition.CENTER;
    const adjacentOpponent = isLeftAttacker ? TargetType.OPPONENT_CENTER : TargetType.OPPONENT_CENTER;

    return [
      adjacentOpponent, // Can hit opponent's center
      isLeftAttacker ? TargetType.OPPONENT_RIGHT : TargetType.OPPONENT_LEFT, // Diagonal
      TargetType.ALLY_CENTER // Can hit adjacent ally
    ];
  }

  /**
   * Spread moves hit differently based on scope
   * - Spread moves from center: hit all opponents OR all allies
   * - Spread moves from side: hit center + adjacent opponent
   */
  isValidSpreadTarget(
    attackerPos: TripleBattlePosition,
    targetPos: TripleBattlePosition,
    isSelfTeam: boolean
  ): boolean {
    if (attackerPos === TripleBattlePosition.CENTER) {
      // Center spread hits all on one team
      return true;
    }

    // Side spread: center and adjacent
    if (targetPos === TripleBattlePosition.CENTER) return true;
    if (attackerPos === TripleBattlePosition.LEFT && targetPos === TripleBattlePosition.RIGHT) return true;
    if (attackerPos === TripleBattlePosition.RIGHT && targetPos === TripleBattlePosition.LEFT) return true;

    return false;
  }
}
```

### Special Triple Battle Rules

- **Switching**: Can switch in a new Pokémon, which enters a specific position
- **Fainted Pokémon**: When a Pokémon faints, positions shift: Right moves to Center, leaving Right empty
- **Full Shift Example**:
  - Start: Left (A), Center (B), Right (C) vs Opponent
  - C faints: Left (A), Center (C), Right (empty)
  - When A faints: Left (empty), Center (C), Right (next sent out)

## Rotation Battles

Rotation Battles are a unique format: 3 Pokémon are sent out, but only 1 is active at a time.

### Battle Format

```typescript
class RotationBattle extends Battle {
  private activeIndex: number = 0;
  private rotationTeam: Pokemon[] = [];

  constructor(playerTeam: Pokemon[], opponentTeam: Pokemon[]) {
    super();
    // Take first 3 Pokémon
    this.rotationTeam = playerTeam.slice(0, 3);
  }

  getActivePokemon(): Pokemon {
    return this.rotationTeam[this.activeIndex];
  }

  /**
   * Rotate to next Pokémon in the rotation
   * Can be done before attacking as a free action
   * Wraps around: 0 -> 1 -> 2 -> 0
   */
  rotate(): void {
    this.activeIndex = (this.activeIndex + 1) % 3;
  }

  /**
   * In Rotation Battles, you can:
   * 1. Attack with current active Pokémon
   * 2. Rotate to next Pokémon (no damage taken during rotation)
   * 3. Switch out entire rotation (if main team Pokémon available)
   *
   * Opponent cannot see which Pokémon you'll use after rotating
   * This creates a mindgame aspect
   */
  isValidAction(action: BattleAction): boolean {
    if (action.type === 'move') {
      return action.pokemonIndex === this.activeIndex;
    }
    if (action.type === 'rotate') {
      return true; // Always valid
    }
    if (action.type === 'switch') {
      // Can switch the entire rotation set
      return true;
    }
    return false;
  }
}
```

### Key Differences from Triple Battles

| Aspect | Triple Battle | Rotation Battle |
|--------|---------------|-----------------|
| Active Pokémon | All 3 at once | 1 active, 2 benched |
| Positioning | Left, Center, Right | N/A (no positioning) |
| Targeting | Multiple options per position | Always hit active |
| Rotation | N/A | Can rotate before attacking |
| Damage During Rotation | N/A | No damage taken |

## Gems

Gems are single-use held items that boost a move's damage by 1.5x if the move's type matches the Gem type.

### Mechanics

```typescript
interface Gem {
  type: PokemonType;
  name: string; // "Psychic Gem", "Water Gem", etc.
  consumed: boolean;
}

enum PokemonType {
  NORMAL = 'normal',
  FIGHTING = 'fighting',
  FLYING = 'flying',
  POISON = 'poison',
  GROUND = 'ground',
  ROCK = 'rock',
  BUG = 'bug',
  GHOST = 'ghost',
  STEEL = 'steel',
  FIRE = 'fire',
  WATER = 'water',
  GRASS = 'grass',
  ELECTRIC = 'electric',
  PSYCHIC = 'psychic',
  ICE = 'ice',
  DRAGON = 'dragon',
  DARK = 'dark'
  // FAIRY removed — Fairy type does not exist in Gen 5 (introduced in Gen 6)
}

> **Note**: Fairy type does not exist in Gen 5. It was introduced in Gen 6. Clefairy, Jigglypuff, Togekiss, etc. are Normal-type in Gen 5.

class GemSystem {
  /**
   * When a Pokémon uses a move that matches its held Gem's type:
   * 1. Damage is multiplied by 1.5x
   * 2. Gem is consumed (lost after use)
   * 3. Cannot use same Gem again this battle
   */
  applyGemDamageBoost(
    pokemon: Pokemon,
    move: Move,
    baseDamage: number
  ): number {
    const gem = pokemon.heldItem as Gem;

    if (!gem || gem.consumed) {
      return baseDamage;
    }

    // Check if move type matches gem type
    if (move.type === gem.type) {
      gem.consumed = true; // Consume the gem
      return Math.floor(baseDamage * 1.5);
    }

    return baseDamage;
  }
}
```

### Gem Types in Gen 5

- All single-type Gems available: Normal, Fighting, Flying, Poison, Ground, Rock, Bug, Ghost, Steel, Fire, Water, Grass, Electric, Psychic, Ice, Dragon, Dark
- Multi-type Gems: None (introduced later in Gen 6+)
- Usage: Most viable on special/physical attackers with single strong move to abuse the 1.5x boost

## Sturdy Update

Sturdy changes from OHKO immunity to a survival mechanic similar to Focus Sash (not released until later).

### Mechanic Change

```typescript
enum Ability {
  STURDY = 'sturdy'
}

class SturdyAbility implements AbilityEffect {
  /**
   * Gen 4 and earlier: Sturdy prevented OHKO moves (Horn Drill, Guillotine, etc.)
   * Gen 5: Sturdy prevents fainting from any attack IF the Pokémon is at full HP
   *
   * Effect: If Sturdy Pokémon is hit by any move while at maximum HP,
   * it survives with 1 HP instead of fainting
   */
  processDamage(
    pokemon: Pokemon,
    incomingDamage: number,
    context: BattleContext
  ): DamageResult {
    if (
      pokemon.getAbility().name === 'Sturdy' &&
      pokemon.currentHP === pokemon.maxHP &&
      incomingDamage >= pokemon.currentHP
    ) {
      // Survive with 1 HP
      return {
        damageDealt: pokemon.currentHP - 1,
        survived: true,
        ability: 'Sturdy'
      };
    }

    return {
      damageDealt: incomingDamage,
      survived: incomingDamage < pokemon.currentHP
    };
  }
}
```

### Key Differences

| Aspect | Gen 4 Sturdy | Gen 5 Sturdy |
|--------|--------------|-------------|
| Effect | Blocks OHKO moves | Survives any hit at full HP |
| Usage | Prevents specific moves | General defensive utility |
| Comparable Item | N/A | Focus Sash (Gen 5+) |

## New Abilities

Generation 5 introduced many powerful new abilities that influenced the metagame.

```typescript
interface AbilityEffect {
  onAbilityActivated(context: BattleContext): void;
  processAction?(action: BattleAction, context: BattleContext): void;

}

class PranksterAbility implements AbilityEffect {
  /**
   * Prankster (Gen 5)
   * All status moves used by this Pokémon gain +1 priority
   * Makes status moves go first
   * Holders: Sableye, Murkrow line
   */

      action.priority = (action.priority || 0) + 1;
    }
  }

  private isStatusMove(move: Move): boolean {
    // Status moves: moves that don't deal damage
    // Examples: Will-O-Wisp, Thunder Wave, Spikes, Reflect
    return move.category === 'status';
  }
}

class MagicBounceAbility implements AbilityEffect {
  /**
   * Magic Bounce (Gen 5)
   * Status moves that hit this Pokémon are reflected back to the user
   * Does NOT bounce damaging moves
   * Holders: Espeon (hidden), Mismagius (B2W2)
   */
  processDamageEvent(event: DamageEvent, context: BattleContext): void {
    if (event.move.category === 'status') {
      event.bounceBack = true;
      event.newTarget = event.source;
    }
  }
}

class MultiscaleAbility implements AbilityEffect {
  /**
   * Multiscale (Gen 5)
   * Reduces damage taken by 50% if the holder is at full HP
   * Broken by any damage brought it below full HP
   * Scales break: 1 per damage instance
   * Holders: Dragonite (hidden)
   */
  private scalesActive: boolean = true;

  processDamage(
    pokemon: Pokemon,
    incomingDamage: number,
    context: BattleContext
  ): DamageResult {
    if (
      pokemon.getAbility().name === 'Multiscale' &&
      this.scalesActive &&
      pokemon.currentHP === pokemon.maxHP
    ) {
      // Reduce damage by 50%
      const reducedDamage = Math.floor(incomingDamage / 2);

      // Break the scales
      this.scalesActive = false;

      return {
        damageDealt: reducedDamage,
        scalesBroken: true
      };
    }

    return { damageDealt: incomingDamage };
  }

  onHeal(pokemon: Pokemon, context: BattleContext): void {
    // Scales restore when Pokémon reaches full HP again
    if (pokemon.currentHP === pokemon.maxHP) {
      this.scalesActive = true;
    }
  }
}

class RegeneraatorAbility implements AbilityEffect {

   * Regenerator (Gen 5)
   * Restores 1/3 of max HP when switched out
   * Holders: Slowbro, Audino, Reuniclus (B2W2)
   */

   */
  onBattleStart(pokemon: Pokemon, context: BattleContext): void {
    // Mark opponent items as disabled
  }
}
```

## Weather System

Weather effects in Gen 5 remain permanent when set by abilities, unlike later generations.

### Weather Mechanics

```typescript
enum WeatherType {
  CLEAR = 'clear',
  RAIN = 'rain',
  HAIL = 'hail',
  SANDSTORM = 'sandstorm',
  HARSH_SUNLIGHT = 'harshsunlight'
}

interface WeatherState {
  type: WeatherType;
  turnsDuration: number; // -1 = infinite/permanent
  abilitySource?: Ability; // Ability that set the weather
  moveSource?: Move; // Move that set the weather
}

class WeatherSystem {
  private activeWeather: WeatherState | null = null;

  /**
   * Gen 5 Weather:
   * - Abilities set permanent weather (lasts entire battle)
   * - Moves set weather for 5 turns
   * - Weather-setting abilities: Drought, Drizzle, Hail, Sand Stream
   * - Weather moves: Rain Dance, Hail, Sunny Day, Sandstorm
   *
   * Weather persists until changed by another ability/move
   * Multiple weather-setting abilities = first one in turn order wins initially
   */
  setWeather(
    weatherType: WeatherType,
    source: 'ability' | 'move',
    sourceEntity: Ability | Move,
    duration: number
  ): void {
    if (source === 'ability') {
      // Abilities set permanent weather (-1 turns)
      this.activeWeather = {
        type: weatherType,
        turnsDuration: -1,
        abilitySource: sourceEntity as Ability
      };
    } else {
      // Moves set weather for 5 turns
      this.activeWeather = {
        type: weatherType,
        turnsDuration: 5,
        moveSource: sourceEntity as Move
      };
    }
  }

  getActiveWeather(): WeatherState | null {
    return this.activeWeather;
  }

  decrementWeatherDuration(): void {
    if (this.activeWeather && this.activeWeather.turnsDuration > 0) {
      this.activeWeather.turnsDuration--;

      if (this.activeWeather.turnsDuration === 0) {
        this.activeWeather = null;
      }
    }
  }

  /**
   * Weather effects in Gen 5:
   *
   * RAIN:
   * - Water moves +50% damage
   * - Fire moves -50% damage
   * - Hydration cures status
   *
   * HARSH SUNLIGHT (Drought):
   * - Fire moves +50% damage
   * - Water moves -50% damage
   * - Solar Power +50% Special Attack (but -1/8 HP per turn)
   *
   * SANDSTORM:
   * - Rock Pokemon +50% Special Defense
   * - Non-Rock/Ground/Steel take 1/8 HP damage per turn
   * - Sand Stream ability setter
   *
   * HAIL:
   * - Non-Ice Pokemon take 1/8 HP damage per turn
   * - Aurora Veil would block damage (Gen 7+)
   */
}
```

### Key Gen 5 Weather Differences

- **Permanent from abilities**: Weather set by Drought, Drizzle, Hail, Sand Stream lasts entire battle
- **5 turns from moves**: Weather moves only last 5 turns
- **No priority switching**: If two abilities both set weather (e.g., Groudon vs Kyogre), first one to act wins
- **Weather moves don't suppress abilities**: Unlike later generations, weather moves don't suppress weather-setting abilities

## Damage Formula

The damage formula in Gen 5 is identical to Gen 4 with the addition of Gem damage multipliers.

```typescript
function calculateDamageGen5(
  attacker: Pokemon,
  defender: Pokemon,
  move: Move,
  context: BattleContext
): number {
  /**
   * Gen 5 Damage Formula:
   * (((2 * Level / 5 + 2) * Power * Attack / Defense) / 50 + 2) * Modifiers
   *
   * Where:
   * - Level: Attacker's level
   * - Power: Move power (0 for non-damaging moves)
   * - Attack: Attacker's Attack or Special Attack stat
   * - Defense: Defender's Defense or Special Defense stat
   * - Modifiers: Various multipliers applied in order
   */

  let damage: number;

  // Determine if physical or special
  const isPhysical = move.category === 'physical';
  const attackStat = isPhysical ? attacker.stats.attack : attacker.stats.spAtk;
  const defenseStat = isPhysical ? defender.stats.defense : defender.stats.spDef;

  // Core damage formula
  damage = Math.floor(
    ((2 * attacker.level / 5 + 2) * move.power * attackStat / defenseStat) / 50 + 2
  );

  // Apply multipliers in order
  let multiplier = 1.0;

  // Weather modifier
  const weather = context.getWeather();
  if (weather?.type === WeatherType.RAIN && move.type === PokemonType.WATER) {
    multiplier *= 1.5; // Rain boosts water
  }
  if (weather?.type === WeatherType.HARSH_SUNLIGHT && move.type === PokemonType.FIRE) {
    multiplier *= 1.5; // Sun boosts fire
  }

  // STAB (Same Type Attack Bonus): 1.5x if move type matches Pokémon type
  if (attacker.types.includes(move.type)) {
    multiplier *= 1.5;
  }

  // Type effectiveness (defender's typing)
  multiplier *= calculateTypeEffectiveness(move.type, defender.types);

  // Ability modifiers
  if (attacker.getAbility().name === 'Adaptability') {
    // Adaptability changes STAB from 1.5x to 2.0x (if STAB applies)
    if (attacker.types.includes(move.type)) {
      multiplier /= 1.5; // Remove normal STAB
      multiplier *= 2.0; // Apply Adaptability STAB
    }
  }

  // Gem modifier (NEW in Gen 5)
  const gem = attacker.heldItem as Gem;
  if (gem && gem.type === move.type && !gem.consumed) {
    multiplier *= 1.5;
    gem.consumed = true;
  }

  // Item modifiers
  if (attacker.heldItem?.name === 'Choice Band' && isPhysical) {
    multiplier *= 1.5;
  }
  if (attacker.heldItem?.name === 'Choice Specs' && !isPhysical) {
    multiplier *= 1.5;
  }

  // Screen effects (Reflect, Light Screen, Aurora Veil)
  if (
    !isPhysical &&
    context.isLightScreenActive(defender.team)
  ) {
    multiplier *= 0.5; // Gen 5 divides by 2
  }
  if (
    isPhysical &&
    context.isReflectActive(defender.team)
  ) {
    multiplier *= 0.5; // Gen 5 divides by 2
  }

  // Critical hits: 1.5x damage (Gen 5+)
  if (context.isCriticalHit) {
    multiplier *= 1.5;
  }

  // Random factor: 0.85 to 1.00 (15% variance)
  const randomFactor = (85 + Math.floor(Math.random() * 16)) / 100;
  multiplier *= randomFactor;

  // Final damage with modifiers
  damage = Math.floor(damage * multiplier);

  // Minimum 1 damage
  return Math.max(1, damage);
}

function calculateTypeEffectiveness(
  moveType: PokemonType,
  defenderTypes: PokemonType[]
): number {
  /**
   * Gen 5 has these type effectiveness relationships:
   * Most are same as Gen 4, but some changes:
   *
   * Bug vs Dark: 1.0x (was 1.5x) - rebalance
   * Dark vs Ghost: Normal effectiveness - Ghost type buff
   *
   * Against 2 types: multiply both
   * Example: Steel/Flying vs Ground attack = 1.0x (neutral) * 4.0x (super effective to steel) = 4.0x
   */
  let effectiveness = 1.0;

  for (const defType of defenderTypes) {
    effectiveness *= getTypeMatchup(moveType, defType);
  }

  return effectiveness;
}

function getTypeMatchup(attackType: PokemonType, defendType: PokemonType): number {
  // Simplified example - full type chart would be much larger
  const typeChart: { [key: string]: { [key: string]: number } } = {
    [PokemonType.NORMAL]: {
      [PokemonType.ROCK]: 0.5,
      [PokemonType.GHOST]: 0.0,
      [PokemonType.STEEL]: 0.5
    },
    [PokemonType.FIRE]: {
      [PokemonType.GRASS]: 2.0,
      [PokemonType.ICE]: 2.0,
      [PokemonType.BUG]: 2.0,
      [PokemonType.STEEL]: 2.0,
      [PokemonType.WATER]: 0.5,
      [PokemonType.GROUND]: 0.5,
      [PokemonType.ROCK]: 0.5,
      [PokemonType.FIRE]: 0.5
    }
    // ... full type chart omitted for brevity
  };

  return typeChart[attackType]?.[defendType] ?? 1.0;
}
```

## Status Changes

Generation 5 modifies status conditions and introduces Sleep Clause for competitive battles.

### Sleep Changes

```typescript
enum StatusCondition {
  NONE = 'none',
  BURN = 'burn',
  FREEZE = 'freeze',
  PARALYSIS = 'paralysis',
  POISON = 'poison',
  BADLY_POISONED = 'badly_poisoned',
  SLEEP = 'sleep'
}

class SleepStatus implements StatusEffect {
  /**
   * Gen 5 Sleep Changes:
   * - Duration: 1-3 turns (Gen 4 was 1-7 turns)
   * - Sleep Clause in competitive: Only 1 Pokémon per team can be asleep
   *   - If one team member is put to sleep, Sleep moves fail on other team members
   *   - Except for moves that cure sleep (Wake-Up Slap always hits)
   */
  private sleepTurnsRemaining: number;

  applyStatus(pokemon: Pokemon, source?: string): boolean {
    // Roll for sleep duration: 1-3 turns in Gen 5
    this.sleepTurnsRemaining = 1 + Math.floor(Math.random() * 3);
    pokemon.status = StatusCondition.SLEEP;
    pokemon.sleepTurnsRemaining = this.sleepTurnsRemaining;
    return true;
  }

  processTurn(pokemon: Pokemon): void {
    if (pokemon.status === StatusCondition.SLEEP) {
      pokemon.sleepTurnsRemaining--;

      if (pokemon.sleepTurnsRemaining <= 0) {
        pokemon.status = StatusCondition.NONE;
      }
    }
  }

  canUseMoveWhileAsleep(): boolean {
    // Sleep prevents move usage (unlike Gen 2)
    // Exception: Wake-Up Slap can be used while asleep
    return false;
  }
}

class SleepClause implements CompetitiveRule {
  /**
   * Sleep Clause (Competitive Rule)
   * Only 1 Pokémon per team can be put to sleep at a time
   *
   * Implementation:
   * - Track if any Pokémon on team is asleep
   * - Prevent sleep-inducing moves if clause violated
   * - Affects: Sleep Powder, Spore, Leech Seed, etc.
   */
  private teamAsleeCount: number = 0;
  private asleepPokemon: Set<Pokemon> = new Set();

  validateSleepInduction(
    move: Move,
    target: Pokemon,
    context: BattleContext
  ): { valid: boolean; reason?: string } {
    if (move.effect !== 'sleep') {
      return { valid: true };
    }

    const opposingTeam = context.getOpposingTeam(target);
    const alreadyAsleep = Array.from(this.asleepPokemon).some(
      p => opposingTeam.includes(p)
    );

    if (alreadyAsleep) {
      return {
        valid: false,
        reason: 'Sleep Clause: Only one Pokémon per team can be put to sleep'
      };
    }

    return { valid: true };
  }

  recordSleep(pokemon: Pokemon): void {
    this.asleepPokemon.add(pokemon);
  }

  recordWakeUp(pokemon: Pokemon): void {
    this.asleepPokemon.delete(pokemon);
  }
}
```

```typescript
   * Badly Poisoned (Toxic):

    }
  }
```

## End-of-Turn Order

The order of effects that trigger at the end of each turn is precise and important for competitive play.

```typescript
class EndOfTurnResolver {
  /**
   * Gen 5 End-of-Turn Order (executed in this sequence):
   *
   * 1. Damage from damage-dealing weather (Sandstorm, Hail)
   *    - Non-Rock/Ground/Steel take 1/8 HP in Sandstorm
   *    - Non-Ice take 1/8 HP in Hail
   *
   * 2. Damage from status conditions (Poison, Burn, Badly Poisoned)
   *    - Poison: 1/8 HP
   *    - Burn: 1/8 HP
   *    - Badly Poisoned (Toxic): 1/8 HP, 2/8 HP, etc. (escalating)
   *
   * 3. Abilities that trigger at end of turn
   *    - Regenerator: Restore 1/3 HP if in battle (note: this doesn't apply in Gen 5,
   *      it only triggers on switch. Included for example of ordering)
   *    - Hydration: Cure status in rain
   *    - Dry Skin: Take 1/8 damage in sun, heal in rain
   *
   * 4. Move effects (Leech Seed, entry hazards damage, etc.)
   *    - Leech Seed: Opponent loses 1/8, user gains 1/4
   *    - Entry hazards (Stealth Rock, Spikes, Toxic Spikes)
   *
   * 5. Item effects
   *    - Leftovers: Heal 1/8 HP
   *    - Black Sludge: Poison type heals 1/8, others take 1/8
   *    - Life Orb: Recoil damage (1/10 of damage dealt)
   *
   * 6. Trick Room, Tailwind, etc. duration decrease
   *    - These screen effects last exactly 5 turns
   *
   * 7. Fainting check and potential battle end
   *    - If any Pokémon faint, remove them from battle
   *    - Check for battle end conditions
   */

  resolveEndOfTurn(battle: Battle): void {
    for (const pokemon of battle.allActivePokemon) {
      // 1. Weather damage
      this.applyWeatherDamage(pokemon, battle);

      // 2. Status damage
      this.applyStatusDamage(pokemon, battle);

      // 3. Ability end-of-turn effects
      this.triggerAbilityEffects(pokemon, battle);

      // 4. Move effect damage (e.g., Leech Seed)
      this.applyMoveDamage(pokemon, battle);

      // 5. Item effects
      this.applyItemEffects(pokemon, battle);

      // 6. Duration decrements
      this.decrementDurations(pokemon, battle);

      // 7. Check for fainting
      if (pokemon.currentHP <= 0) {
        this.faintPokemon(pokemon, battle);
      }
    }
  }

  private applyWeatherDamage(pokemon: Pokemon, battle: Battle): void {
    const weather = battle.getWeather();
    if (!weather) return;

    switch (weather.type) {
      case WeatherType.SANDSTORM:
        if (!pokemon.types.includes(PokemonType.ROCK) &&
            !pokemon.types.includes(PokemonType.GROUND) &&
            !pokemon.types.includes(PokemonType.STEEL)) {
          pokemon.takeDamage(Math.floor(pokemon.maxHP / 8));
        }
        break;

      case WeatherType.HAIL:
        if (!pokemon.types.includes(PokemonType.ICE)) {
          pokemon.takeDamage(Math.floor(pokemon.maxHP / 8));
        }
        break;
    }
  }

  private applyStatusDamage(pokemon: Pokemon, battle: Battle): void {
    switch (pokemon.status) {
      case StatusCondition.POISON:
        pokemon.takeDamage(Math.floor(pokemon.maxHP / 8));
        break;

      case StatusCondition.BURN:
        pokemon.takeDamage(Math.floor(pokemon.maxHP / 8));
        break;

      case StatusCondition.BADLY_POISONED:
        const toxicCounter = pokemon.toxicCounter || 1;
        pokemon.takeDamage(Math.floor(pokemon.maxHP * toxicCounter / 8));
        pokemon.toxicCounter = (pokemon.toxicCounter || 0) + 1;
        break;
    }
  }

  private triggerAbilityEffects(pokemon: Pokemon, battle: Battle): void {
    const ability = pokemon.getAbility();

    // Dry Skin: takes damage in sun, heals in rain
    if (ability.name === 'Dry Skin') {
      const weather = battle.getWeather();
      if (weather?.type === WeatherType.HARSH_SUNLIGHT) {
        pokemon.takeDamage(Math.floor(pokemon.maxHP / 8));
      }
      if (weather?.type === WeatherType.RAIN) {
        pokemon.heal(Math.floor(pokemon.maxHP / 8));
      }
    }
  }

  private applyMoveDamage(pokemon: Pokemon, battle: Battle): void {
    // Leech Seed: opponent loses 1/8, user gains 1/4
    if (pokemon.hasVolatileStatus('leechseed')) {
      const damage = Math.floor(pokemon.maxHP / 8);
      pokemon.takeDamage(damage);

      // Find who has Leech Seed active and heal them
      const leechSeedUser = battle.getOpposingSide(pokemon.team)[0];
      if (leechSeedUser) {
        leechSeedUser.heal(Math.floor(pokemon.maxHP / 4));
      }
    }
  }

  private applyItemEffects(pokemon: Pokemon, battle: Battle): void {
    const item = pokemon.heldItem;
    if (!item) return;

    if (item.name === 'Leftovers') {
      pokemon.heal(Math.floor(pokemon.maxHP / 8));
    }
    if (item.name === 'Black Sludge') {
      if (pokemon.types.includes(PokemonType.POISON)) {
        pokemon.heal(Math.floor(pokemon.maxHP / 8));
      } else {
        pokemon.takeDamage(Math.floor(pokemon.maxHP / 8));
      }
    }
  }

  private decrementDurations(pokemon: Pokemon, battle: Battle): void {
    // Trick Room duration
    if (battle.isTrickRoomActive()) {
      battle.decrementTrickRoomDuration();
    }

    // Tailwind duration (not in Gen 5 but example)
    // Reflect, Light Screen, etc.
  }

  private faintPokemon(pokemon: Pokemon, battle: Battle): void {
    pokemon.faint();
    battle.recordFaint(pokemon);
  }
}
```

```typescript
/**
 * Gen5Ruleset extends BaseRuleset and introduces:
 * - Hidden Abilities system
 * - Scaled EXP formula
 * - Triple/Rotation battles
 * - Team Preview
 * - Gem items
 * - Sturdy rework
 * - Sleep Clause
 */
class Gen5Ruleset extends BaseRuleset {
  protected gen: number = 5;
  protected name: string = 'Generation 5 (Black/White/B2W2)';

  /**
   * Key overrides from BaseRuleset (Gen 3):
   */

  // 1. Experience calculation - overridden completely
  override calculateExperience(
    baseExp: number,
    ownLevel: number,
    oppLevel: number,
    context: BattleContext
  ): number {
    return calculateExpGen5(baseExp, ownLevel, oppLevel, context.isBoss, context.isExpShare, context.isTraded);
  }

  // 2. Damage formula - adds Gem modifier
  override calculateDamage(
    attacker: Pokemon,
    defender: Pokemon,
    move: Move,
    context: BattleContext
  ): number {
    return calculateDamageGen5(attacker, defender, move, context);
  }

  // 3. Status application - adds Sleep Clause checking
  override applyStatus(
    pokemon: Pokemon,
    status: StatusCondition,
    context: BattleContext
  ): boolean {
    if (status === StatusCondition.SLEEP && this.isSleepClauseActive()) {
      if (this.sleepClauseViolated(pokemon)) {
        return false; // Sleep move fails
      }
    }
    return super.applyStatus(pokemon, status, context);
  }

  // 4. Ability processing - includes Hidden Abilities
  override initializePokemonAbility(pokemon: Pokemon): void {
    // Check if this Gen 5+ has hidden ability
    if (pokemon.abilitySlot === 'hidden') {
      if (!pokemon.species.abilities.hidden) {
        throw new Error(`Hidden ability not available for ${pokemon.species.name} in Gen 5`);
      }
    }
    super.initializePokemonAbility(pokemon);
  }

  // 5. Sturdy rework from OHKO immunity to survival mechanic
  override processDamageWithAbility(
    pokemon: Pokemon,
    incomingDamage: number,
    attacker: Pokemon,
    move: Move,
    context: BattleContext
  ): DamageResult {
    if (pokemon.getAbility().name === 'Sturdy') {
      // Gen 5+: Sturdy survives any hit at full HP
      if (pokemon.currentHP === pokemon.maxHP && incomingDamage >= pokemon.currentHP) {
        return {
          damageDealt: pokemon.maxHP - 1,
          survived: true,
          message: 'Sturdy protected it!'
        };
      }
    }
    return super.processDamageWithAbility(pokemon, incomingDamage, attacker, move, context);
  }

  // 6. Weather system - permanent from abilities
  override setWeatherFromAbility(ability: Ability, context: BattleContext): void {
    // Abilities set permanent weather in Gen 5 (5 turn nerf is Gen 6+)
    let weatherType: WeatherType | null = null;

    switch (ability.name) {
      case 'Drought':
        weatherType = WeatherType.HARSH_SUNLIGHT;
        break;
      case 'Drizzle':
        weatherType = WeatherType.RAIN;
        break;
      case 'Hail':
        weatherType = WeatherType.HAIL;
        break;
      case 'Sand Stream':
        weatherType = WeatherType.SANDSTORM;
        break;
    }

    if (weatherType) {
      context.setWeather(weatherType, -1); // -1 = permanent
    }
  }

  // 7. Battle format support
  override getSupportedBattleFormats(): BattleFormat[] {
    return [
      BattleFormat.SINGLES,
      BattleFormat.DOUBLES,
      BattleFormat.TRIPLE,
      BattleFormat.ROTATION,
      // Note: Multi Battles exist but are trainer-only
    ];
  }

  // 8. Team Preview for competitive
  isSupportedForTeamPreview(): boolean {
    return true;
  }

  // 9. Gem item support
  override getHeldItemBonuses(pokemon: Pokemon, move: Move): { multiplier: number; consumed?: boolean } {
    const item = pokemon.heldItem;
    if (item && item.type === 'gem' && (item as Gem).type === move.type) {
      (item as Gem).consumed = true;
      return { multiplier: 1.5, consumed: true };
    }
    return super.getHeldItemBonuses(pokemon, move);
  }

  // 10. Sleep Clause implementation
  private isSleepClauseActive(): boolean {
    // Only active in competitive/rated formats
    return this.competitiveMode === true;
  }

  private sleepClauseViolated(pokemon: Pokemon): boolean {
    const team = pokemon.team;
    return team.some(p => p !== pokemon && p.status === StatusCondition.SLEEP);
  }
}
```

### Notes on Implementation

1. **Experience Scaling**: The scaled EXP formula is a radical departure from Gen 4. It requires careful testing to ensure enemy trainers are appropriately leveled for the expected EXP gain.

2. **Hidden Abilities**: Track the source of a Pokémon's ability carefully. Dream World encounters in BW and Pokémon Dream Radar in B2W2 are special mechanics.

3. **Triple/Rotation Battles**: The positioning system in Triple Battles is critical for move targeting. Implement careful validation.

4. **Team Preview**: Must prevent infinite information loops—once locked in, teams are fixed.

5. **Gems**: Implement as one-time consumable items. Track consumption state carefully.

6. **Sturdy Rework**: This is the first generation where Sturdy acts as a survival mechanism, presaging Focus Sash.

7. **Sleep Clause**: Only active in competitive formats (rated battles, tournaments). Allow Pokédex entries to sleep in single-player.

8. **Weather Permanence**: A key difference from Gen 6+. Permanent weather from abilities is intentional in Gen 5.
