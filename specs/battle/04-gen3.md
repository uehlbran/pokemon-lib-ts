## Overview

Generation 3 is the **modernization generation** that established the framework used by all subsequent generations (4-9). This is the BASE generation upon which `BaseRuleset` is implemented. All core mechanics introduced here become the default foundation that later generations only modify or extend.

### Major Introductions

- **Abilities**: The first defining mechanic beyond species and moves. Every Pokémon now has an ability that affects battle behavior, item interaction, and team synergy.
- **Natures**: 25 different natures replacing IVs as the stat variation mechanism. 20 provide asymmetric stat boosts/reductions; 5 are neutral.
- **IV Overhaul**: Individual Values (IVs) expanded from 0-15 (dependent on DVs) to an independent 0-31 range. IVs now affect ability power distribution.
- **EV System Rework**: Effort Values completely restructured—252 points per stat maximum, 510 total per Pokémon. This becomes the standard cap forever.
- **Stat Formula Modernization**: New HP and stat formulas that replace the outdated Gen 1-2 system. This formula persists through Gen 9.
- **Double Battles**: New battle format with 2v2 positioning, spread move mechanics, targeting rules, and support moves like Protect.
- **Weather System**: Permanent weather from abilities (Drizzle, Drought, Sand Stream) vs. 5-turn weather from moves. New weather types: Hail.
- **Critical Hit System**: Normalized crit stage table (Stage 0-4+) with 1/16, 1/8, 1/4, 1/3, 1/2 rates and consistent 2x multiplier.

## 1. Stat System

### Stat Formula (Gen 3+, the "Modern" Era)

This formula is used by `BaseRuleset` and inherited by all later generations unless overridden.

<!-- VERIFY: Charizard Lv50, all 31 IVs, 0 EVs, Neutral nature: HP=153, Atk=104, Def=98, SpA=156, SpD=106, Spe=155. Showdown confirms. -->

#### HP Stat Formula
```typescript
// Gen 3+ HP formula
HP = floor(((2 * Base + IV + floor(EV / 4)) * Level) / 100) + Level + 10
```

**Parameters**:
- `Base` = species base HP stat (e.g., Charizard = 78)
- `IV` = Individual Value (0–31)
- `EV` = Effort Value (0–252, floor-divided by 4)
- `Level` = Pokémon level (1–100)

**Example** (Charizard, Base HP=78, IV=31, EV=0, Level=50):
```
floor(((2*78 + 31 + 0) * 50) / 100) + 50 + 10
= floor((187 * 50) / 100) + 60
= floor(9350 / 100) + 60
= 93 + 60 = 153 ✓
```

#### Other Stats (Atk, Def, SpA, SpD, Spe)
```typescript
// Gen 3+ non-HP stat formula
Stat = floor((floor(((2 * Base + IV + floor(EV / 4)) * Level) / 100) + 5) * NatureMod)
```

**Parameters**: Same as HP, plus:
- `NatureMod` = 1.1 (boosting nature), 0.9 (reducing nature), or 1.0 (neutral)

**Example** (Charizard, Base Atk=84, IV=31, EV=0, Level=50, neutral nature):
```
floor((floor(((2*84 + 31 + 0) * 50) / 100) + 5) * 1.0)
= floor((floor(199 * 50 / 100) + 5))
= floor(99 + 5)
= 104 ✓
```

### IV Range
- **Range**: 0–31 per IV
- **Calculation from Seed**: In-game, IVs are randomly generated; for mechanics, treat as fixed values.
- **Default (if unspecified)**: 31 across all stats (perfect IVs).

### EV System
- **Per-Stat Cap**: 252 Effort Values maximum per stat
- **Total Cap**: 510 Effort Values total across all stats
- **EV Yield**: Pokémon defeated yield EVs (e.g., Machoke yields 2 Attack EVs). Total EV gains in a battle depend on opponent array.
- **Reset**: No in-game EV reset in Gen 3 (Berries are introduced for this purpose later).

### Nature System

**25 Total Natures** (5 groups of 5):
- **Neutral** (5 natures): Hardy, Docile, Quirky, Serious, Bashful
  - No stat modification.
  - Ability Synchronize copies neutral natures to opponent if active (Gen 3 only).

- **Atk/Def** (5 natures):
  - Lonely: +Atk / -Def
  - Brave: +Atk / -Spe
  - Adamant: +Atk / -SpA
  - Naughty: +Atk / -SpD

- **Def/Atk** (5 natures):
  - Bold: +Def / -Atk
  - Relaxed: +Def / -Spe
  - Impish: +Def / -SpA
  - Lax: +Def / -SpD

- **SpA/Atk** (5 natures):
  - Modest: +SpA / -Atk
  - Mild: +SpA / -Def
  - Rash: +SpA / -SpD
  - Quiet: +SpA / -Spe

- **SpD/Atk** (5 natures):
  - Calm: +SpD / -Atk
  - Gentle: +SpD / -Def
  - Careful: +SpD / -Spe
  - Sassy: +SpD / -SpA

- **Spe/Atk** (5 natures):
  - Timid: +Spe / -Atk
  - Hasty: +Spe / -Def
  - Jolly: +Spe / -SpA
  - Naive: +Spe / -SpD

Nature modifiers apply a **1.1x multiplier to boosted stat** and **0.9x multiplier to reduced stat** (after IV/EV calculation).

### Example Calculation

Pokémon: Salamence, Level 100, Perfect IVs (31), 252 Atk EV, Adamant Nature (+Atk / -SpA)

```
Atk = floor((floor(((2*135 + 31 + floor(252/4)) * 100) / 100) + 5) * 1.1)
```

## 3. Damage Formula

### Core Formula

```typescript
Damage = ((2 * Level / 5 + 2) * Power * Atk / Def / 50 + 2) * Modifiers
```

Where:
- **Level**: Attacker's level
- **Power**: Move's base power
- **Atk**: Attacker's Attack stat (or Sp. Atk for special moves)
- **Def**: Defender's Defense stat (or Sp. Def for special moves)
- **Modifiers**: Applied in strict order (see below)

### Modifier Order (Strictly Enforced)

1. **Type Effectiveness** (2x super-effective, 0.5x resistant, 0.25x double-resistant, etc.)
   - Order matters: apply all type modifiers in single multiplication.

2. **Weather** (if applicable):
   - Rain: Water moves 1.5x, Fire moves 0.5x
   - Sun: Fire moves 1.5x, Water moves 0.5x
   - Sand: Rock-type Pokémon's Special Defense boosted by 50% (does NOT boost Rock/Ground/Steel move power)
   - Hail: No move power boost (does NOT boost Ice move power)

3. **Item Modifier** (Choice Band effect, Assault Vest equivalent):
   - Choice Specs/Band-like items: 1.5x attack power
   - STAB is calculated separately (see below)

4. **STAB (Same Type Attack Bonus)**: 1.5x if attacker's type matches move type

5. **Critical Hit**: 2x multiplier if crit succeeds

6. **Spread Move Penalty** (Double Battles): 0.5x if move targets multiple Pokémon

7. **Ability Modifiers**:
   - Adaptability: 2x STAB (so 1.5x → 2x if STAB, 1.0x otherwise)
   - Flash Fire: 1.5x Fire-type moves (after activation)
   - Technician: 1.5x moves with 60 BP or less
   - Pure Power/Huge Power: 2x physical attack

8. **Randomness**: 0.85–1.0 damage roll (random factor in games, but library should allow deterministic or random)

### Special Move Interactions

- **Fixed Damage Moves** (Seismic Toss, Night Shade): Damage = Level (ignores other calculations)
- **OHK Moves** (Horn Drill, Guillotine): Instant KO if it hits; Sturdy (Gen 3) prevents these
- **Percentage-based Moves** (Dragon Rage = 40 damage flat, not percentage in Gen 3)

### Example Damage Calculation

Salamence (Level 100, Atk 234) vs. Gyarados (Def 200) with Earthquake (Power 100, Ground-type, STAB):

```
Base Damage = ((2 * 100 / 5 + 2) * 100 * 234 / 200 / 50 + 2) * Modifiers
            = ((40 + 2) * 100 * 234 / 200 / 50 + 2)
            = (42 * 100 * 234 / 200 / 50 + 2)
            = (42 * 23400 / 200 / 50 + 2)
            = (42 * 117 / 50 + 2)
            = (4914 / 50 + 2)
            = 98.28 + 2
            = 100.28 → 100

// With STAB (1.5x) and no other modifiers:
Final Damage = 100 * 1.5 = 150

// Random roll (0.85–1.0):
Final Range: 127–150 damage
```

## 4. Abilities

Abilities are the core identity of Gen 3 and represent the first major mechanic layer beyond moves and stats. They are mandatory; every Pokémon has exactly one ability (no hidden abilities in Gen 3).

### Ability Integration

**Trigger Types**:
1. **Passive/Permanent**: Always active (e.g., Levitate, Wonder Guard)
2. **Switchin-based**: Activate upon entry (e.g., Intimidate, Trace)
3. **Turn-based**: Activate at specific turn events (e.g., Regenerator, Speed Boost)
4. **Weather/Status-based**: Trigger on field conditions (e.g., Rain Dish, Dry Skin)
5. **Damage/Interaction-based**: Trigger on taking/dealing damage (e.g., Static, Flame Body, Rough Skin)

### Core Gen 3 Abilities

#### Tier 0: Broken Abilities (Presence-defining)

**Intimidate** (Onix, Gyarados, Salamence, Arcanine, etc.)
- **Effect**: Upon switch-in, lowers the opponent's Attack stat by 1 stage (-10% multiplier).
- **Stacking**: Multiple Intimidate effects from allied Pokémon do NOT stack (single check per switch).
- **Interaction**: Ignores Shields (no protection moves yet).
- **Trigger**: Switchin event, before any move execution.

**Levitate** (Gengar, Flygon, Bronzong, etc.)
- **Effect**: Pokémon ignores Ground-type damage, including Earthquake from allies in double battles.
- **Interaction**: Removes immunity to Ground moves (e.g., Thousand Waves in later gens apply only in Gen 5+).
- **Switchin**: No stat change, purely damage reduction.

**Wonder Guard** (Shedinja)
- **Effect**: Takes damage only from super-effective moves (4x effective is not special).
- **Vulnerability**: Susceptible to entry hazards (Spikes, Stealth Rock), weather damage, status damage.
- **Interaction**: Overridden if Pokémon gains a different ability (Multitype, etc.).

**Drizzle** (Kyogre)
- **Effect**: Summons permanent rain (no turn limit) upon switch-in.
- **Interaction**: Overrides other permanent weather if summoned after another weather ability.
- **Persistence**: Lasts until switch-out or weather-changing ability overwrites it.

**Drought** (Groudon)
- **Effect**: Summons permanent sun upon switch-in.
- **Interaction**: Same priority as Drizzle (later switch-in determines field condition).

**Sand Stream** (Tyranitar)
- **Effect**: Summons permanent sandstorm upon switch-in.
- **Interaction**: Boosts Rock-type Pokémon's Special Defense by 50%; deals 1/16 max HP damage per turn to all non-Rock/Ground/Steel Pokémon. Does NOT boost Rock/Ground/Steel move power.

#### Tier 1: Common/Utility Abilities

**Trace** (Porygon, Alakazam, Natu)
- **Effect**: Copies the opponent's ability upon switch-in.
- **Limitation**: Cannot copy certain abilities (e.g., Trace itself leads to infinite loop prevention; implementation should forbid copying Trace).
- **Interaction**: Applies before stat stages, so if Intimidate is copied, Intimidate applies.

**Pressure** (Suicune, Articuno, Moltres, Zapdos)
- **Effect**: Opponent's move costs 2 PP per use instead of 1 when targeting this Pokémon.
- **Stacking**: Does NOT stack if both Pokémon have Pressure.
- **Interaction**: Client-side confirmation required for accurate PP tracking.

**Sturdy** (Gen 3 version: Onix, Golem, Rhyhorn, etc.)
- **Effect**: Immune to OHKO moves (Horn Drill, Guillotine, Spikes, Sheer Cold).
- **Note**: Gen 3 Sturdy ≠ Gen 5+ Sturdy (which grants immunity to 1-hit KO moves and priority moves as Block effect).

**Static** (Manectric, Raichu, Ampharos)
- **Effect**: 30% chance to paralyze an attacker that makes contact.
- **Trigger**: Activates after damage is dealt.
- **Interaction**: Does not affect moves that do NOT make contact (e.g., Thunderbolt).

**Flame Body** (Rapidash, Magcargo, Houndoom)
- **Effect**: 30% chance to burn an attacker that makes contact.
- **Trigger**: Same as Static.

**Synchronize** (Espeon, Umbreon, Ralts, Gardevoir)
- **Effect**: If this Pokémon is burned, frozen, paralyzed, or poisoned, so is the opponent.
- **Scope**: Affects the active opponent only (not switched-in replacement).
- **Neutral Nature Copy** (Gen 3 only): Copied nature is sent to opponent if your nature is neutral (Gen 4+ removes this).

**Liquid Ooze** (Muk, Weezing)
- **Effect**: Reverses draining effect: opponent recovers 25% of damage dealt, this Pokémon heals instead.
- **Example**: If opponent uses Drain Punch, opponent loses HP, this Pokémon gains it.

**Speed Boost** (Blaziken, Vivillon)
- **Effect**: Speed stage increases by 1 at end of turn (NOT introduced until Gen 5, listed here for reference only).

**Rain Dish** (Squirtle line, Lotad)
- **Effect**: Recovers 1/8 Max HP at end of turn while raining.
- **Interaction**: Does NOT activate in Gen 3 sandstorm, only rain.

**Dry Skin** (Bulbasaur line, Shedinja)
- **Effect**: Takes 1/8 Max HP Water damage each turn; takes 1/4 Max HP from Fire moves; recovers 1/8 Max HP from Water moves.
- **Interaction**: Weather-dependent (not rain-specific).

**Water Absorb** (Poliwag, Lapras, Vaporeon)
- **Effect**: Recovers 1/4 Max HP when hit by a Water move (instead of taking damage).
- **Interaction**: Single-target and spread moves both apply.

**Magnet Pull** (Magnemite, Magneton)
- **Effect**: Steel-type Pokémon cannot switch out (forced to stay).
- **Interaction**: Does NOT apply if the Steel-type Pokémon switches before Magnet Pull is activated.

**Motor Drive** (Jolteon)
- **Effect**: When hit by an Electric move, raises Speed by 1 stage and ignores damage.
- **Interaction**: Applies even if the Electric move would do no damage otherwise.

**Flash Fire** (Vulpix, Growlithe, Entei)
- **Effect**: When hit by a Fire move, ignores damage and raises Fire move power by 1.5x until switch-out.
- **Activation**: Persists through turn cycles; resets upon switch-out.

## 5. Double Battles

### Mechanics

Double Battles (2v2, 2v1) introduce new targeting rules and positioning.

#### Targeting Rules
- **Default**: Attacks target a specific opponent (adjacent opponent in double, single opponent in singles).
- **Spread Moves**: Hit both opponents (Earthquake, Blizzard, etc.).
  - **Damage Penalty**: 0.5x damage per target in Gen 3 (vs. 0.75x later).
  - **Example**: Earthquake in a 1v2 scenario deals 0.5x damage to each opponent.

- **Self/Partner**: Moves like Reflect or Growl affect the user's team.
- **Partner-only**: Future moves like Follow Me (Gen 4+) redirect attacks; not present in Gen 3.

#### Positioning
- **Ally/Enemy**: Treated as distinct battle positions.
- **Speed Priority**: Same as singles; higher Speed acts first (no priority-based ordering yet).

#### Key Support Moves (Gen 3 Introductions)

**Protect** (Alakazam, Blissey, Gengar):
- **Effect**: Prevents all damage for one turn (move targets self).
- **Interaction**: Can be used consecutively with diminishing success (33% → 25% → 17% → 11% on 2nd+ uses).
- **Application**: Critical for double battle pivoting and stall strategies.

**Feint** (not in Gen 3; Gen 4+):
- **Replaced by**: Double moves like Double Hit, Double Kick exist but no Feint mechanic.

### Field Effects
- **Stealth Rock** (not in Gen 3; Gen 4+)
- **Trick Room** (not in Gen 3; Gen 5+)
- **Terrain** (not in Gen 3; Gen 5+)

## 6. Weather System

### Weather Types & Effects

#### Rain
- **Source**: Ability (Drizzle) or move (Rain Dance, 5 turns).
- **Duration**: Permanent (abilities) / 5 turns (moves).
- **Effects**:
  - Water moves deal 1.5x damage.
  - Fire moves deal 0.5x damage.
  - Thunder has 100% accuracy (100% hit rate; normally 70%).
  - Moves dependent on rain (e.g., Swift Swim, increases Speed 2x).

#### Sun
- **Source**: Ability (Drought) or move (Sunny Day, 5 turns).
- **Duration**: Permanent (abilities) / 5 turns (moves).
- **Effects**:
  - Fire moves deal 1.5x damage.
  - Water moves deal 0.5x damage.
  - Solar Beam requires 1 turn charging (no charge in sunlight; acts immediately).

#### Sandstorm
- **Source**: Ability (Sand Stream) or move (Sandstorm, 5 turns).
- **Duration**: Permanent (abilities) / 5 turns (moves).
- **Effects**:
  - Deals 1/16 max HP damage per turn to all non-Rock, non-Ground, non-Steel Pokémon.
  - Boosts Rock-type Pokémon's Special Defense by 50%.
  - Does NOT boost Rock/Ground/Steel move power (that is not a Gen 3 mechanic).
  - Weather Ball is Rock-type in sandstorm.

#### Hail (New in Gen 3)
- **Source**: Move (Hail, 5 turns) only. No ability in Gen 3 (Snow Warning is Gen 4+).
- **Duration**: 5 turns.
- **Effects**:
  - Deals 1/16 max HP damage per turn to all non-Ice Pokémon.
  - Does NOT boost Ice move power (that is not a Gen 3 mechanic).
  - Blizzard does NOT have 100% accuracy in Hail in Gen 3 (that mechanic is Gen 4+).
  - Weather Ball is Ice-type in hail.

### Weather Priority
- **Overwrite**: Later weather overwrites earlier weather (last switched-in weather ability determines field).
- **Moves vs. Abilities**: Ability weather is permanent; move weather lasts 5 turns and resets if another weather is set (including re-rain).

### Duration Mechanics
```typescript
// Permanent Weather (Ability)
weather = "rain" // lasts until switched out or overridden

// Temporary Weather (Move)
weatherTurns = 5
// Each turn, decrement. At 0, clear weather.
```

## 7. Critical Hits

### Stage System

| Crit Stage | Probability | Multiplier |
|---|---|---|
| 0 | 1/16 | 2x |
| 1 | 1/8 | 2x |
| 2 | 1/4 | 2x |
| 3 | 1/3 | 2x |
| 4+ | 1/2 | 2x |

### Crit Stage Modifiers
- **High Crit Moves** (e.g., Slash, Psychic, Scope Lens): Start at Stage 1 (1/8 chance).
- **Lansat Berry** (Gen 4+ item): Increases crit stage by 1 (not in Gen 3).
- **Focus Energy**: Increases crit stage by 1 (lasts 1 turn).

### Calculation

```typescript
const critStages = [
  1/16, // Stage 0
  1/8,  // Stage 1
  1/4,  // Stage 2
  1/3,  // Stage 3
  1/2   // Stage 4+
];

const critProb = critStages[Math.min(stage, 4)];
const roll = Math.random();
const isCrit = roll < critProb;
```

### Crit Effects
- **Damage**: 2x multiplier applied after all other modifiers.
- **Stat Drops**: Crits ignore opponent's stat boosts (always use base stat for damage calc).
- **Abilities/Items**: Abilities that reduce damage (e.g., Filter) do NOT apply to crits.

## 8. Experience & Leveling

### EXP Formula (Gen 3: Flat)

```typescript
BaseEXP = PokémonBaseEXP // (e.g., Salamence = 300)
LevelDifference = AttackerLevel - DefenderLevel

EXP = (BaseEXP * LevelDifference / 7) + 1
// Minimum 1, no caps
```

### Special Cases
- **Trade**: Experience gain is 1.5x after trade (applies once).
- **Exp Share**: Distributed equally to all Pokémon holding Exp Share item (not in Gen 3, introduced later).
- **Critical Hits**: No bonus or penalty to EXP yield.

### Leveling Speed
- No tiered experience rates (fast/medium/slow in later gens). All Pokémon use the same rate.

## 9. Status Conditions

### Non-Volatile Status

| Status | Cause | Effect | Cure |
|---|---|---|---|
| Burn | Fire move, Ability (Flame Body) | 50% damage each turn, -50% Attack | Full Restore, Switch to healing item |
| Paralysis | Thunder Wave, Static ability | 25% chance to freeze action, -25% Speed | Full Restore |
| Poison | Poison move, Poison ability | 12.5% damage each turn (badly poisoned in Gen 2) | Full Restore, Antidote item |
| Sleep | Sing, Spore, Sleep Powder | Unable to act for 1-3 turns (determined at application) | Full Restore, Awakening item |
| Freeze | Ice move (30% chance) | Cannot act until thawed | Thaw (attack, Sunny Day) |

### Volatile Status

- **Confusion**: 33% fail rate on own move, 40 damage recoil. Duration 1-4 turns.
- **Curse** (Ghost-type): Lowers Speed, takes 25% recoil each turn. Stays until switch-out.
- **Leech Seed**: Takes 12.5% damage per turn, damage heals attacker.
- **Attract**: 50% fail rate on move if attracted to opponent. Cross-gender only.

## 10. Items

### Item System (Introduced in Gen 3)

Pokémon now hold items. Key Gen 3 items:

**Stat Modifiers**:
- Assault Vest: +50% Sp. Def (1.5x). Gen 5 item; not in Gen 3.
- Choice Band/Specs/Scarf: 1.5x attack/sp.atk/speed, locks move selection.

**Recovery**:
- Leftovers: Recover 1/8 Max HP each turn.
- Sitrus Berry: Recover 30% Max HP when HP drops below 25%.

**Type Modifiers**:
- Type-specific berries (e.g., Cheri Berry cures paralysis, Aspear cures burn).

## 11. Implementation Notes

### BaseRuleset Structure

`BaseRuleset` implements the `GenerationRuleset` interface with all Gen 3 defaults. Later generations override specific methods.

```typescript
interface GenerationRuleset {
  // Stat calculations
  calculateHP(base: number, iv: number, ev: number, level: number): number;
  calculateStat(base: number, iv: number, ev: number, level: number, nature: Nature): number;

  // Damage calculation
  calculateDamage(
    attacker: Pokémon,
    defender: Pokémon,
    move: Move,
    field: BattleField
  ): number;

  // Ability handling
  applyAbility(ability: Ability, trigger: TriggerType, context: BattleContext): void;

  // Weather
  applyWeather(weather: Weather, turns: number | "permanent"): void;

  // Critical hits
  getCritRate(stage: number): number;

  // Experience
  calculateEXP(baseEXP: number, defenderLevel: number, attackerLevel: number): number;

  // Status conditions
  applyStatus(pokemon: Pokémon, status: StatusCondition): void;
}

class BaseRuleset implements GenerationRuleset {
  // Gen 3 implementations for all methods
  // Gen 4+ rulesets override only changed methods

  calculateHP(base: number, iv: number, ev: number, level: number): number {
    return Math.floor(((2 * base + iv + Math.floor(ev / 4)) * level) / 100) + level + 10;
  }

  calculateStat(base: number, iv: number, ev: number, level: number, nature: Nature): number {
    const inner = Math.floor(((2 * base + iv + Math.floor(ev / 4)) * level) / 100) + 5;
    return Math.floor(inner * nature.multiplier);
  }

  calculateDamage(attacker, defender, move, field): number {
    // Gen 3 damage formula with all modifiers in strict order
    const baseDamage = this.getBaseDamage(attacker, defender, move);
    let damage = baseDamage;

    damage *= this.getTypeEffectiveness(move.type, defender.type);
    damage *= this.getWeatherModifier(move, field.weather);
    damage *= this.getItemModifier(attacker, move);
    damage *= this.getSTAB(attacker, move);
    damage *= this.getCritMultiplier();
    damage *= this.getSpreadPenalty(move, field); // 0.5x for multi-target
    damage *= this.getAbilityModifier(attacker, move);
    damage *= 0.85 + Math.random() * 0.15; // Randomness

    return Math.floor(damage);
  }

  applyAbility(ability: Ability, trigger: TriggerType, context: BattleContext): void {
    // Ability activation logic
    switch (ability.name) {
      case "intimidate":
        context.opponent.stages.atk -= 1;
        break;
      case "levitate":
        context.pokemon.groundImmune = true;
        break;
      // ... other abilities
    }
  }

  applyWeather(weather: Weather, turns: number | "permanent"): void {
    this.field.weather = weather;
    this.field.weatherTurns = turns === "permanent" ? Infinity : turns;
  }

  getCritRate(stage: number): number {
    const rates = [1/16, 1/8, 1/4, 1/3, 1/2];
    return rates[Math.min(stage, 4)];
  }

  calculateEXP(baseEXP: number, defenderLevel: number, attackerLevel: number): number {
    const levelDiff = attackerLevel - defenderLevel;
    return Math.floor((baseEXP * levelDiff / 7) + 1);
  }
}
```

### Inheritance Pattern

class Gen4Ruleset extends BaseRuleset {
  // Sandstorm now deals 1/16 HP damage per turn (same as Gen 3) but SpD boost already in Gen 3

      return;

### Key Implementation Principles

1. **Strict Modifier Order**: Damage calculation must apply modifiers in the exact order specified (type → weather → item → STAB → crit → spread → ability → random).

2. **Ability Triggering**: Each ability must define its trigger condition (switchin, turn-start, damage, etc.) and call the activation method only when conditions are met.

3. **Weather Persistence**: Distinguish between ability-granted weather (permanent until switch-out) and move-granted weather (5 turns, decrements each turn).

4. **Stat Capping**: Some abilities modify stats; cap results to prevent overflow (typically 0-2088 range for modern Pokémon).

5. **No Hidden Abilities**: All Gen 3 Pokémon have exactly one ability; the ability system has no secondary option.
