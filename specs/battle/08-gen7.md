## Overview

Generation 7 introduces **Z-Moves** as the primary battle gimmick, expanding the strategic depth of battle systems. Key characteristics:

- **Z-Moves**: A powerful turn-based gimmick allowing one Z-Move per team per battle
- **Terrain from Abilities**: Surge abilities (Electric Surge, Grassy Surge, Psychic Surge, Misty Surge) automatically set terrain on switch-in
- **Mega Evolution**: Still available and functional, but cannot be used in the same battle as Z-Moves

## 1. Z-Moves

### Concept

Z-Moves are powerful one-time moves that can be used once per team per battle. Each Pokémon holding a matching Z-Crystal can access a unique Z-Move variant of one of its learned moves. Z-Moves coexist with Mega Evolution in Generation 7 battle rules, but a team must choose to focus on either Z-Moves or Mega Evolution for a given battle.

### Z-Crystal Mechanics

- **Held Item**: Z-Crystal is a held item specific to a type (e.g., Normalium-Z, Firium-Z) or species (e.g., Pikachu-specific Pikashunium-Z)
- **Activation**: On a Pokémon's turn, if it holds the matching Z-Crystal and knows a move compatible with that Z-Crystal, the player may choose to use the Z-Move instead
- **One Per Team**: Across all Pokémon on a team, only one Z-Move can be used per battle
- **Simultaneous Mega Prevention**: Cannot use Z-Move and Mega Evolution in the same battle (team-wide restriction)

### Z-Move Physical/Special Power Table

Z-Moves convert standard moves into powered-up variants. The base power is determined by the original move's power:

| Original Move Power | Z-Move Power |
|--------------------|-------------|
| — (Status move)    | Varies      |
| 20–30              | 100         |
| 35–40              | 110         |
| 50–60              | 130         |
| 65–70              | 140         |
| 75–85              | 160         |
| 90–100             | 175         |
| 110+               | 180         |

**Examples**:
- Thunderbolt (90 power) → Gigavolt Havoc (175 power)
- Dragon Claw (75 power) → Devastating Drake (160 power)
- Close Combat (120 power) → All-Out Pummeling (180 power)

### Status Z-Moves

Status moves converted to Z-Moves gain additional beneficial effects:

| Original Move      | Z-Move               | Effect                                       |
|--------------------|----------------------|----------------------------------------------|
| Swords Dance       | Clanging Scales      | +2 Atk, then damage based on move type       |
| Calm Mind          | Prismatic Power      | +2 Sp.Atk, +2 Sp.Def, then damage            |
| Hone Claws         | Splintered Stormshards | +1 Acc, +1 Atk, then damage                |
| Recover/Refresh    | Z-Recovery           | Restore 50% max HP; cure status if used      |
| Sword Dance        | Devastating Drake    | Attack move Z-Move with +2 Atk boost         |
| Nasty Plot         | Prismatic Power      | +2 Sp.Atk, +2 Sp.Def combo                   |

### Implementation Structure

```typescript
// Z-Move type and interface
interface ZCrystal extends Item {
  type: 'zcrystal';
  zType: PokémonType | 'species'; // 'species' for Pokémon-specific crystals
  speciesId?: string; // e.g., 'pikachu' for Pikashunium-Z
}

interface ZMove extends Move {
  isZMove: true;
  baseMove: Move;
  zPower: number;
  zEffect?: {
    type: 'stat_boost' | 'heal' | 'misc';
    details: Record<string, any>;
  };
}

// Z-Move as BattleGimmick
class ZMoveBattleGimmick implements BattleGimmick {
  name = 'z-move';
  usedByTeam: boolean = false;
  activeZMove: ZMove | null = null;

  canUse(pokémon: Pokémon, move: Move): boolean {
    if (this.usedByTeam) return false;
    if (!pokémon.heldItem || pokémon.heldItem.type !== 'zcrystal') return false;

    const zCrystal = pokémon.heldItem as ZCrystal;
    return this.matchesZCrystal(move, zCrystal);
  }

  private matchesZCrystal(move: Move, crystal: ZCrystal): boolean {
    if (crystal.zType === 'species' && crystal.speciesId) {
      // Species-specific Z-Moves (e.g., Pikachu)
      return move.type === 'electric'; // Simplified check
    }
    return move.type === crystal.zType;
  }

  execute(pokémon: Pokémon, move: Move): ZMove {
    const zMove = this.convertToZMove(move);
    this.usedByTeam = true;
    this.activeZMove = zMove;
    return zMove;
  }

  private convertToZMove(move: Move): ZMove {
    const basePower = move.power || 0;
    const zPower = this.calculateZPower(basePower);

    return {
      ...move,
      isZMove: true,
      baseMove: move,
      zPower: zPower,
      power: zPower,
    };
  }

  private calculateZPower(basePower: number): number {
    if (basePower >= 110) return 180;
    if (basePower >= 90) return 175;
    if (basePower >= 75) return 160;
    if (basePower >= 65) return 140;
    if (basePower >= 50) return 130;
    if (basePower >= 35) return 110;
    if (basePower >= 20) return 100;
    return 100; // Default for status moves
  }

  reset(): void {
    this.usedByTeam = false;
    this.activeZMove = null;
  }
}
```

## 2. Terrain from Abilities (Surge Abilities)

### Concept

Generation 7 introduces abilities that automatically set terrain on switch-in. These **Surge abilities** provide Pokémon with additional strategic value by controlling the field environment.

### Surge Abilities

| Ability          | Type    | Terrain         | Effect                                    |
|------------------|---------|-----------------|-------------------------------------------|
| Electric Surge   | —       | Electric Terrain | Grounds Pokémon gain +30% Electric power |
| Grassy Surge     | —       | Grassy Terrain  | Grounds Pokémon gain +30% Grass power   |
| Psychic Surge    | —       | Psychic Terrain | Blocks priority moves against grounded   |
| Misty Surge      | —       | Misty Terrain   | Halves Dragon damage to grounded         |

### Terrain Duration

- **Base Duration**: 5 turns
- **Extended Duration**: 8 turns if the ability holder holds **Terrain Extender**
- **Replacement**: If another ability sets a different terrain, the previous terrain is replaced

### Implementation

```typescript
// Surge ability interface
interface SurgeAbility extends Ability {
  isSurgeAbility: true;
  terrainType: TerrainType;
}

// Terrain types
type TerrainType = 'electric' | 'grassy' | 'psychic' | 'misty';

interface Terrain {
  type: TerrainType;
  turnsRemaining: number;
  setBy: string; // Ability name
}

// Battle state with terrain
class Gen7Battle extends BaseBattle {
  terrain: Terrain | null = null;

  onSwitchIn(pokémon: Pokémon): void {
    super.onSwitchIn(pokémon);
    this.checkForSurgeAbility(pokémon);
  }

  private checkForSurgeAbility(pokémon: Pokémon): void {
    const ability = pokémon.ability;
    if (!ability || !this.isSurgeAbility(ability)) return;

    const surgeAbility = ability as SurgeAbility;
    const duration = pokémon.heldItem?.name === 'Terrain Extender' ? 8 : 5;

    this.setTerrain({
      type: surgeAbility.terrainType,
      turnsRemaining: duration,
      setBy: ability.name,
    });
  }

  private isSurgeAbility(ability: Ability): boolean {
    const surgeAbilities = ['Electric Surge', 'Grassy Surge', 'Psychic Surge', 'Misty Surge'];
    return surgeAbilities.includes(ability.name);
  }

  setTerrain(terrain: Terrain): void {
    this.terrain = terrain;
    this.logTerrain(`${this.terrain.setBy} set ${this.terrain.type} terrain!`);
  }

  decrementTerrainTurns(): void {
    if (this.terrain) {
      this.terrain.turnsRemaining--;
      if (this.terrain.turnsRemaining <= 0) {
        this.terrain = null;
        this.logTerrain('The terrain wore off.');
      }
    }
  }

  private logTerrain(message: string): void {
    console.log(`[Terrain] ${message}`);
  }
}
```

## 3. Terrain Effects in Detail

### Electric Terrain

- **Power Boost**: Grounded Pokémon attacking with Electric moves gain +30% power multiplier
- **Sleep Immunity**: Prevents sleep status for all grounded Pokémon
- **Interaction**: Overrides other terrains; prevents abilities that set alternative terrain from activating

**Example**: Pikachu (Electric type, grounded) using Thunderbolt gains 1.3× power multiplier.

### Grassy Terrain

- **Power Boost**: Grounded Pokémon attacking with Grass moves gain +30% power multiplier
- **HP Heal**: Grounded Pokémon restore 1/16 of max HP at the end of each turn
- **Move Effect**: Earthquake, Bulldoze, and Magnitude are treated as if base power is halved for grounded targets

**Example**: Decidueye using Leaf Blade against a grounded target while Grassy Terrain is active deals normal damage, but that same Decidueye heals 1/16 HP each turn.

### Psychic Terrain

- **Power Boost**: Grounded Pokémon attacking with Psychic moves gain +30% power multiplier
- **Priority Blocking**: Priority moves (moves with positive priority) fail against grounded targets while Psychic Terrain is active
- **Interaction**: Prevents moves like Quick Attack, Aqua Jet from reaching grounded foes

**Example**: Alakazam (grounded, Psychic type) cannot be hit by Mach Punch while Psychic Terrain is active.

### Misty Terrain

- **Dragon Resistance**: Grounded Pokémon take 50% damage from Dragon-type moves
- **Status Immunity**: Prevents all non-volatile status conditions (poison, burn, paralysis, sleep, freeze) for grounded Pokémon

**Example**: A grounded Pokémon cannot be poisoned and takes 50% reduced damage from Dragon Claw.

### Implementation

```typescript
type TerrainType = 'electric' | 'grassy' | 'psychic' | 'misty';

interface TerrainEffect {
  modifyMovePower?(
    attacker: Pokémon,
    defender: Pokémon,
    move: Move,
    terrain: Terrain
  ): number; // Multiplier (e.g., 1.3 for +30%)

  modifyDamage?(
    attacker: Pokémon,
    defender: Pokémon,
    move: Move,
    baseDamage: number,
    terrain: Terrain
  ): number; // Adjusted damage

  onEndOfTurn?(
    pokémon: Pokémon,
    terrain: Terrain
  ): void; // Apply HP heal, etc.

  blockEffect?(
    effect: string,
    pokémon: Pokémon,
    terrain: Terrain
  ): boolean; // Block sleep, priority moves, etc.
}

const terrainEffects: Record<TerrainType, TerrainEffect> = {
  electric: {
    modifyMovePower(attacker, defender, move, terrain) {
      if (defender.grounded && move.type === 'electric') {
        return 1.3;
      }
      return 1.0;
    },
    blockEffect(effect, pokémon, terrain) {
      if (pokémon.grounded && effect === 'sleep') {
        return true; // Block sleep
      }
      return false;
    },
  },

  grassy: {
    modifyMovePower(attacker, defender, move, terrain) {
      if (defender.grounded && move.type === 'grass') {
        return 1.3;
      }
      return 1.0;
    },

    modifyDamage(attacker, defender, move, baseDamage, terrain) {
      if (
        defender.grounded &&
        ['earthquake', 'bulldoze', 'magnitude'].includes(move.id)
      ) {
        return Math.floor(baseDamage * 0.5);
      }
      return baseDamage;
    },

    onEndOfTurn(pokémon, terrain) {
      if (pokémon.grounded) {
        const healAmount = Math.max(1, Math.floor(pokémon.maxHP / 16));
        pokémon.hp = Math.min(pokémon.maxHP, pokémon.hp + healAmount);
      }
    },
  },

  psychic: {
    modifyMovePower(attacker, defender, move, terrain) {
      if (defender.grounded && move.type === 'psychic') {
        return 1.3;
      }
      return 1.0;
    },

    blockEffect(effect, pokémon, terrain) {
      if (pokémon.grounded && effect === 'priority-move') {
        return true; // Block priority moves
      }
      return false;
    },
  },

  misty: {
    modifyDamage(attacker, defender, move, baseDamage, terrain) {
      if (defender.grounded && move.type === 'dragon') {
        return Math.floor(baseDamage * 0.5);
      }
      return baseDamage;
    },

    blockEffect(effect, pokémon, terrain) {
      if (pokémon.grounded && ['poison', 'burn', 'paralysis', 'sleep', 'freeze'].includes(effect)) {
        return true; // Block status conditions
      }
      return false;
    },
  },
};
```

## 4. Mega Evolution (Gen 7 Continuity)

### Overview

Mega Evolution continues from Generation 6 with unchanged mechanics. A team may choose to use **either** Mega Evolution **or** Z-Moves, but not both in the same battle.

### Mechanics

- **Held Item**: Mega Stone specific to Pokémon species (e.g., Charizardite X)
- **Evolution on Turn**: Pokémon mega-evolves during its action on the first turn
- **Duration**: Lasts until the Pokémon is switched out or faints
- **Stat Changes**: Base stats increase according to species-specific Mega Evolution form
- **Ability Change**: May change to a Mega-exclusive ability (e.g., Mega Charizard X gains Dragon Dance boost from new ability)

### Battle Gimmick Implementation

```typescript
class MegaEvolutionGimmick implements BattleGimmick {
  name = 'mega-evolution';
  usedByTeam: boolean = false;

  canUse(pokémon: Pokémon): boolean {
    if (this.usedByTeam) return false;
    if (!pokémon.heldItem?.isMegaStone) return false;
    if (pokémon.isMegaEvolved) return false;
    return true;
  }

  execute(pokémon: Pokémon): void {
    const megaForm = this.getMegaForm(pokémon);
    pokémon.megaEvolve(megaForm);
    this.usedByTeam = true;
  }

  private getMegaForm(pokémon: Pokémon): MegaForm {
    // Look up Mega form from species + stone combination
    return megaRegistry.get(`${pokémon.species}-${pokémon.heldItem!.name}`);
  }

  reset(): void {
    this.usedByTeam = false;
  }
}
```

## 5. Prankster Nerf

### Mechanic

In Generation 7, **Prankster-boosted status moves now fail against Dark-type targets**.

### Details

- **Before (Gens 5-6)**: Prankster allowed status moves to always go first regardless of target type
- **Gen 7 Change**: Prankster status moves are blocked if the target is Dark-type
- **Affected Moves**: Any status move (non-damaging) used with Prankster boost
- **Non-Status Moves**: Unaffected; damaging moves used by Prankster still gain priority

### Example

```
Liepard (Prankster) uses Thunder Wave against Umbreon (Dark type)
→ Move fails; Umbreon is immune due to Dark type + Prankster interaction
```

### Implementation

```typescript
function canUsePranksterMove(
  attacker: Pokémon,
  defender: Pokémon,
  move: Move
): boolean {
  if (attacker.ability?.name !== 'Prankster') return true;
  if (move.power > 0) return true; // Damaging move, always allowed
  if (defender.type.includes('dark')) return false; // Status move blocked vs Dark
  return true;
}
```

## 6. Burn Nerf

### Mechanic

In Generation 7, **Burn damage reduces from 1/8 max HP per turn to 1/16 max HP per turn**.

### Details

- **Before (Gens 3-6)**: Burned Pokémon lose 1/8 of max HP per turn
- **Gen 7 Change**: Burned Pokémon lose 1/16 of max HP per turn (half damage)
- **Attack Reduction**: Remains unchanged at -50% physical attack

### Example

| Generation | Max HP | Burn Damage/Turn |
|-----------|--------|-----------------|
| Gen 3-6   | 320    | 40 HP (1/8)      |
| Gen 7+    | 320    | 20 HP (1/16)     |

### Implementation

```typescript
function applyBurnDamage(pokémon: Pokémon): void {
  if (!pokémon.status.includes('burn')) return;

  const burnDamage = Math.max(1, Math.floor(pokémon.maxHP / 16));
  pokémon.hp = Math.max(0, pokémon.hp - burnDamage);

  console.log(`${pokémon.name} took ${burnDamage} burn damage!`);
}
```

## 7. Confusion Nerf

### Mechanic

In Generation 7, **Confusion self-hit chance reduces from 50% to 33%**.

### Details

- **Before (Gens 3-6)**: Confused Pokémon have 50% chance each turn to hit itself instead of using its selected move
- **Gen 7 Change**: Confused Pokémon have 33% chance (1/3) each turn to hit itself

### Example

```
Alakazam is confused.
- 33% chance to hurt itself in confusion (damage = (Sp.Atk / Def) × 40)
- 67% chance to execute its selected move normally
```

### Implementation

```typescript
function checkConfusion(pokémon: Pokémon): boolean {
  if (!pokémon.status.includes('confusion')) return false;

  const selfHitChance = 1 / 3; // 33%
  if (Math.random() < selfHitChance) {
    const damage = calculateSelfdamage(pokémon);
    pokémon.hp = Math.max(0, pokémon.hp - damage);
    console.log(`${pokémon.name} hurt itself in confusion for ${damage} damage!`);
    return true;
  }
  return false;
}

function calculateSelfdamage(pokémon: Pokémon): number {
  // Simplified: 1/2 of max HP as base self-damage from confusion
  return Math.max(1, Math.floor(pokémon.maxHP / 8));
}
```

## 8. Paralysis Nerf

### Mechanic

In Generation 7, **Paralysis speed reduction changes from 75% to 50%**.

### Details

- **Before (Gens 3-6)**: Paralyzed Pokémon have speed reduced to 25% of its normal value (75% reduction)
- **Gen 7 Change**: Paralyzed Pokémon have speed reduced to 50% of its normal value (50% reduction)

### Example

| Pokémon    | Base Speed | Gen 3-6 Paralysis | Gen 7 Paralysis |
|-----------|-----------|------------------|-----------------|
| Alakazam  | 120       | 30                | 60              |
| Dragonite | 80        | 20                | 40              |

### Implementation

```typescript
function getEffectiveSpeed(pokémon: Pokémon): number {
  let speed = pokémon.stats.speed;

  if (pokémon.status.includes('paralysis')) {
    speed = Math.floor(speed * 0.5); // 50% reduction (50% remaining)
  }

  return speed;
}

function calculatePriority(
  attacker: Pokémon,
  defender: Pokémon
): 'first' | 'second' {
  const attackerSpeed = getEffectiveSpeed(attacker);
  const defenderSpeed = getEffectiveSpeed(defender);

  return attackerSpeed >= defenderSpeed ? 'first' : 'second';
}
```

## 9. Aurora Veil

### Overview

**Aurora Veil** is a new screen move introduced in Generation 7, combining the protective effects of Reflect and Light Screen.

### Mechanics

- **Base Power**: Reduces physical and special damage by 50% each turn
- **Duration**: 5 turns (8 turns if used by a Pokémon holding Screens Pass)
- **Usage Restriction**: Can only be used during Hail or Snow weather
- **Team Effect**: Affects the entire team, similar to Reflect and Light Screen
- **Stacking**: Does not stack with Reflect or Light Screen; if active, they are replaced

### Details

| Attribute       | Value       |
|-----------------|-----------|
| Type            | Ice        |
| Category        | Status     |
| Power/Accuracy  | —/—       |
| Base Power      | —         |
| PP              | 20        |
| Weather Requirement | Hail/Snow |

### Implementation

```typescript
interface Screen {
  type: 'reflect' | 'light-screen' | 'aurora-veil';
  turnsRemaining: number;
  damageReduction: number; // 0.5 for 50% reduction
}

class AuroraVeilMove extends Move {
  name = 'Aurora Veil';
  type = 'ice';
  category = 'status';
  accuracy = 100; // Always succeeds if conditions met
  pp = 20;

  canUse(user: Pokémon, battle: Gen7Battle): boolean {
    // Check if weather is Hail or Snow
    const validWeather = ['hail', 'snow'].includes(battle.weather?.type || '');
    if (!validWeather) {
      console.log(`Aurora Veil can only be used in Hail or Snow!`);
      return false;
    }
    return true;
  }

  execute(user: Pokémon, battle: Gen7Battle): void {
    const duration = user.heldItem?.name === 'Screens Pass' ? 8 : 5;

    // Replace existing screens
    battle.screens = {
      type: 'aurora-veil',
      turnsRemaining: duration,
      damageReduction: 0.5,
    };

    battle.log(`${user.name} set up Aurora Veil!`);
  }
}

function applyScreenDamageReduction(
  baseDamage: number,
  screen: Screen | null,
  moveCategory: 'physical' | 'special'
): number {
  if (!screen) return baseDamage;

  // Reflect reduces physical damage; Light Screen reduces special damage
  if (screen.type === 'reflect' && moveCategory === 'physical') {
    return Math.floor(baseDamage * 0.5);
  }
  if (screen.type === 'light-screen' && moveCategory === 'special') {
    return Math.floor(baseDamage * 0.5);
  }
  // Aurora Veil reduces both physical and special damage
  if (screen.type === 'aurora-veil') {
    return Math.floor(baseDamage * 0.5);
  }
  return baseDamage;
}
```

## 10. New Gen 7 Abilities

### Beast Boost

- **Type**: Trigger-based
- **Effect**: When the Pokémon faints an opponent, its highest stat (Atk, Def, Sp.Atk, Sp.Def, or Spe) increases by 1 stage
- **Interaction**: Applies after the opponent faints, before switching

```typescript
class BeastBoost extends Ability {
  name = 'Beast Boost';

  onKill(user: Pokémon, defeated: Pokémon): void {
    const highest = this.getHighestStat(user);
    user.boosts[highest]++;
    console.log(`${user.name}'s ${highest} increased!`);
  }

  private getHighestStat(pokémon: Pokémon): keyof Stats {
    const stats = {
      atk: pokémon.stats.atk,
      def: pokémon.stats.def,
      spAtk: pokémon.stats.spAtk,
      spDef: pokémon.stats.spDef,
      spe: pokémon.stats.spe,
    };

    return Object.entries(stats).reduce((max, [key, val]) =>
      val > stats[max as keyof Stats] ? (key as keyof Stats) : max
    );
  }
}
```

### Disguise (Mimikyu)

- **Type**: Damage reduction
- **Effect**: First time this Pokémon takes damage, damage is reduced to 0 and the ability breaks (visual change: Mimikyu's disguise breaks)
- **Persistence**: Once broken, does not restore even after switching out during battle

```typescript
class Disguise extends Ability {

      defender.isDisguiseBroken = true;
      console.log(`${defender.name}'s disguise broke!`);
      return 0;
    }
    return damage;
  }
}
```

### Schooling (Wishiwashi)

- **Type**: Form change
- **Effect**: When Pokémon's HP is above 25% max HP, it takes the School Form (higher stats). Below 25%, it reverts to Solo Form
- **Trigger**: Checked at the start of each turn and after taking damage

```typescript
class Schooling extends Ability {
  name = 'Schooling';

  checkFormChange(pokémon: Pokémon): void {
    const hpPercentage = (pokémon.hp / pokémon.maxHP) * 100;

    if (hpPercentage > 25 && !pokémon.isSchoolForm) {
      pokémon.isSchoolForm = true;
      pokémon.species = 'Wishiwashi-School';
      console.log(`${pokémon.name} formed a school!`);
    } else if (hpPercentage <= 25 && pokémon.isSchoolForm) {
      pokémon.isSchoolForm = false;
      pokémon.species = 'Wishiwashi';
      console.log(`${pokémon.name} is alone!`);
    }
  }
}
```

## 11. End-of-Turn Order (Gen 7 Specifics)

### Turn Resolution Order

Generation 7 battle systems follow this end-of-turn sequence:

1. **Residual Damage**
   - Burn damage (1/16 max HP)
   - Poison damage (1/8 max HP or 1/4 for badly poisoned)
   - Sandstorm damage (1/8 max HP, except Rock/Ground/Steel)
   - Hail damage (1/8 max HP, except Ice-type)

2. **Terrain Effects**
   - Grassy Terrain healing (1/16 max HP to grounded Pokémon)
   - Continued status condition checks

3. **Weather Effects**
   - Aurora Veil turns decrement
   - Terrain turns decrement

4. **Stat Resets**
   - Temporary stat changes from moves like Dragon Dance persist
   - Sticky Web persists until replaced

5. **Switch Checks**
   - Determine if forced switches occur (e.g., Roar, Whirlwind)

### Implementation

```typescript
function executeEndOfTurn(battle: Gen7Battle): void {
  // 1. Residual damage
  battle.activePokémon.forEach((poke) => {
    if (poke) {
      applyBurnDamage(poke);
      applyPoisonDamage(poke);
      applyWeatherDamage(poke, battle.weather);
    }
  });

  // 2. Terrain effects
  if (battle.terrain) {
    const effect = terrainEffects[battle.terrain.type];
    battle.activePokémon.forEach((poke) => {
      if (poke) effect.onEndOfTurn?.(poke, battle.terrain!);
    });
  }

  // 3. Weather/Terrain countdown
  battle.decrementTerrainTurns();
  battle.decrementWeatherTurns();

  // 4. Screen countdown
  if (battle.screens) {
    battle.screens.turnsRemaining--;
    if (battle.screens.turnsRemaining <= 0) {
      battle.screens = null;
    }
  }

  // 5. Switch checks (Roar, Whirlwind, etc.)
  checkForForcedSwitches(battle);
}
```

```typescript
class Gen7Ruleset extends BaseRuleset {
  name = 'Generation 7';
  generation = 7;

  // Battle gimmick: Choose Z-Moves OR Mega Evolution
  battleGimmicks: BattleGimmick[] = [
    new ZMoveBattleGimmick(),
    new MegaEvolutionGimmick(),
  ];

  canUseBothGimmicks(): boolean {
    return false; // Either Z-Moves or Mega, not both
  }

  // Override base mechanics
  calculateDamage(
    attacker: Pokémon,
    defender: Pokémon,
    move: Move
  ): number {
    let damage = super.calculateDamage(attacker, defender, move);

    // Apply terrain modifiers
    if (this.battle.terrain) {
      const terrainMult = terrainEffects[this.battle.terrain.type]
        .modifyMovePower?.(attacker, defender, move, this.battle.terrain) || 1.0;
      damage = Math.floor(damage * terrainMult);
    }

    // Apply Aurora Veil reduction
    if (this.battle.screens?.type === 'aurora-veil') {
      damage = applyScreenDamageReduction(damage, this.battle.screens, move.category);
    }

    return damage;
  }

  getEffectiveSpeed(pokémon: Pokémon): number {
    let speed = pokémon.stats.spe;

    // Gen 7: Paralysis reduces speed by 50% (not 75%)
    if (pokémon.status.includes('paralysis')) {
      speed = Math.floor(speed * 0.5);
    }

    return speed;
  }

  applyEndOfTurn(battle: Gen7Battle): void {
    executeEndOfTurn(battle);
  }
}
```
