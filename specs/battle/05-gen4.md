# Generation 4 Battle Mechanics Spec

## Quick Start

Gen 4 is the **generation of game mechanics overhauls**. The two headline changes are:

1. **Physical/Special split (per-move category)**: In Gen 1–3, damage type (physical vs special) was determined by the move's TYPE (e.g., Fire-type moves were always special). Gen 4 assigns each move its own category: 'physical', 'special', or 'status'. This is the DEFINING change of the generation and requires rewriting every move in the game.

2. **Entry hazard system**: Stealth Rock, Toxic Spikes, and the new Defog move overhaul competitive teambuilding.

Other Gen 4 introductions:
- **Trick Room, Gravity, Tailwind** — field effects beyond weather
- **47 new abilities** — many with complex on-hit or on-turn mechanics
- **Weather rocks** — extend move-summoned weather from 5 to 8 turns (ability weather still indefinite)
- **Sandstorm Rock-type SpDef boost** — first time weather grants defensive bonus to a type
- **Blizzard 100% accuracy in Hail** — first Hail-specific move buff
- **Choice Scarf, Choice Specs, Life Orb, Focus Sash** — new held items
- **Spread move damage penalty 0.75x** (changed from Gen 3's 0.5x)
- **High-crit move list** updated with 5 new moves
- **New priority moves** (Aqua Jet, Bullet Punch, Ice Shard, Shadow Sneak, Sucker Punch, Vacuum Wave, Feint)

**What does NOT exist in Gen 4** (commonly confused — LLM training data is biased toward Gen 5+ mechanics):
- No Fairy type (Gen 6) — type chart is still 17 types
- No hidden abilities (Gen 5) — each Pokémon has exactly 1 or 2 ability slots, no Dream World slot
- No Team Preview (Gen 5)
- No permanent weather nerf (Gen 6) — ability weather (Drizzle, Drought, Sand Stream, Snow Warning) is still INDEFINITE
- No Mega Evolution (Gen 6)
- No Z-Moves (Gen 7)
- No Dynamax (Gen 8)
- No Terastallization (Gen 9)
- No Terrain (Gen 5+)
- No Assault Vest (Gen 6)
- No Rocky Helmet (Gen 5)
- No Gems (Gen 5)
- No Eviolite (Gen 5)
- ExtremeSpeed is +1 priority (not +2 — changed in Gen 5)
- Explosion STILL halves target's Defense in damage calc (removed in Gen 5)
- Crit multiplier is still 2.0x (not 1.5x — changed in Gen 6)
- Tailwind lasts 3 turns (changed to 4 in Gen 5)
- Electric types are immune to paralysis from Electric-type MOVES only (blanket immunity is Gen 6)
- Paralysis speed is ×0.25 (not ×0.5 — changed in Gen 7)
- Burn damage is 1/8 max HP (not 1/16 — changed in Gen 7)
- Knock Off does NOT get 1.5x damage boost (that's Gen 5+)
- Prankster, Magic Bounce, Regenerator, Multiscale do NOT exist (Gen 5)

**Gen 5+ abilities that do NOT exist in Gen 4** (do not implement these):
Prankster, Magic Bounce, Regenerator, Multiscale, Sheer Force, Sand Rush, Sand Force, Overcoat, Harvest, Moody, Cursed Body, Telepathy, Illusion, Imposter, Analytic, Defiant, Justified, Rattled, Contrary, Weak Armor, Heavy Metal, Light Metal, Zen Mode, Victory Star, Turboblaze, Teravolt, Mummy

## 1. Stat System

### Formula

The HP and stat formulas are **identical to Gen 3**. No changes in Gen 4.

**HP formula:**
```
HP = floor(((2 * Base + IV + floor(EV / 4)) * Level) / 100) + Level + 10
```

**Non-HP stat formula:**
```
stat = floor((floor(((2 * Base + IV + floor(EV / 4)) * Level) / 100) + 5) * NatureMod)
```

Where:
- **Base**: Pokémon's base stat (0–255)
- **IV**: Individual value (0–31)
- **EV**: Effort value (0–255 per stat, 510 total)
- **Level**: Pokémon's level (1–100)
- **NatureMod**: Nature multiplier (1.1 = +10%, 0.9 = −10%, 1.0 = neutral)

### Shedinja Exception

Shedinja always has 1 HP, regardless of stats.

### Natures

25 natures, same as Gen 3:
- 6 neutral natures: Hardy, Docile, Serious, Bashful, Quirky
- 20 boosting natures: 5 Atk modifiers × 4 stat reductions each

Each nature boosts one stat by +10% and reduces another by −10%.

### Example Verification

**Level 50 Charizard with 31 HP IV, 252 HP EV, neutral nature (Hardy):**
```
Base = 78
IV = 31
EV = 252
Level = 50

HP = floor(((2 * 78 + 31 + floor(252 / 4)) * 50) / 100) + 50 + 10
   = floor(((156 + 31 + 63) * 50) / 100) + 60
   = floor((250 * 50) / 100) + 60
   = floor(125) + 60
   = 185
```

Expected from Bulbapedia: **185 HP** ✓

## 3. Damage Formula

### Base Formula

```
levelFactor = floor(2 * Level / 5) + 2
baseDamage = floor(floor(levelFactor * Power * A / D) / 50) + 2
```

Where:
- **A**: Attacker's Attack stat (physical moves) or Special Attack stat (special moves)
- **D**: Defender's Defense stat (physical moves) or Special Defense stat (special moves)
- **Power**: Move's base power (0 for status moves, non-damaging)

### Key Gen 4 Change: Move Category

In Gen 1–3, move type determined damage category. A Fire-type move always dealt special damage.

**In Gen 4+, each move has a category**: 'physical', 'special', or 'status'.

Example moves that changed damage type:
- **Fire Punch**: Was special (Gen 3), now physical (Gen 4)
- **Strength**: Was physical, still physical
- **Thunderbolt**: Was special, still special

The Physical/Special split affects:
- Which stat (Atk vs SpAtk, Def vs SpDef) is used in damage calculation
- Burn status effect (only halves physical attack damage now, not all damage)
- Move-specific abilities (e.g., Technician, Iron Fist, Muscle Band all check move.category)

### Damage Modifiers

Modifiers are applied sequentially. Each result is floored before the next modifier is applied:

1. **Targets** (spread moves in doubles/triples): 0.75x
2. **Weather**: 1.5x (relevant type in favorable weather), 0.5x (opposing type in harsh weather)
3. **Critical hit**: 2x (ignores negative Atk and positive Def stages; see Critical Hits section)
4. **Random factor**: floor(85 + PRNG(0, 15)) / 100 ≈ 0.85–1.0x
5. **STAB** (Same-Type Attack Bonus): 1.5x if move type matches any Pokémon type; 2.0x with Adaptability ability
6. **Type effectiveness**: Product of type1 effectiveness and type2 effectiveness (if dual-typed target)
7. **Burn penalty**: 0.5x for physical moves only (attacker must be burned)
8. **Other modifiers** (Life Orb, Expert Belt, Plates, Muscle Band, Wise Glasses, Zoom Lens, items, abilities)

Final damage is floored and minimum 1 HP.

### Example Verification

**Level 100 Charizard (Fire/Flying) uses Flamethrower (90 base, Special) against level 100 Rhyperior (Ground/Rock):**

```
Charizard stats: Atk 166, SpAtk 202, Def 158, SpDef 155
Rhyperior stats: Atk 229, SpAtk 119, Def 194, SpDef 131

levelFactor = floor(2 * 100 / 5) + 2 = floor(40) + 2 = 42
baseDamage = floor(floor(42 * 90 * 202 / 131) / 50) + 2
           = floor(floor(764754 / 131) / 50) + 2
           = floor(floor(5837.23) / 50) + 2
           = floor(5837 / 50) + 2
           = floor(116.74) + 2
           = 116 + 2
           = 118

Type effectiveness: Fire vs Ground = 2x, Fire vs Rock = 2x → 2 * 2 = 4x
Random: 0.85–1.0x (assume 1.0x for verification)
STAB: Charizard is Fire/Flying, Flamethrower is Fire → 1.5x
Weather: assume neutral
Burn: Charizard not burned, no penalty
No critical hit

damage = floor(floor(floor(floor(118 * 1.0) * 1.5) * 4) * 1.0)
       = floor(floor(floor(118 * 1.5) * 4) * 1.0)
       = floor(floor(177 * 4) * 1.0)
       = floor(708 * 1.0)
       = 708

Percentage: 708 / 131 = 540% (4-hit KO worst case with minimum damage roll)
```

Expected: ~2–3 hit KO vs Rhyperior at level 100. ✓

## 4. Critical Hits

### Critical Hit Chance

| Stage | Chance |
|-------|--------|
| 0 | 1/16 (6.25%) |
| +1 | 1/8 (12.5%) |
| +2 | 1/4 (25%) |
| +3 | 1/3 (33.3%) |
| +4+ | 1/2 (50%) |

### Critical Hit Mechanics

- **Multiplier**: 2x damage (same as Gen 3)
- **Stat stage bypass**: Critical hits ignore negative Atk stages and positive Def stages (only used for damage calc)
- **Ability modifiers**:
  - **Sniper**: Critical hit damage is 3x instead of 2x (ability effect, new in Gen 4)
  - **Super Luck**: Attacker's critical hit stage +1 (new in Gen 4)
- **High-crit moves** (+1 stage): Slash, Karate Chop, Razor Leaf, Crabhammer, Stone Edge, Cross Chop, Aeroblast, Cross Poison, Psycho Cut, Shadow Claw, Night Slash, Leaf Blade, Attack Order (updated list in Gen 4)
- **Items**:
  - **Scope Lens**: +1 crit stage (Gen 3 item, unchanged)
  - **Razor Claw**: +1 crit stage (new item in Gen 4)

### Example Verification

**Level 50 Alakazam (Psychic) uses Psycho Cut (70 base, Physical, high-crit) with Scope Lens against level 50 Machamp:**

```
Alakazam: Atk 79, Def 66
Machamp: Def 130

Base damage (no STAB, assume neutral type):
levelFactor = floor(2 * 50 / 5) + 2 = 22
baseDamage = floor(floor(22 * 70 * 79 / 130) / 50) + 2 = floor(floor(939.07) / 50) + 2 = 20

Critical stage: +1 (high-crit move) + 1 (Scope Lens) = +2
Critical chance: 1/4 (25%)

Non-critical damage: 20 × 1.5 (STAB) × 0.5x (Psychic weak to Psychic?)
Actually Alakazam is Psychic, Machamp is Fighting/Normal.
Type: Psycho Cut is Psychic. Psychic vs Fighting = 2x.
Non-critical: floor(20 * 1.5 * 2) = 60
Critical: 60 * 2 = 120 (25% of the time)
```

Expected: ~5–8 hit non-crit, ~2–4 hit crit. ✓

## 5. Abilities

### Ability Count

Gen 4 introduces **47 new abilities**. Total abilities across Gen 1–4: 76 (33 Gen 1–3 + 47 Gen 4).

### Core Ability Categories

#### Damage Modifiers

- **Adaptability**: STAB 1.5x → 2.0x (ability only; Normals still get 1.5x base STAB)
- **Technician**: Moves with base power ≤60 gain 1.5x multiplier (new in Gen 4)
- **Iron Fist**: Moves in the "punch" category gain 1.2x multiplier (new in Gen 4). Punch moves (per pokeplatinum `battle_lib.c:6566-6582` `sPunchingMoves`): Ice Punch, Fire Punch, Thunder Punch, Mach Punch, Focus Punch, Dizzy Punch, Dynamic Punch, Hammer Arm, Mega Punch, Comet Punch, Meteor Mash, Shadow Punch, Drain Punch, Bullet Punch, Sky Uppercut
- **Muscle Band** (item): 1.1x physical moves
- **Wise Glasses** (item): 1.1x special moves
- **Expert Belt** (item): 1.2x on super effective moves
- **Plates** (Arceus items): 1.2x to matching type moves (e.g., Flame Plate → Fire moves 1.2x)
- **Tinted Lens**: "Not very effective" moves deal 2x instead of their reduced damage (effectively neutral; new in Gen 4)
- **Filter / Solid Rock**: Super effective damage reduced to 0.75x (e.g., 4x becomes 3x, 2x becomes 1.5x; new in Gen 4)

#### Stat/Speed Modifiers

- **Download**: On switch-in, +1 Atk if foe's Def < SpDef, else +1 SpAtk. In doubles, compares the **average** Def vs average SpDef of both opposing Pokémon. (new in Gen 4)
- **Simple**: Stat stage changes doubled (e.g., +1 becomes +2, −1 becomes −2; new in Gen 4)
- **Unaware**: Ignores opponent's stat changes (ability only; new in Gen 4)
- **Motor Drive**: Electric immunity, +1 Speed if hit by Electric move (new in Gen 4)
- **Steadfast**: +1 Speed when flinched (new in Gen 4)
- **Stall**: User always moves last in its priority bracket (new in Gen 4; negates priority)
- **Slow Start**: Halves Atk and Spe for 5 turns on switch-in (new in Gen 4)
- **Choice Scarf** (item): Speed × 1.5, locks to first move used (new in Gen 4)

#### Type/Effect Immunity & Absorption

- **Mold Breaker**: User's moves ignore target's ability (including immunities; new in Gen 4)
- **Scrappy**: Normal and Fighting moves hit Ghost types (new in Gen 4)
- **Normalize**: All moves used by this Pokémon become Normal type (new in Gen 4)
- **Levitate**: Ground immunity. Gravity removes this immunity.
- **Dry Skin**: Rain heals 1/8 max HP per turn; Sun damages 1/8 max HP per turn; Fire moves deal 1.25x damage; Water moves heal 1/4 max HP (new in Gen 4)
- **Heatproof**: Fire damage halved; burn status damage halved (new in Gen 4)
- **Poison Heal**: Poison status heals 1/8 max HP per turn instead of damaging (new in Gen 4)
- **Magic Guard**: Immune to passive damage (weather, Leech Seed, Curse, entry hazards) but not direct move damage (new in Gen 4 or Gen 3; check source)
- **Klutz**: Held item has no effect (new in Gen 4)
- **Hydration**: Status conditions cured when in Rain (existing; new effect in Gen 4)

#### Weather & Terrain

- **Snow Warning**: Auto-Hail on switch-in (indefinite; not affected by weather rocks; new in Gen 4)
- **Snow Cloak**: Evasion +20% in Hail (new in Gen 4; analogous to Sand Veil in sandstorm)
- **Ice Body**: Heals 1/16 max HP per turn in Hail (new in Gen 4)
- **Leaf Guard**: Prevents status conditions when in Sun (new in Gen 4)
- **Flower Gift**: In Sun, holder gains +50% Atk and holder + allies gain +50% SpDef (new in Gen 4)
- **Solar Power**: In Sun, holder gains +50% SpAtk but loses 1/8 max HP per turn (existing; mechanics confirmed Gen 4)

#### Tracking & Prediction

- **Anticipation**: Warns if opponent has a super effective move or OHKO move on switch-in (new in Gen 4)
- **Forewarn**: Reveals opponent's strongest move (by base power) on switch-in (new in Gen 4)
- **Frisk**: Reveals opponent's held item on switch-in (new in Gen 4)

#### Passive Damage & Effects

- **Bad Dreams**: Sleeping opponents lose 1/8 max HP per turn (new in Gen 4)
- **Aftermath**: Attacker takes 1/4 max HP damage if they KO this Pokémon via contact (new in Gen 4)
- **Tangled Feet**: Evasion +50% when confused (new in Gen 4)
- **Rivalry**: Damage dealt × 1.25x if target is same gender; 0.75x if opposite gender (new in Gen 4)

#### Multi-Hit & Recoil

- **Skill Link**: Multi-hit moves always hit maximum times (5 hits; new in Gen 4)
- **Reckless**: Recoil moves (Brave Bird, Close Combat, Double-Edge, Head Charge, Jump Kick, etc.) gain 1.2x multiplier but recoil unchanged (new in Gen 4)

#### Special Mechanics

- **Multitype**: Arceus ability; Pokémon type changes based on held Plate (Fire Plate → Fire type, etc.; new in Gen 4)
- **No Guard**: All moves by and against this Pokémon always hit (accuracy checks bypassed; new in Gen 4)
- **Suction Cups**: Prevents forced switching (existing; mechanics unchanged from Gen 3)
- **Gluttony**: Eat Berry at 50% max HP instead of 25% (changed behavior; already existed but updated in Gen 4)

## 6. Double Battles

### Spread Move Penalty

In Gen 3, spread moves (Earthquake, Surf, Explosion in doubles) dealt 0.5x damage per additional target.

**In Gen 4, the penalty is 0.75x** per additional target. This applies to:
- Moves that explicitly target multiple opponents (Earthquake, Surf, Discharge, Blizzard, etc.)
- "All Pokémon" moves in singles (e.g., Explosion, Perish Song still affect all, but in doubles with 2 foes on each side)

### Explosion Damage

In Gen 4, **Explosion still halves the target's Defense** in the damage calculation. This was removed in Gen 5.

```
baseDamage = floor(floor(levelFactor * 250 * Atk / (D * 0.5)) / 50) + 2
```

## 7. Weather System

### Weather Types

Four weather types: rain, sun, sandstorm, hail. Same as Gen 3, with Gen 4 additions.

### Weather Duration

- **Move-summoned** (Rain Dance, Sunny Day, Sandstorm, Hail): 5 turns default, **8 turns with weather rocks**
- **Ability-summoned** (Snow Warning): Indefinite (not affected by weather rocks)
- Weather rocks (new in Gen 4):
  - **Damp Rock** → Rain: 8 turns
  - **Heat Rock** → Sun: 8 turns
  - **Smooth Rock** → Sandstorm: 8 turns
  - **Icy Rock** → Hail: 8 turns

### Weather Effects

#### Rain
- Water-type moves: 1.5x damage
- Fire-type moves: 0.5x damage
- Thunder (move): 100% accuracy (unchanged from Gen 2)
- Solar Beam: half power (charge still required, and half power applies)
- Rain Dish: heals 1/16 max HP per turn
- Dry Skin: heals 1/4 max HP per turn (not 1/8)
- Hydration: cures status conditions at end of turn

#### Sun
- Fire-type moves: 1.5x damage
- Water-type moves: 0.5x damage
- Solar Beam: no charge (full power)
- Solar Power: +50% SpAtk, −1/8 max HP per turn
- Leaf Guard: cures status conditions (prevents new status)
- Dry Skin: −1/8 max HP per turn (damage, not healing)
- Flower Gift: user and allies gain +50% Atk and +50% SpDef

#### Sandstorm
- Sandstorm damage: 1/16 max HP per turn (non-Rock, non-Ground, non-Steel types)
- Rock-type Pokémon: **+50% SpDef** (NEW in Gen 4; did NOT exist in Gen 3)
- Sand Veil: evasion +20%
- Sand Stream: auto-sandstorm on switch-in
- Solar Beam: half power (charge still required)

#### Hail
- Hail damage: 1/16 max HP per turn (non-Ice types)
- **Blizzard: 100% accuracy** (NEW in Gen 4; was 70% in Gen 3)
- Ice Body: heals 1/16 max HP per turn
- Snow Warning: auto-hail on switch-in

## 8. Status Conditions

### Sleep

- Duration: **1–5 turns** (international Gen 4); **2–6 turns** (Japanese Diamond/Pearl only)
- Counter decrements at start of turn before action selection; if counter reaches 0, Pokémon wakes and acts normally that turn
- Moves that cure sleep: Refresh, Awaken (Gen 2+)
- Abilities that cure sleep: Hydration (in Rain)
- Fire moves thaw frozen targets but do NOT cure sleep
- Sleep counter persists through switches (not reset on switch-out)
<!-- Source: Bulbapedia "Sleep (status condition)" article. Note: ERRATA.md says "Gen 3-4: 2-5 turns" but Bulbapedia confirms 1-5 for international Gen 4. The ERRATA entry may be conflating Japanese/international or Gen 3/Gen 4 differences. -->

### Paralysis

- Speed: × 0.25 (75% reduction; same as Gen 3)
- Full paralysis: 25% chance to lose a turn
- **Electric-type immunity**: Electric types are immune to paralysis from **Electric-type MOVES only** (Body Slam, Static, Stun Spore, Thunder Wave if used by non-Electric, etc. can still paralyze). Blanket Electric immunity is Gen 6.

### Burn

- Damage: 1/8 max HP per turn
- Attack reduction: **physical moves only**, × 0.5x (same as Gen 3)
- Special moves are unaffected by burn in Gen 4
- Flame Orb (item): inflicts burn on holder (synergy with Guts ability)

### Poison

- Damage: 1/8 max HP per turn
- Toxic (Badly Poisoned): N/16 max HP per turn, where N starts at 1 and increments each turn (NOT 1/8 fixed; this is a key difference from regular poison)
- Toxic counter resets on switch-out
- Toxic Orb (item): inflicts badly poisoned on holder

### Freeze

- Thaw chance: 20% per turn (same as Gen 3)
- Fire moves thaw user before attacking
- Being hit by Fire-type move thaws the Pokémon

### Confusion

- Self-hit chance: 50% (1/2; same as Gen 3)
- Duration: 2–5 turns

### Key Gen 4 Changes

- **ExtremeSpeed**: +1 priority (increased to +2 in Gen 5)
- **NEW +1 moves**: Aqua Jet, Bullet Punch, Ice Shard, Shadow Sneak, Sucker Punch, Vacuum Wave
- **NEW +2 move**: Feint (breaks through Protect/Detect; unique mechanic)

### Feint Mechanics

Feint (+2 priority) is the only move that breaks through Protect and Detect:
- Feint hits through Protect/Detect and deals damage (50 base power, Normal type, 'physical')
- Target's Protect/Detect effect ends after being hit by Feint
- In Gen 4, Feint only works if the target is protecting (fails otherwise); this restriction was removed in Gen 5

### Sucker Punch Mechanics

Sucker Punch (+1 priority) **fails** if the target is not using an attacking move:
- If target uses a status move, Sucker Punch fails (no effect)
- If target uses a damaging move, Sucker Punch executes at +1 priority

## 11. Accuracy & Evasion

### Accuracy Calculation

```
accuracy = move_accuracy * (user_accuracy_stage / target_evasion_stage)
```

Accuracy roll: PRNG(0, 100) ≤ accuracy_value. Move succeeds if roll ≤ accuracy.

| +4 | 7/3 (2.33x) |
| +3 | 2.0x |
| +2 | 5/3 (1.67x) |
| +1 | 4/3 (1.33x) |
| 0 | 1.0x |
| −1 | 3/4 (0.75x) |
| −2 | 3/5 (0.6x) |
| −3 | 1/2 (0.5x) |
| −4 | 3/7 (0.43x) |
| −5 | 3/8 (0.375x) |

### New Accuracy Modifiers

- **Gravity** (move, field effect): All moves have accuracy × 5/3 (~1.67x); prevents use of Fly, Bounce, Hi Jump Kick, Jump Kick; cancels Magnet Rise and Levitate immunity
- **Wide Lens** (item): Holder's accuracy × 1.1
- **Zoom Lens** (item): Holder's accuracy × 1.2 if moving after target
- **No Guard** (ability): All moves by and against this Pokémon always hit (accuracy checks bypassed; can still miss due to other effects like Protect)

### OHKO Moves

Horn Drill, Fissure, Guillotine, Sheer Cold:
- Base accuracy formula: 30% + (user_level − target_level)%
- Fails automatically if target is higher level
- Fails if user's accuracy is lower than 50% (after stage calculation)

## 12. Entry Hazards

### Spikes (Unchanged from Gen 3)

- 1 layer: 1/8 max HP damage on switch-in
- 2 layers: 1/6 max HP damage on switch-in
- 3 layers: 1/4 max HP damage on switch-in
- Only affects grounded Pokémon
- Flying types and Pokémon using Levitate are unaffected
- Barrier Pokémon can remove all layers with Rapid Spin or inherit via move

### Stealth Rock (NEW in Gen 4)

Damages all switching Pokémon based on Rock-type effectiveness against their type(s):

```
damage = floor(max_HP * effectiveness / 8), minimum 1
```

Where effectiveness is:
- 4x weak (e.g., Charizard Fire/Flying): 50% max HP
- 2x weak: 25% max HP
- Neutral: 12.5% max HP
- 2x resist: 6.25% max HP
- 4x resist: 3.125% max HP
- Immune: never (no type is immune to Rock)

For dual-typed Pokémon, multiply Rock effectiveness vs type1 × Rock effectiveness vs type2.

**Example**: Charizard (Fire/Flying)
- Rock vs Fire = 0.5x
- Rock vs Flying = 2x
- Total: 0.5 × 2 = 1x (neutral)
- Damage: floor(max_HP × 1 / 8) = floor(max_HP / 8) = 12.5% max HP

**Example**: Dragonite (Dragon/Flying)
- Rock vs Dragon = 1x
- Rock vs Flying = 2x
- Total: 1 × 2 = 2x
- Damage: floor(max_HP × 2 / 8) = floor(max_HP / 4) = 25% max HP

### Toxic Spikes (NEW in Gen 4)

- 1 layer: Regular poison on switch-in (1/8 max HP per turn)
- 2 layers: Badly poisoned on switch-in (N/16 max HP per turn; N increments)
- Only affects grounded Pokémon
- Grounded Poison types entering a Toxic Spikes layer remove ALL layers from the field
- Steel types and Flying types (and Levitate users) ignore Toxic Spikes

## 13. Protect & Detect

### Mechanics

Protect and Detect are identical moves. They block all damage and effects of opponent's moves for one turn.

### Consecutive Activation

If Protect/Detect is used on consecutive turns, success rate halves:
- 1st use: 100% (0xFFFF/0xFFFF)
- 2nd use in a row: ~50% (0x7FFF/0xFFFF)
- 3rd use in a row: ~25% (0x3FFF/0xFFFF)
- 4th+ use in a row: ~12.5% (0x1FFF/0xFFFF) — **caps here, does NOT continue decaying**

Success rate resets on a failed Protect/Detect or when Protect/Detect is not used that turn.

## 14. Multi-Hit Moves

### Standard Distribution

Most multi-hit moves (Icicle Spear, Rock Blast, Bullet Seed, Pin Missile, etc.) use this distribution:
- 2 hits: 37.5% (6/16)
- 3 hits: 37.5% (6/16)
- 4 hits: 12.5% (2/16)
- 5 hits: 12.5% (2/16)

### Skill Link Ability

Pokémon with Skill Link (new in Gen 4) **always hit 5 times** with multi-hit moves. No variance.

### Triple Kick Exception

Triple Kick always hits exactly 3 times, with power scaling: 10 base power for hit 1, 20 for hit 2, 30 for hit 3. Each hit has independent accuracy check.

### Reckless Modifier

Reckless (ability, new in Gen 4) applies 1.2x to multi-hit moves that use recoil mechanics (e.g., Brave Bird). The move power scales, but recoil remains at the move's defined percentage.

## 15. Flinch Mechanics

### Flinch Condition

Flinched Pokémon skip their turn. Flinch is removed at the end of the turn.

### Flinch Priority

Flinch only activates if the attacking move's user moved first in turn order (Speed-based or priority-based). If the flinch-proccing move is slower, flinch does not take effect.

### Flinch-Causing Effects

- **King's Rock** (item): 10% flinch on contact moves
- **Razor Fang** (item): 10% flinch on contact moves
- **Flinch-inherent moves**: Iron Head (30%), Zen Headbutt (30%), Astonish (30%), Air Cutter (30%), Waterfall (20%), Headbutt (30%), etc.
- **Steadfast** (ability): +1 Speed when flinched (new in Gen 4)
- **Stench** (ability): 10% flinch on moves (new effect in Gen 4; previously had no battle effect)

### Inner Focus

Pokémon with Inner Focus immunity ignore all flinch effects.

## 16. Experience System

### Base Formula

```
EXP = (b * L * t) / (7 * s)
```

Where:
- **b**: Base EXP yield (varies by species)
- **L**: Fainted Pokémon's level
- **t**: 1.0 if wild, 1.5 if trainer-owned
- **s**: Number of Pokémon in the winning party that participated in battle (not EXP Share holders)

**Note**: EXP Share in Gen 4 is a held item, not party-wide. Holders gain a separate 50% share of defeated Pokémon's base EXP.

### EXP Share Item

Holders of EXP Share gain:
```
EXPShare = (b * L * t) * 0.5 / 7
```

This is independent of party size **s**.

### Lucky Egg Item

Lucky Egg multiplies all EXP gain (base + EXP Share) by 1.5x (new in Gen 4).

### Example Verification

**Level 5 wild Pokémon with base EXP 40, defeated by level 5 Pikachu in a 1v1 solo run (no EXP Share):**

```
EXP = (40 * 5 * 1.0) / (7 * 1) = 200 / 7 = 28.57 → 28 EXP
```

With Lucky Egg: 28 * 1.5 = 42 EXP ✓

## 18. Implementation Notes

### Physical/Special Split

The most critical implementation detail in Gen 4. Every move must have a category field:

```typescript
type MoveCategory = 'physical' | 'special' | 'status'
```

When damage is calculated, consult `move.category`:
- 'physical' → use Atk and Def
- 'special' → use SpAtk and SpDef
- 'status' → no damage, apply status/effect only

Burn status (1/8 damage per turn) **only applies to physical moves**. Special moves ignore burn's damage penalty.

### Move Type ≠ Move Category

These are orthogonal:
- Move type: fire, water, electric, etc. (17 types)
- Move category: physical, special, status (3 categories)

A move can be Fire-type and physical, Fire-type and special, or Fire-type and status (rare).

### Damage Formula Order

The order of damage modifier application matters for integer rounding:

1. Base damage (as calculated above)
2. Targets (0.75x if multiple targets)
3. Weather (1.5x or 0.5x)
4. Critical (2x or 3x with Sniper)
5. Random (0.85–1.0x)
6. STAB (1.5x or 2.0x)
7. Type effectiveness (product of types)
8. Burn (0.5x if physical and attacker burned)
9. Other (Life Orb 1.3x, items, abilities)

Each application is floored before the next modifier.

### Ability Implementations

Key implementation patterns:

- **On switch-in abilities** (Download, Anticipation, Forewarn, Frisk, Snow Warning): Execute during switch-in, before turn 1 action selection
- **Passive abilities** (Dry Skin, Poison Heal, Ice Body, Rain Dish): Execute at end-of-turn in the order specified (section 17)
- **Damage modifiers** (Technician, Adaptability, Tinted Lens): Apply during damage calculation
- **Stat stage modifiers** (Simple, Download, Unaware): Apply when reading stat values for damage or move effects

### Stat Stage Interactions

When reading a stat for damage calculation, apply Unaware (ignore opponent's changes) and Simple (double friendly changes) before using:

```typescript
let atkStat = attacker.atk
if (attacker.abilities.includes('simple')) {
  atkStat = applyStages(atkStat, atkStat_stageMultiplier * 2)
}
if (defender.abilities.includes('unaware')) {
  defStat = defender.def  // ignore stages
} else {
  defStat = applyStages(defender.def, defStat_stage)
}
```

### Weather System

Weather has two sources:
1. **Move-summoned** (Rain Dance, Sunny Day, etc.): 5 turns default, extended by weather rocks
2. **Ability-summoned** (Snow Warning): Indefinite

Implement weather rocks as a check on move application:
- If summoning move is used and summoner holds corresponding weather rock, set timer to 8 instead of 5
- Ability weather is never affected by rocks

## 19. Verification Notes

### Known Game Bugs / Quirks to Emulate

#### Metronome Item Bug (Gen 4 Specific)

Metronome item in Gen 4 can have edge-case behavior with move changes (Copycat, Mimic, etc.). For simplicity, reset the multiplier on switch-out and apply linearly within a battle.

#### Explosion Damage Formula

In Gen 4, Explosion **still halves Defense** in the denominator:
```
baseDamage = floor(floor(levelFactor * 250 * Atk / (Def * 0.5)) / 50) + 2
```

This was changed in Gen 5 (Def is no longer halved). Emulating Gen 4 requires explicit handling.
