## 1. Overview

Generation 4 introduced the **physical/special split**, fundamentally restructuring how moves deal damage. This is the single most impactful change in competitive Pokémon history. Additionally, Gen 4 brought powerful new entry hazards (Stealth Rock), field effects (Trick Room, Gravity), held items (Choice Scarf, Life Orb, Focus Sash), and abilities that would define the metagame for years to come.

**Key Innovation**: Before Gen 4, move damage type (Physical vs. Special) was determined by the move's TYPE (e.g., all Fire moves were Special). Starting in Gen 4, each move has its own immutable category, independent of type. This single change made Alakazam viable physical attackers and allowed physical Fire-type moves like Fire Punch to exist.

## 2. Physical/Special Split

### 2.1 Core Mechanic Change

> **Code style**: This spec uses enum-style identifiers (`MoveCategory.Physical`, `MoveCategory.Special`, `MoveCategory.Status`) in several places. The actual codebase uses lowercase string literals: `'physical'`, `'special'`, `'status'`. Follow the string literal convention when implementing.

**Gen 4+ (Move-Based Split)**:
```typescript
enum MoveCategory {
  Physical = 'Physical',
  Special = 'Special',
  Status = 'Status',
}

interface MoveData {
  id: string;
  name: string;
  type: PokemonType;
  category: MoveCategory;        // NEW: per-move, not per-type
  power: number;
  accuracy: number;
  pp: number;
  priority: number;
  effects: MoveEffect[];
}

// Gen 4+ Damage calculation
let attackStat = moveCategory === MoveCategory.Physical ? atk : spAtk;
let defenseStat = moveCategory === MoveCategory.Physical ? def : spDef;
```

### 2.2 Metagame Impact

The split allows for **Pokémon role flexibility**:
- Alakazam: Can now run Physical moves (Focus Blast, Dazzling Gleam) alongside Special moves
- Machamp: Can run Psychic as a special move, not type-locked to physical
- Gyarados: Stone Edge (Physical Rock) becomes viable; doesn't need to settle for Power Gem (Special)

## 3. New Entry Hazards

### 3.1 Stealth Rock

The most impactful entry hazard introduced. Sets a hazard on the opponent's side that damages Pokémon switching in based on their type matchup to Rock.

```typescript
interface FieldEffect {
  id: string;
  name: string;
  turnsRemaining: number | null;  // null = indefinite (Stealth Rock, Reflect, etc.)
}

interface StealthRockEffect extends FieldEffect {
  id: 'stealth_rock';
  name: 'Stealth Rock';
  // Damage calculation on switch-in:
  // damage = (maxHP / 8) * typeEffectiveness(Rock vs switchInPokemon)
}

// Damage table:
// x4 weak (Flying, Fire, Ice, Bug): (maxHP / 2)
// x2 weak (Normal, Water, Grass, Poison, Ground, Rock, Steel): (maxHP / 4)
// x1 neutral (Electric, Flying, Psychic, Ghost, Dragon, Dark): (maxHP / 8)
// x0.5 resist (Fighting, Ground): (maxHP / 16)

> **Note**: Fairy type does not exist in Gen 4. It was introduced in Gen 6.

enum TypeEffectiveness {
  Immune = 0,
  Quarter = 0.25,   // 1/4
  Eighth = 0.125,   // 1/8
  Half = 0.5,       // 1/2
  Normal = 1,
  Double = 2,
  Quad = 4,
}

const stealthRockDamageMultiplier = {
  Normal: 1 / 8,
  Fire: 1 / 2,      // x4 weak to Rock
  Water: 1 / 8,
  Electric: 1 / 8,
  Grass: 1 / 8,
  Ice: 1 / 2,       // x4 weak to Rock
  Fighting: 1 / 16, // x0.5 resist
  Poison: 1 / 8,
  Ground: 1 / 16,   // x0.5 resist
  Flying: 1 / 2,    // x4 weak to Rock
  Psychic: 1 / 8,
  Bug: 1 / 2,       // x4 weak to Rock
  Rock: 1 / 4,
  Ghost: 1 / 8,
  Dragon: 1 / 8,
  Dark: 1 / 8,
  Steel: 1 / 4,
};

// On switch-in damage calculation:
const damageFromStealthRock = (pokemon: PokemonInstance): number => {
  const pokemonType = pokemon.baseStats.type[0]; // Check primary type
  const damagePercentage = stealthRockDamageMultiplier[pokemonType];
  return Math.max(1, Math.floor(pokemon.hp.max * damagePercentage));
};
```

### 3.2 Spikes

No change from Gen 3, but Stealth Rock adds to the hazing strategy. Spikes still layer 1-3 times (1 layer = 1/8, 2 layers = 1/6, 3 layers = 1/4 damage to grounded Pokémon).

### 3.3 Toxic Spikes

Variant of Spikes introduced. 1 layer = regular poison on switch-in (grounded). 2 layers = badly poisoned (badly poisoned damage increases each turn). Grounded Poison-type Pokémon absorb the layer, curing existing Toxic Spikes.

```typescript
interface ToxicSpikesEffect extends FieldEffect {
  id: 'toxic_spikes';
  name: 'Toxic Spikes';
  layers: 1 | 2;  // Max 2 layers (Gen 4+)
}

const applyToxicSpikesOnSwitch = (pokemon: PokemonInstance, layers: number): void => {
  // Grounded Poison-types absorb the layer instead of taking damage
  if (isGrounded(pokemon) && pokemon.type.includes('Poison')) {
    // Remove 1 layer of Toxic Spikes from opponent's field
    return;
  }

  if (!isGrounded(pokemon)) {
    return; // No effect on non-grounded
  }

  if (layers === 1) {
    applyStatus(pokemon, PokemonStatus.Poison);
  } else if (layers === 2) {
    applyStatus(pokemon, PokemonStatus.ToxicPoison);
  }
};
```

## 4. New Field Effects

### 4.1 Trick Room

Reverses Speed order for 5 turns. The slowest Pokémon moves first during Trick Room's duration. Useful for slow, bulky teams.

```typescript
interface TrickRoomEffect extends FieldEffect {
  id: 'trick_room';
  name: 'Trick Room';
  turnsRemaining: number;
}

// Implemented by:
const move = {
  id: 'trick_room',
  name: 'Trick Room',
  type: 'Psychic',
  category: MoveCategory.Status,
  power: 0,
  accuracy: 0,  // Can't miss
  pp: 5,
  priority: -7,  // Lowest priority, always moves last
  effects: [
    {
      type: 'field_effect',
      effect: TrickRoomEffect,
      duration: 5,
    },
  ],
};

// Speed ordering with Trick Room:
const getSpeedOrder = (pokemon1: PokemonInstance, pokemon2: PokemonInstance, isTrickRoom: boolean): [PokemonInstance, PokemonInstance] => {
  const speed1 = pokemon1.stats.spe;
  const speed2 = pokemon2.stats.spe;

  if (isTrickRoom) {
    // Slowest moves first
    return speed1 <= speed2 ? [pokemon1, pokemon2] : [pokemon2, pokemon1];
  } else {
    // Fastest moves first (normal)
    return speed1 >= speed2 ? [pokemon1, pokemon2] : [pokemon2, pokemon1];
  }
};
```

## 5. Key New Items

### 5.1 Choice Scarf

Multiplies Speed by 1.5x. Locks the holder to using the first move selected; can only switch moves by switching out.

```typescript
interface ItemData {
  id: string;
  name: string;
  holdEffect: HoldEffect;
}

interface ChoiceScarfEffect extends HoldEffect {
  id: 'choice_scarf';
  speedMultiplier: 1.5;
  locksToMove: boolean;  // true: locked to first move used
}

// Speed calculation:
const calculateSpeed = (pokemon: PokemonInstance): number => {
  let speed = pokemon.stats.spe;

  if (pokemon.heldItem?.id === 'choice_scarf') {
    speed *= 1.5;
  }

  return Math.floor(speed);
};

// Move selection with lock:
interface BattleState {
  pokemon: PokemonInstance;
  lockedMove: MoveData | null;  // Current locked move (if holding Choice item)
}

const selectMove = (pokemon: PokemonInstance, moveIndex: number): MoveData => {
  const move = pokemon.moves[moveIndex];

  if (pokemon.heldItem?.id.includes('choice_')) {
    // Can't switch moves while holding Choice item
    if (pokemon.battleState.lockedMove && pokemon.battleState.lockedMove.id !== move.id) {
      throw new Error('Locked to previous move by Choice item');
    }
  }

  return move;
};
```

### 5.2 Life Orb

Multiplies damage by 1.3x (130%). User takes 10% recoil damage (of damage dealt) on each hit.

```typescript
interface LifeOrbEffect extends HoldEffect {
  id: 'life_orb';
  damageMultiplier: 1.3;
  recoilPercent: 0.1;  // 10% of damage dealt

// Damage calculation with Life Orb:
const calculateDamageWithLifeOrb = (baseDamage: number, pokemon: PokemonInstance): number => {
  if (pokemon.heldItem?.id === 'life_orb') {
    return Math.floor(baseDamage * 1.3);
  }
  return baseDamage;
};

// End-of-turn recoil:
const applyLifeOrbRecoil = (pokemon: PokemonInstance, damageDealt: number): void => {
  if (pokemon.heldItem?.id === 'life_orb') {
    const recoilDamage = Math.floor(damageDealt * 0.1);
    pokemon.hp.current -= recoilDamage;
    logMessage(`${pokemon.name} is hurt by Life Orb recoil!`);
  }
};
```

### 5.3 Focus Sash

Held item that allows a Pokémon to survive any hit that would reduce it to 0 HP, leaving it at 1 HP instead. Only works when at full HP at start of turn. Single-use per battle.

```typescript
interface FocusSashEffect extends HoldEffect {
  id: 'focus_sash';
  survivesOnce: boolean;
  hasBeenUsed: boolean;
}

// Damage application with Focus Sash:
const applyDamage = (pokemon: PokemonInstance, damage: number): void => {
  const wouldFaint = pokemon.hp.current - damage <= 0;

  if (
    wouldFaint &&
    pokemon.heldItem?.id === 'focus_sash' &&
    !pokemon.battleState.focusSashUsed &&
    pokemon.hp.current === pokemon.hp.max  // Must be at full HP
  ) {
    pokemon.hp.current = 1;
    pokemon.battleState.focusSashUsed = true;
    logMessage(`${pokemon.name} survived using Focus Sash!`);
    return;
  }

  pokemon.hp.current -= damage;
};
```

### 5.4 Leftovers (Unchanged)

Restores 1/8 maxHP at the end of each turn. Behavior identical to Gen 3.

```typescript
    turnsRemaining: null,  // Indefinite, NOT affected by rocks
    source: 'ability',
  };
};
```

## 7. New Abilities

### 7.1 Adaptability

Increases STAB multiplier from 1.5x to 2x. Same-type attack bonus now doubles damage instead of multiplying by 1.5x.

```typescript
interface AdaptabilityEffect extends AbilityEffect {

  stabMultiplier: 2.0;  // Up from Gen 3's 1.5x

// STAB calculation:
const calculateSTAB = (pokemon: PokemonInstance, moveType: PokemonType): number => {
  if (!pokemon.type.includes(moveType)) {
    return 1.0;  // No STAB
  }

  if (pokemon.ability.id === 'adaptability') {
    return 2.0;  // Adaptability: 2x STAB
  }

  return 1.5;  // Standard STAB (all gens)
```

Upon switching in, raises either Attack or Special Attack by one stage, depending on which stat the opponent's team is lower in (on average or on active Pokémon).

## 8. Status Changes

### 8.1 Minor Tweaks

No major overhauls to status conditions from Gen 3. Toxic Spikes is the main new mechanic (covered in Section 3.3).

**Badly Poisoned (Toxic Poison)** is introduced as an intensified poison variant:
- Applied by 2 layers of Toxic Spikes
- Damage increases each turn: 1/8, 2/8, 3/8, 4/8, etc.
- Resets to 1/8 if switched out

```typescript
enum PokemonStatus {
  None = 'healthy',
  Burn = 'burn',
  Freeze = 'freeze',
  Paralysis = 'paralysis',
  Poison = 'poison',
  ToxicPoison = 'badly_poisoned',  // NEW in Gen 4
  Sleep = 'sleep',
}

interface StatusState {
  condition: PokemonStatus;
  turnsRemaining?: number;
  toxicCounter?: number;  // For badly poisoned escalation
}

// Badly poisoned damage:
const applyBadlyPoisonedDamage = (pokemon: PokemonInstance): void => {
  if (pokemon.status.condition !== PokemonStatus.ToxicPoison) return;

  const damageCounter = pokemon.status.toxicCounter || 1;
  const damage = Math.floor(pokemon.hp.max * (damageCounter / 8));

  pokemon.hp.current -= damage;
  pokemon.status.toxicCounter = Math.min(damageCounter + 1, 8);  // Cap at 8/8
};

// Reset counter on switch:
const onSwitchOut = (pokemon: PokemonInstance): void => {
  if (pokemon.status.condition === PokemonStatus.ToxicPoison) {
    pokemon.status.toxicCounter = 1;  // Reset on next switch-in
  }
};
```

## 9. Damage Formula Changes

The **base formula remains identical** to Gen 3, but the physical/special split (Section 2) changes which Attack and Defense stats are used:

```typescript
interface DamageCalculationInputs {
  level: number;
  attackStat: number;       // Now determined by move category, not type
  defenseStat: number;      // Now determined by move category, not type
  movePower: number;
  typeEffectiveness: number;
  stab: number;            // Same-Type Attack Bonus (1.0 or 1.5x or 2.0x with Adaptability)
  burned: boolean;         // Reduces physical moves by 50% if attacker is burned
  criticalHit: boolean;
  weatherModifier: number; // e.g., 1.5x for Water in rain
  abilityModifier: number; // e.g., Technician, Huge Power, etc.
  itemModifier: number;    // e.g., Life Orb 1.3x
}

const gen3Gen4DamageFormula = (inputs: DamageCalculationInputs): number => {
  // ===== STAB and type effectiveness =====
  let baseDamage = 0.5 * inputs.level / 5 + 2;
  baseDamage = baseDamage * inputs.movePower;
  baseDamage = baseDamage * inputs.attackStat / inputs.defenseStat;
  baseDamage = Math.floor(baseDamage / 50) + 2;

  // ===== Physical/Special split factor =====
  // Burned Pokémon deal 50% damage on PHYSICAL moves only
  if (inputs.burned && inputs.moveCategory === MoveCategory.Physical) {
    baseDamage = Math.floor(baseDamage * 0.5);
  }

  // ===== Type effectiveness =====
  baseDamage = baseDamage * inputs.typeEffectiveness;

  // ===== STAB (now affected by Adaptability) =====
  baseDamage = baseDamage * inputs.stab;

  // ===== Damage variance =====
  const variance = Math.floor(Math.random() * 16 + 85) / 100; // 85-100%
  baseDamage = Math.floor(baseDamage * variance);

  // ===== Ability, item, and weather modifiers =====
  baseDamage = Math.floor(baseDamage * inputs.abilityModifier);
  baseDamage = Math.floor(baseDamage * inputs.itemModifier);
  baseDamage = Math.floor(baseDamage * inputs.weatherModifier);

  return Math.max(1, baseDamage);
};
```

**Key Differences from Gen 3**:
- Attack/Defense selection is now based on move.category, not moveType
- This affects nearly every Pokémon's damage output
- No other formula changes

## 10. End-of-Turn Order (Gen 4 Platinum/HGSS)

Exact sequence executed at the end of each turn:

```typescript
enum EndOfTurnPhase {
  // 1. Damage effects (Weather, Poison, Burn, etc.)
  // 2. Ability effects (Dry Skin, Hydration, etc.)
  // 3. Item recovery (Leftovers, etc.)
  // 4. Volatile status turndown (Leech Seed counter, Taunt, etc.)
  // 5. Field effect turndown (Reflect, Light Screen, Stealth Rock, Trick Room, etc.)
}

const executeEndOfTurn = (battleState: BattleState): void => {
  // Active Pokémon on both sides
  const allActive = [battleState.player.active, battleState.opponent.active];

  // ===== PHASE 1: Weather and entry hazard damage =====
  for (const pokemon of allActive) {
    applyWeatherDamage(pokemon, battleState.weather);
    applyStatusDamage(pokemon);  // Poison, burn, etc.
  }

  // ===== PHASE 2: Ability end-of-turn effects =====
  for (const pokemon of allActive) {
    if (pokemon.ability.id === 'hydration' && battleState.weather.type === WeatherType.Rain) {
      removeStatus(pokemon);
    }
    if (pokemon.ability.id === 'dry_skin') {
      applyDrySkinEffect(pokemon, battleState.weather);
    }
  }

  // ===== PHASE 3: Item recovery (Leftovers, Assault Vest passive, etc.) =====
  for (const pokemon of allActive) {
    if (pokemon.heldItem?.id === 'leftovers') {
      pokemon.hp.current = Math.min(
        pokemon.hp.max,
        pokemon.hp.current + Math.floor(pokemon.hp.max / 8)
      );
      logMessage(`${pokemon.name} restored HP!`);
    }
  }

  // ===== PHASE 4: Volatile status countdown =====
  for (const pokemon of allActive) {
    pokemon.volatileStatuses.forEach(vs => {
      vs.turnsRemaining = Math.max(0, vs.turnsRemaining - 1);
    });
    pokemon.volatileStatuses = pokemon.volatileStatuses.filter(vs => vs.turnsRemaining > 0);
  }

  // ===== PHASE 5: Field effect countdown =====
  battleState.fieldEffects.forEach(fe => {
    if (fe.turnsRemaining !== null) {
      fe.turnsRemaining = Math.max(0, fe.turnsRemaining - 1);
    }
  });
  battleState.fieldEffects = battleState.fieldEffects.filter(fe => fe.turnsRemaining !== 0);

  battleState.turn += 1;
};
```

## 11. Implementation Notes

### 11.1 Gen4Ruleset Class Structure

```typescript
/**
 * Gen 4 Battle Ruleset (Diamond/Pearl/Platinum/HGSS)
 *
 * Extends: BaseRuleset (Gen 3)
 * Key Changes:
 *   - Physical/Special split per-move, not per-type
 *   - Stealth Rock, Trick Room, Gravity
 *   - Choice Scarf, Life Orb, Focus Sash
 *   - Adaptability, Download, Technician, Mold Breaker, Snow Warning
 *   - Badly poisoned (Toxic Spikes)
 *   - Weather rocks extend duration to 8 turns
 */
export class Gen4Ruleset extends BaseRuleset {
  public generation = 4;
  public name = 'Diamond/Pearl/Platinum/HGSS';

  // ===== OVERRIDDEN METHODS =====

  /**
   * Override: Damage calculation uses move.category instead of type-based split
   */
  override calculateDamage(
    attacker: PokemonInstance,
    defender: PokemonInstance,
    move: MoveData
  ): number {
    const isMovePhysical = move.category === MoveCategory.Physical;
    const attackStat = isMovePhysical ? attacker.stats.atk : attacker.stats.spAtk;
    const defenseStat = isMovePhysical ? defender.stats.def : defender.stats.spDef;

    // Continue with standard damage formula using derived stats
    return this.damageFormula(attackStat, defenseStat, move, attacker, defender);
  }

  /**
   * Override: Apply Stealth Rock, Toxic Spikes on switch-in
   */
  override onPokemonSwitchIn(
    pokemon: PokemonInstance,
    fieldState: FieldState
  ): void {
    super.onPokemonSwitchIn(pokemon, fieldState);

    // NEW: Stealth Rock damage
    if (fieldState.hasStealthRock) {
      const stealthRockDamage = this.calculateStealthRockDamage(pokemon);
      pokemon.hp.current -= stealthRockDamage;
    }

    // NEW: Toxic Spikes damage/status
    if (fieldState.toxicSpikesLayers > 0) {
      applyToxicSpikesOnSwitch(pokemon, fieldState.toxicSpikesLayers);
    }

    // NEW: Download ability
    if (pokemon.ability.id === 'download') {
      this.applyDownloadBoost(pokemon, fieldState.opponent.active);
    }

    // NEW: Snow Warning ability
    if (pokemon.ability.id === 'snow_warning') {
      fieldState.weather = { type: WeatherType.Hail, turnsRemaining: null, source: 'ability' };
    }
  }

  /**
   * Override: Validate move selection (Choice item locking)
   */
  override validateMoveSelection(
    pokemon: PokemonInstance,
    moveIndex: number,
    battleState: BattleState
  ): boolean {
    if (!super.validateMoveSelection(pokemon, moveIndex, battleState)) {
      return false;
    }

    // NEW: Choice item move locking
    if (pokemon.heldItem?.id.startsWith('choice_')) {
      if (battleState.lockedMoveByChoice !== null) {
        // Can only use the same move
        return battleState.lockedMoveByChoice === moveIndex;
      }
    }

    return true;
  }

  /**
   * Override: Apply move with new field effects (Stealth Rock, Trick Room, Gravity)
   */
  override applyMoveEffect(
    move: MoveData,
    attacker: PokemonInstance,
    defender: PokemonInstance,
    fieldState: FieldState
  ): void {
    super.applyMoveEffect(move, attacker, defender, fieldState);

    // NEW: Stealth Rock setup
    if (move.id === 'stealth_rock') {
      fieldState.stealthRockActive = true;
      logMessage(`Pointed stones float in the air!`);
    }

    // NEW: Trick Room
    if (move.id === 'trick_room') {
      fieldState.trickRoomActive = true;
      fieldState.trickRoomTurns = 5;
      logMessage(`Trick Room was set up!`);
    }

    // NEW: Gravity
    if (move.id === 'gravity') {
      fieldState.gravityActive = true;
      fieldState.gravityTurns = 5;
      logMessage(`Gravity intensified!`);
    }
  }

  /**
   * Override: Speed calculation with Trick Room and Choice Scarf
   */
  override getSpeedOrder(
    pkmn1: PokemonInstance,
    pkmn2: PokemonInstance,
    fieldState: FieldState,
    priority1: number,
    priority2: number
  ): [PokemonInstance, PokemonInstance] {
    // Priority is checked first (no change from Gen 3)
    if (priority1 !== priority2) {
      return priority1 > priority2 ? [pkmn1, pkmn2] : [pkmn2, pkmn1];
    }

    // Calculate actual Speed stats
    let speed1 = this.calculateSpeed(pkmn1);
    let speed2 = this.calculateSpeed(pkmn2);

    // NEW: Trick Room reverses speed order
    if (fieldState.trickRoomActive) {
      return speed1 <= speed2 ? [pkmn1, pkmn2] : [pkmn2, pkmn1];
    }

    // Normal: fastest first
    return speed1 >= speed2 ? [pkmn1, pkmn2] : [pkmn2, pkmn1];
  }

  /**
   * NEW: Calculate speed with Choice Scarf
   */
  protected calculateSpeed(pokemon: PokemonInstance): number {
    let speed = pokemon.stats.spe;

    // NEW: Choice Scarf
    if (pokemon.heldItem?.id === 'choice_scarf') {
      speed = Math.floor(speed * 1.5);
    }

    return speed;
  }

  /**
   * NEW: Calculate Stealth Rock damage based on type matchup
   */
  protected calculateStealthRockDamage(pokemon: PokemonInstance): number {
    const primaryType = pokemon.baseStats.type[0];
    const damagePercentage = stealthRockDamageTable[primaryType];
    return Math.max(1, Math.floor(pokemon.hp.max * damagePercentage));
  }

  /**
   * Override: End-of-turn with new item/ability interactions
   */
  override executeEndOfTurn(battleState: BattleState): void {
    // Phase 1: Status and weather damage
    for (const pokemon of [battleState.player.active, battleState.opponent.active]) {
      applyWeatherDamage(pokemon, battleState.fieldState.weather);
      applyStatusDamage(pokemon);

      // NEW: Life Orb recoil
      if (pokemon.heldItem?.id === 'life_orb' && pokemon.lastMoveDealt > 0) {
        const recoilDamage = Math.floor(pokemon.lastMoveDealt * 0.1);
        pokemon.hp.current -= recoilDamage;
      }
    }

    // Phase 2: Ability effects (no changes specific to Gen 4 in this phase)
    super.executeEndOfTurn(battleState);

    // Phase 3-5: Inherited from BaseRuleset

    // NEW: Decrement field effects (Trick Room, Gravity, Stealth Rock removal if ever needed)
    battleState.fieldState.trickRoomTurns = Math.max(0, battleState.fieldState.trickRoomTurns - 1);
    battleState.fieldState.gravityTurns = Math.max(0, battleState.fieldState.gravityTurns - 1);

    if (battleState.fieldState.trickRoomTurns === 0) {
      battleState.fieldState.trickRoomActive = false;
    }
    if (battleState.fieldState.gravityTurns === 0) {
      battleState.fieldState.gravityActive = false;
    }
  }

  /**
   * Override: Handle item locking for Choice items
   */
  override onMoveUsed(
    pokemon: PokemonInstance,
    moveIndex: number,
    battleState: BattleState
  ): void {
    super.onMoveUsed(pokemon, moveIndex, battleState);

    // NEW: Lock to move if using Choice item for the first time
    if (pokemon.heldItem?.id.startsWith('choice_')) {
      if (battleState.lockedMoveByChoice === null) {
        battleState.lockedMoveByChoice = moveIndex;
      }
    }
  }

  /**
   * Override: Apply Focus Sash survival mechanic
   */
  override beforeDamageApplied(
    pokemon: PokemonInstance,
    damage: number,
    attacker: PokemonInstance,
    battleState: BattleState
  ): void {
    const wouldFaint = pokemon.hp.current - damage <= 0;

    // NEW: Focus Sash survives from full HP
    if (
      wouldFaint &&
      pokemon.heldItem?.id === 'focus_sash' &&
      pokemon.hp.current === pokemon.hp.max &&
      !battleState.focusSashUsed[pokemon.id]
    ) {
      pokemon.hp.current = 1;
      battleState.focusSashUsed[pokemon.id] = true;
      logMessage(`${pokemon.name} survived using Focus Sash!`);
      return;
    }

    super.beforeDamageApplied(pokemon, damage, attacker, battleState);
  }

  /**
   * Override: Weather rock duration
   */
  override setWeatherFromMove(
    weather: WeatherType,
    source: PokemonInstance,
    battleState: BattleState
  ): void {
    let duration = 5; // Default move duration

    // NEW: Weather rock extends to 8 turns
    const weatherRocks = {
      [WeatherType.Rain]: 'damp_rock',
      [WeatherType.Sun]: 'heat_rock',
      [WeatherType.Hail]: 'icy_rock',
      [WeatherType.Sandstorm]: 'smooth_rock',
    };

    if (source.heldItem?.id === weatherRocks[weather]) {
      duration = 8;
    }

    battleState.fieldState.weather = {
      type: weather,
      turnsRemaining: duration,
      source: 'move',
    };
  }
}
```
