## 1. Overview

**Generation 8** (Pokémon Sword and Shield, released November 2019) represents a major shift in battle mechanics:

- **Dynamax/Gigantamax** replaces Mega Evolution and Z-Moves as the primary battle gimmick
- **Max Raids** introduce cooperative Dynamax battles (not covered in 1v1/multi battle mechanics here)
- **Nature Minting** allows players to change a Pokémon's nature after catching (via Mints)
- **Limited Pokédex** (DLC changes availability, but only ~400 Pokémon available in base game)
- **EXP Candy** provides alternative leveling mechanism via items
- **Auto-EXP Share** is always active for the entire party — no toggle to disable it
- **Ability Patch** allows changing a Pokémon's Hidden Ability
- **Heavy-Duty Boots** is a new held item that blocks entry hazard damage

### Key Changes from Gen 7
- **Terrain power boost**: Nerfed from 1.5x (Gen 6-7) to **1.3x** (Gen 8+)
- **Burn damage**: Remains 1/16 max HP (same as Gen 7)
- **Paralysis speed**: Remains ×0.5 (same as Gen 7)
- **Confusion self-hit**: Remains 33% chance (same as Gen 7)

### Key Removals from Previous Generations
- **Mega Evolution** (removed; was Gen 6-7 gimmick)
- **Z-Moves** (removed; was Gen 7 gimmick)
- **Pursuit** move (removed; no longer in the game data)
- **Hidden Power** move (removed)
- **Return / Frustration** moves (removed)

## 2. Dynamax Battle Gimmick

**Dynamax** is the primary battle gimmick in Generation 8. It temporarily transforms a Pokémon into a gigantic form with enhanced stats and modified moves.

### Dynamax Rules

```typescript
interface DynamaxState {
  isActive: boolean;
  turnsRemaining: number;
  dynamaxLevel: number; // 0-10, affects Max HP multiplier
  gigantamaxForm?: string; // Species name if applicable (e.g., "charizard-gmax")
}

interface BattleGimmick {
  type: "dynamax" | "mega" | "zMove" | null;
  canActivate(pokemon: Pokemon): boolean;
  activate(pokemon: Pokemon, moveIndex: number): BattleAction;
  deactivate(pokemon: Pokemon): void;
}
```

### Dynamax Mechanics

| Property | Value | Details |
|----------|-------|---------|
| **One per Battle** | 1 per team | Each team can Dynamax once per battle (singles or doubles) |
| **Duration** | 3 turns | Lasts 3 consecutive turns; counts down at end of turn |
| **HP Multiplier** | × (1.5 + dynamaxLevel × 0.05) | At Level 0: ×1.50; at Level 10: ×2.00 |
| **Move Transformation** | All moves → Max Moves | Every move becomes a Max Move based on its type |
| **Entry Restriction** | Cannot Dynamax if holding Z-Crystal or Mega Stone | Z-Crystal, Mega Stone usage prevents Dynamax |
| **Activation** | On turn order before move execution | Player selects to Dynamax instead of using a normal move |

### Dynamax HP Calculation

```typescript
function calculateDynamaxHP(normalHP: number, dynamaxLevel: number): number {
  // dynamaxLevel is 0-10
  // CORRECT formula: multiply normal HP by (1.5 + dynamaxLevel * 0.05)
  // - dynamaxLevel 0:  normalHP * 1.50  (150%)
  // - dynamaxLevel 5:  normalHP * 1.75  (175%)
  // - dynamaxLevel 10: normalHP * 2.00  (200%)
  const ratio = 1.5 + (dynamaxLevel * 0.05);
  return Math.floor(normalHP * ratio);
}
// VERIFIED Showdown data/conditions.ts:771: `const ratio = 1.5 + (pokemon.dynamaxLevel * 0.05);`
```

### Turn Countdown

The Dynamax state is tracked per Pokémon:

```typescript
endOfTurn() {
  // At end of each turn, decrement Dynamax counters
  activeTeamA.forEach(pokemon => {
    if (pokemon.dynamax.isActive) {
      pokemon.dynamax.turnsRemaining--;
      if (pokemon.dynamax.turnsRemaining === 0) {
        pokemon.dynamax.isActive = false;
        // Revert to normal form; move slots restore to original moves
        pokemon.moves = pokemon.originalMoves;
      }
    }
  });
  activeTeamB.forEach(pokemon => {
    if (pokemon.dynamax.isActive) {
      pokemon.dynamax.turnsRemaining--;
      if (pokemon.dynamax.turnsRemaining === 0) {
        pokemon.dynamax.isActive = false;
        pokemon.moves = pokemon.originalMoves;
      }
    }
  });
}
```

### Dynamax Immunities and Restrictions

Dynamaxed Pokémon have several special immunities:

- **Flinch immune**: Cannot be flinched
- **OHKO immune**: Fissure, Horn Drill, Guillotine, and Sheer Cold automatically fail
- **Forced-switch immune**: Roar, Whirlwind, Dragon Tail, and Circle Throw cannot force out a Dynamaxed Pokémon
- **Weight-based effects**: Weight-based moves (Low Kick, Grass Knot, Heavy Slam, Heat Crash) treat Dynamaxed Pokémon as having indeterminate weight
- **Stat changes apply normally**: Intimidate, stat-boosting moves, etc. work as usual during Dynamax
- **Choice items**: Choice Band/Specs/Scarf lock is suppressed during Dynamax — the Pokémon can use any of its (Max) moves
- **Cannot Dynamax**: Zacian, Zamazenta, and Eternatus (including alternate forms) cannot Dynamax

## 3. Max Moves

When a Pokémon is Dynamaxed, all of its moves are transformed into **Max Moves**. The Max Move used depends on the original move's type and category.

### Max Move Power Conversion

Max Move power depends on **both the original move's base power AND its type**. Fighting and Poison types use a lower power table; all other types use a higher table.

**Fighting / Poison types:**

| Original Move Power | Max Move Power |
|-------|---------|
| < 45 | 70 |
| 45–54 | 75 |
| 55–64 | 80 |
| 65–74 | 85 |
| 75–109 | 90 |
| 110–149 | 95 |
| ≥ 150 | 100 |

**All other types:**

| Original Move Power | Max Move Power |
|-------|---------|
| < 45 | 90 |
| 45–54 | 100 |
| 55–64 | 110 |
| 65–74 | 120 |
| 75–109 | 130 |
| 110–149 | 140 |
| ≥ 150 | 150 |

<!-- VERIFIED Showdown sim/dex-moves.ts:517-549 -->

**Examples**:
- Flamethrower (90 BP, Fire) → Max Flare (130 power, not 85)
- Close Combat (120 BP, Fighting) → Max Knuckle (95 power, not 90)
- Ice Beam (90 BP, Ice) → Max Hailstorm (130 power)

## 4. Gigantamax

**Gigantamax** is a special form of Dynamax available only to certain Pokémon species. A Gigantamax Pokémon has a unique appearance and one move of a matching type becomes a **G-Max Move** with a unique effect instead of a generic Max Move.

### Gigantamax Requirements

```typescript
interface GigantamaxData {
  species: string; // e.g., "charizard", "pikachu"
  form: string; // Gigantamax form (e.g., "charizard-gmax")
  requiredMove?: PokemonType | null; // Type of move that gets unique G-Max Move
  requiredLevel?: number; // Minimum level (usually none)
}
```

### Gigantamax vs. Regular Dynamax

| Aspect | Regular Dynamax | Gigantamax |
|--------|---|---|
| **Available to** | All Pokémon | Specific species only |
| **Visual** | Giant version of normal form | Unique Gigantamax form |
| **HP Multiplier** | ×1.50–×2.00 (based on Dynamax Level) | ×1.50–×2.00 (same as Dynamax) |
| **Move Transformation** | All moves → generic Max Moves | One matching-type move → unique G-Max Move; others → Max Moves |
| **G-Max Move Effect** | N/A | Species-specific effect |

### Key Gigantamax Pokémon and G-Max Moves

## 5. No Mega Evolution or Z-Moves

Generation 8 **completely removes** Mega Evolution and Z-Moves from competitive mechanics. The ruleset must return `null` for these gimmicks:

```typescript
class Gen8Ruleset extends BaseRuleset {

  // Mega Evolution is NOT available in Gen 8
  getMegaEvolutionGimmick(pokemon: Pokemon): BattleGimmick | null {
    return null; // Mega Evolution disabled
  }

  // Z-Moves are NOT available in Gen 8
  getZMoveGimmick(pokemon: Pokemon): BattleGimmick | null {
    return null; // Z-Moves disabled
  }

  // Dynamax is the ONLY battle gimmick
  getDynamaxGimmick(pokemon: Pokemon): BattleGimmick | null {
    return {
      type: "dynamax",
      canActivate: (pokemon) => this.canDynamax(pokemon),
      activate: (pokemon) => this.activateDynamax(pokemon),
      deactivate: (pokemon) => this.deactivateDynamax(pokemon),
    };
  }

  // Pokémon cannot hold both Mega Stone and Z-Crystal
  // (Mega Stones and Z-Crystals do not exist in Gen 8)
  canHoldItem(pokemon: Pokemon, item: Item): boolean {
    if (item.name === "Mega Stone" || item.effect?.includes("Z-Crystal")) {
      return false;
    }
    // Allow all other items
    return true;
  }
}
```

## 8. Heavy-Duty Boots Held Item

**Heavy-Duty Boots** is a new held item that provides immunity to entry hazards while remaining on the field:

```typescript
const HeavyDutyBootsItem: Item = {
  name: "Heavy-Duty Boots",
  type: "held",
  effect: {
    description: "Prevents damage from entry hazards (Stealth Rock, Spikes, Toxic Spikes)",
    immuneToHazards: true,
    immuneToStealth Rock: true,
    immuneToSpikes: true,
    immuneToToxicSpikes: true,
  },
  battleEffect: (pokemon: Pokemon, battle: Battle) => {
    // On switch-in, check if Stealth Rock/Spikes/Toxic Spikes are active
    if (battle.field.stealthRockActive || battle.field.spikesLayers > 0) {
      // Prevent damage application
      return { preventDamage: true };
    }
  }
};
```

### Heavy-Duty Boots Interaction

- **Blocks all entry hazard damage**: Stealth Rock, Spikes (all layers), Toxic Spikes, Sticky Web, G-Max Steelsurge
- **Does NOT block**: Weather damage (Hail, Sandstorm), terrain damage
- **Does NOT remove hazards**: Only prevents damage; hazards remain on field for other Pokémon
- **Works on switch-in**: Applies automatically when Pokémon switches in
- **Can be knocked off**: If hit by Knock Off move, loses the effect

## 9. EXP Share Always Active

In Generation 8, the **EXP Share** mechanic changed fundamentally:

```typescript
interface ExpShareMechanics {
  alwaysActive: true;
  cannotBeToggled: true;
  affectsEntireParty: true;
  partyMembersNotInBattleShareExp: true;
}
```

### EXP Distribution Rules

| Scenario | Behavior |
|----------|----------|
| **Pokémon in battle gains EXP** | 100% of earned EXP |
| **Other party members (not in battle)** | Receive 50% of earned EXP each |
| **Fainted Pokémon** | Do NOT receive EXP |
| **Traded Pokémon** | Still receive EXP penalty (if applicable per ruleset) |

### Implementation

```typescript
function distributeExperience(
  battleParticipants: Pokemon[],
  inactiveParty: Pokemon[],
  expGained: number
): void {
  // Active participants get full EXP
  battleParticipants.forEach(pokemon => {
    if (!pokemon.fainted) {
      pokemon.experience += expGained;
    }
  });

  // Inactive party members get 50% EXP (Gen 8 "Exp Share Always On")
  inactiveParty.forEach(pokemon => {
    if (!pokemon.fainted) {
      pokemon.experience += Math.floor(expGained * 0.5);
    }
  });

  // Check for level-ups
  [...battleParticipants, ...inactiveParty].forEach(pokemon => {
    checkLevelUp(pokemon);
  });
}
```

## 12. End-of-Turn Order and Dynamax Mechanics

### Turn Order in Pokémon Battles

The execution order within a turn follows this sequence:

```typescript
type TurnPhase =
  | "priorityMoves"
  | "normalMoves"
  | "statChanges"
  | "weather"
  | "terrain"
  | "endOfTurn"
  | "dynamaxCountdown"
  | "turnIncrement";

function executeTurn(
  battle: Battle
): void {
  // 1. Execute priority moves (priority > 0)
  executePriorityMoves(battle);

  // 2. Execute normal moves (priority 0, speed-based order)
  executeNormalMoves(battle);

  // 3. Apply stat changes that occurred during moves
  applyStatChanges(battle);

  // 4. Apply weather effects
  applyWeatherDamage(battle);

  // 5. Apply terrain effects
  applyTerrainEffects(battle);

  // 6. Apply ability end-of-turn effects (Regenerator, etc.)
  applyAbilityEndOfTurn(battle);

  // 7. Apply status effects (Poison, Burn, etc.)
  applyStatusDamage(battle);

  // 8. Dynamax countdown (Decrement Dynamax turnsRemaining)
  decrementDynamaxCounters(battle);

  // 9. Check for fainting
  checkForFainting(battle);

  // 10. Increment turn counter
  battle.turnNumber++;
}

function decrementDynamaxCounters(battle: Battle): void {
  const allPokemon = [
    ...battle.team1.active,
    ...battle.team2.active
  ];

  allPokemon.forEach(pokemon => {
    if (pokemon.dynamax.isActive) {
      pokemon.dynamax.turnsRemaining--;

      if (pokemon.dynamax.turnsRemaining === 0) {
        // Revert to normal form
        pokemon.dynamax.isActive = false;
        pokemon.dynamax.gigantamaxForm = null;
        // Restore original moves (not Max Moves anymore)
        pokemon.moves = pokemon.baseMoveset.slice();

        // Notification
        battle.log.push(`${pokemon.name} reverted to normal form!`);
      }
    }
  });
}
```

### Gen 8-Specific End-of-Turn Changes

- **Dynamax countdown**: Decrements after damage and status damage
- **Terrain boost**: Nerfed to 1.3× (from 1.5× in Gen 6-7); applies to grounded Pokémon in matching terrain
- **Sticky Web**: Reduces Speed by 1 stage of grounded Pokémon on switch-in (not end-of-turn; blocked by Heavy-Duty Boots)
- **Regenerator**: Restores 33% HP if withdrawn; triggers when exiting field
- **Intimidate**: Activates on switch-in, lowers Attack of opponent by 1

## 13. Implementation Notes: Gen8Ruleset Class

### Class Structure

```typescript
class Gen8Ruleset extends BaseRuleset {
  readonly generationNumber: number = 8;
  readonly name: string = "Generation 8 (Sword/Shield)";

  // Battle gimmick methods
  getMegaEvolutionGimmick(pokemon: Pokemon): BattleGimmick | null {
    return null; // Not available in Gen 8
  }

  getZMoveGimmick(pokemon: Pokemon): BattleGimmick | null {
    return null; // Not available in Gen 8
  }

  getDynamaxGimmick(pokemon: Pokemon): BattleGimmick | null {
    // Dynamax is THE gimmick in Gen 8
    return {
      type: "dynamax",
      canActivate: (pokemon) => this.canDynamax(pokemon),
      activate: (pokemon) => this.activateDynamax(pokemon),
      deactivate: (pokemon) => this.deactivateDynamax(pokemon),
    };
  }

  // Ability availability
  isAbilityAvailable(ability: Ability): boolean {
    const gen8Abilities = [
      "libero", "gorilla-tactics", "screen-cleaner", "mirror-armor",
      "intrepid-sword", "dauntless-shield", "neutralizing-gas",
      "regenerator", // Continued from previous gens
      // ... plus all inherited Gen 3-7 abilities still available
    ];
    return gen8Abilities.includes(ability.id.toLowerCase());
  }

  // Move availability
  isMoveAvailable(move: Move): boolean {
    const removedMoves = ["pursuit"]; // Removed in Gen 8
    if (removedMoves.includes(move.name.toLowerCase())) {
      return false;
    }
    // Body Press, Behemoth Blade, Steel Beam are available
    return true;
  }

  // Item availability
  isItemAvailable(item: Item): boolean {
    // Gen 8 items: Heavy-Duty Boots, etc.
    const gen8Items = ["heavy-duty-boots"];
    // Mega Stones and Z-Crystals NOT available
    const removedItems = [
      /Mega Stone/i,
      /Z-Crystal/i,
      /Megastone/i
    ];

    const isRemoved = removedItems.some(pattern => pattern.test(item.name));
    return !isRemoved;
  }

  // Dynamax implementation
  private canDynamax(pokemon: Pokemon): boolean {
    // Cannot Dynamax if holding Mega Stone or Z-Crystal (for legacy compatibility check)
    if (pokemon.heldItem) {
      const itemName = pokemon.heldItem.name.toLowerCase();
      if (itemName.includes("mega") || itemName.includes("z-crystal")) {
        return false;
      }
    }
    // Cannot Dynamax if already Dynamaxed
    if (pokemon.dynamax.isActive) {
      return false;
    }
    // Team can only Dynamax once per battle
    const teamAlreadyDynamaxed = pokemon.team.some(p =>
      p.dynamax.usedThisBattle
    );
    return !teamAlreadyDynamaxed;
  }

  private activateDynamax(pokemon: Pokemon): void {
    pokemon.dynamax.isActive = true;
    pokemon.dynamax.turnsRemaining = 3;
    pokemon.dynamax.usedThisBattle = true;

    // Apply HP multiplier: normalHP * (1.5 + dynamaxLevel * 0.05)
    const ratio = 1.5 + (pokemon.dynamaxLevel * 0.05);
    pokemon.stats.hp = Math.floor(pokemon.stats.hp * ratio);
    pokemon.currentHP = Math.floor(pokemon.currentHP * ratio);

    // Check if Gigantamax form applies
    if (this.canGigantamax(pokemon)) {
      pokemon.dynamax.gigantamaxForm = pokemon.species.id + "-gmax";
    }

    // Transform all moves to Max Moves
    pokemon.moves = pokemon.baseMoveset.map(move =>
      this.getMaxMove(move)
    );
  }

  private deactivateDynamax(pokemon: Pokemon): void {
    pokemon.dynamax.isActive = false;
    // Restore base HP using getUndynamaxedHP (preserves current HP ratio)
    pokemon.currentHP = pokemon.getUndynamaxedHP();
    pokemon.stats.hp = pokemon.baseMaxHP;

    // Restore original moves
    pokemon.moves = pokemon.baseMoveset.slice();

    pokemon.dynamax.gigantamaxForm = null;
  }

  private getMaxMove(originalMove: Move): Move {
    if (originalMove.category === "status") {
      return MaxGuard;
    }

    const maxMoveTable: Record<PokemonType, MaxMove> = {
      "normal": { name: "Max Strike", type: "normal", basePower: this.calculateMaxPower(originalMove.basePower), secondaryEffect: { type: "stat", target: "defender", stat: "speed", change: -1 } },
      "fire": { name: "Max Flare", type: "fire", basePower: this.calculateMaxPower(originalMove.basePower), secondaryEffect: { type: "weather", weather: "sun", duration: 5 } },
      "water": { name: "Max Geyser", type: "water", basePower: this.calculateMaxPower(originalMove.basePower), secondaryEffect: { type: "weather", weather: "rain", duration: 5 } },
      // ... etc for all types
    };

    const originalType = originalMove.type;
    return maxMoveTable[originalType] || maxMoveTable["normal"];
  }

  private calculateMaxPower(originalPower: number, moveType: PokemonType): number {
    if (originalPower === 0) return 0; // Status moves → Max Guard
    const isFightingPoison = ['fighting', 'poison'].includes(moveType);
    if (isFightingPoison) {
      // Fighting/Poison Max Moves use a lower power table
      if (originalPower >= 150) return 100;
      if (originalPower >= 110) return 95;
      if (originalPower >= 75) return 90;
      if (originalPower >= 65) return 85;
      if (originalPower >= 55) return 80;
      if (originalPower >= 45) return 75;
      return 70;
    } else {
      // All other types use a higher power table
      if (originalPower >= 150) return 150;
      if (originalPower >= 110) return 140;
      if (originalPower >= 75) return 130;
      if (originalPower >= 65) return 120;
      if (originalPower >= 55) return 110;
      if (originalPower >= 45) return 100;
      return 90;
    }
  }
  // VERIFIED Showdown sim/dex-moves.ts:517-549

  private canGigantamax(pokemon: Pokemon): boolean {
    const gigantamaxSpecies = [
      "charizard", "pikachu", "alcremie", "eevee",
      "urshifu"
      // ... extended in DLC
    ];
    return gigantamaxSpecies.includes(pokemon.species.id.toLowerCase());
  }

  // Exp Share always active
  getExpShareBehavior(): ExpShareBehavior {
    return {
      alwaysActive: true,
      canToggle: false,
      inactivePartyMultiplier: 0.5, // 50% EXP for inactive party members
    };
  }

  // Nature Minting support
  canChangaNature(pokemon: Pokemon): boolean {
    return true; // Any Pokémon can have its nature changed via Mint
  }

  // Rapid Spin boost
  getMoveEffect(move: Move): MoveEffect {
    if (move.name.toLowerCase() === "rapid-spin") {
      return {
        ...super.getMoveEffect(move),
        boostsSpeed: 1, // NEW in Gen 8
      };
    }
    return super.getMoveEffect(move);
  }

  // Ability check
  validateAbility(pokemon: Pokemon): boolean {
    const ability = pokemon.ability;
    // All Gen 3-8 abilities are available
    return this.isAbilityAvailable(ability);
  }
}
```

## 14. Example Battle Scenario: Dynamax Usage

### Setup

```typescript
// Team A: Trainer with Charizard
const charizard = new Pokemon({
  species: "charizard",
  level: 50,
  ability: "blaze",
  moves: ["flamethrower", "earthquake", "air-slash", "dragon-dance"],
  heldItem: "choice-specs", // Holdable in Gen 8
  dynamaxLevel: 5, // Max HP multiplier = ×1.75 (175% of normal)
});

// Team B: Trainer with Gyarados
const gyarados = new Pokemon({
  species: "gyarados",
  level: 50,
  ability: "intimidate",
  moves: ["waterfall", "earthquake", "stone-edge", "ice-fang"],
  heldItem: "heavy-duty-boots", // NEW Gen 8 item
  dynamaxLevel: 3,
});

const battle = new Battle(
  new Team([charizard]),
  new Team([gyarados]),
  new Gen8Ruleset()
);
```

### Turn Sequence

**Turn 1:**
- Player selects **Dynamax** action (instead of move)
- Charizard enters Dynamax form
  - HP multiplied: 200 (base) × 1.75 = **350 HP** (dynamaxLevel=5: ×1.75)
  - `dynamax.turnsRemaining = 3`
  - Moves transform: Flamethrower → Max Flare
- Opponent uses Waterfall (normal move)
- **End of turn:** Charizard's Dynamax countdown: 3 → 2

**Turn 2:**
- Charizard uses **Max Flare** (from Flamethrower)
  - Base Power: 130 (calculated from Flamethrower's 90 power, Fire type uses non-Fighting/Poison table)
  - Secondary Effect: Sets Sunny Weather for 5 turns
- Opponent switches to Gyarados
  - Intimidate lowers Charizard's Attack by 1 stage as normal (stat changes DO apply during Dynamax)
- Sunny Weather active (+50% Fire-type moves next turn)
- **End of turn:** Charizard's Dynamax countdown: 2 → 1

**Turn 3:**
- Charizard uses **Max Flare** again
  - Damage boosted by Sunny Weather
- Opponent attacks
- **End of turn:** Charizard's Dynamax countdown: 1 → 0
  - **Charizard reverts to normal form**
  - HP restored proportionally: current HP ratio preserved, max HP reverts to 200
  - Moves restored to original moveset
